# DoD: Cecelia 图片支持（飞书 + Dashboard）

## 验收清单

- [ ] **llm-caller.js：callLLM 支持 imageContent 选项**
  - callLLM options 接受 `imageContent: Array`（content blocks）
  - callAnthropicAPI 构建 `[{ type: 'text', text: prompt }, ...imageContent]` 格式传给 Anthropic API
  - callClaudeViaBridge 仅传文字 prompt（不支持 vision，降级处理）
  Test: manual:bash -c "grep -q 'imageContent' packages/brain/src/llm-caller.js"

- [ ] **routes.js：飞书图片消息接收**
  - msgType 过滤改为允许 text/audio/image（不再丢弃 image）
  - 新增 `downloadFeishuImage(messageId, imageKey, accessToken)` 函数
  - image 消息提取 image_key，下载图片，转 base64
  - 传 imageContent 给 handleChat
  Test: manual:bash -c "grep -q 'downloadFeishuImage' packages/brain/src/routes.js"

- [ ] **routes.js：orchestrator/chat 端点支持 image_base64**
  - POST /orchestrator/chat 接收 image_base64 和 image_media_type
  - 透传给 handleChat
  Test: manual:bash -c "grep -q 'image_base64' packages/brain/src/routes.js"

- [ ] **orchestrator-chat.js：handleChat 支持 imageContent**
  - handleChat 第4参数接受 imageContent（可选）
  - callWithHistory 支持 imageContent，构建多模态 content
  Test: manual:bash -c "grep -q 'imageContent' packages/brain/src/orchestrator-chat.js"

- [ ] **ConsciousnessChat.tsx：图片上传 UI**
  - 输入区加📎图片上传按钮（input file accept image/*）
  - 选图后显示缩略图预览
  - 发送时把 image_base64 和 image_media_type 带上
  Test: manual:bash -c "grep -q 'image_base64' apps/api/features/cecelia/pages/ConsciousnessChat.tsx"

- [ ] **CeceliaChat.tsx：图片上传 UI**
  - 同 ConsciousnessChat，输入区加图片上传
  Test: manual:bash -c "grep -q 'image_base64' apps/api/features/shared/components/CeceliaChat.tsx"
