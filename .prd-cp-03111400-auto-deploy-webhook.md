# PRD: 自动部署 — GitHub Actions → Brain Webhook

## 背景

当前 `deploy.yml` SSH 到旧服务器 `146.190.52.84`（已废弃），导致每次 PR 合并后需要手动构建部署。
Cecelia 实际运行在 Mac mini（内网），GitHub Actions 无法直连。

## 目标

PR 合并到 main 后，自动完成前端构建 + 部署到香港 VPS + Brain 重启（如需要）。

## 方案

```
PR merge → GitHub Actions → POST Brain webhook（通过公网入口）
                                    ↓
                            Brain 本地执行部署脚本：
                            git pull → build → rsync HK VPS → restart brain
```

## 成功标准

- Brain 新增 `POST /api/brain/deploy` 端点，带 token 鉴权
- 部署脚本 `scripts/deploy-local.sh` 执行完整部署流程
- `deploy.yml` 改为调用 Brain webhook
- 端点有基本安全防护（deploy token）

## 不做的事

- 不做回滚机制（后续迭代）
- 不做蓝绿部署
