## 三条主要数据流（用不同颜色的箭头）

### 流 A：对话流（蓝色）
```
Alex 说话
  → 嘴巴(orchestrator-chat)
    → 加载：情绪 + self-model + 记忆 + 系统状态
    → LLM 生成回复 → 返回给 Alex
    → 同时后台：提取意图 → 丘脑 → 可能变成 Task
```

### 流 B：自动执行流（绿色）
```
心跳 Tick（每2min）
  → 规划器选任务
  → 执行器派发 → Bridge → Claude Code Agent
  → Agent 完成 → 回调 → 更新状态
  → 自动学习 → 记录经验
```

### 流 C：内心活动流（紫色）
```
心跳 Tick
  → 感知系统收集 13 种信号
  → 情绪层推导当前情绪
  → 反思层（accumulator 满时）→ 生成洞察
  → 欲望形成 → 评分 → 如果够强 → 飞书表达
同时：
  → 反刍（空闲时）→ 消化 learnings → 深度洞察
  → 每日合并（凌晨3点）→ 更新 self-model
```
