# Task: 添加 AGENTS.md + .agent-knowledge/ 知识索引层

## 需求

Cecelia 无法感知自己有哪些能力（44个 Skills），每次 Brain 调度任务时只能靠 CLAUDE.md 猜测可用工具，导致派发不准确。

按照行业标准（AGENTS.md 规范，60,000+ 项目采用）为 Cecelia 添加：

1. **根 `AGENTS.md`**：~300 token 的入口地图，包含模块地图和能力索引指针
2. **`.agent-knowledge/skills-index.md`**：44 个 Skill 完整索引，按类型分组
3. **`.agent-knowledge/brain.md`**：Brain 模块详情（API/评分公式/记忆系统/保护机制）

## 成功标准

- 文件内容与源码一致（端口、版本、常量均来自实际代码）
