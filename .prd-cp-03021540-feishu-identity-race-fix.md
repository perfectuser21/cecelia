# PRD: getGroupMembers 写入 feishu_users Race Condition 修复

## Bug

PR #282 中 `getGroupMembers` 写 feishu_users 的 pool.query 是 fire-and-forget（forEach + .catch()，不 await）。
`getFeishuUserName` 紧接着查 feishu_users 时，写入还没完成，仍返回 guest。

## 修复

将 `getGroupMembers` 中的 pool.query writes 改为 `await Promise.all(...)`，确保写入完成后再返回。
