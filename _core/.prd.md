# PRD: CeceliaPage 重设计 - Conversational Briefing

## 功能描述

将 CeceliaPage 从线性堆叠改为对话式简报模式（Conversational Briefing）。Core 新增 task dispatch API 和 desire acknowledged 状态。Workspace 重写 CeceliaPage.tsx 为三区布局。

## 成功标准

- Migration 076 正确执行，desires 支持 acknowledged 状态
- POST /api/brain/tasks/:id/dispatch 可派发 queued 任务
- CeceliaPage 首屏 30 秒内看到简报+决策+待确认任务
- [派发] 按钮调 dispatch API，[了解] 按钮调 acknowledge API
- 不调用 opening LLM，不自动滚到底部
- CI 通过
