## PRD - 任务生命周期 Actions 扩展

**需求来源**: 当前 ACTION_WHITELIST 缺少任务暂停、恢复、阻塞标记和隔离等操作，LLM 遇到这些场景时无法精确表达意图
**功能描述**: 在 thalamus.js ACTION_WHITELIST 和 decision-executor.js actionHandlers 中添加 4 个任务生命周期 actions：pause_task、resume_task、mark_task_blocked、quarantine_task
**涉及文件**:
- brain/src/thalamus.js（ACTION_WHITELIST 新增 4 个 actions）
- brain/src/decision-executor.js（新增 4 个 action handlers）
- brain/src/__tests__/decision-executor.test.js（新增 4 个单元测试）
- DEFINITION.md（action_count 同步更新，action 列表同步）
**成功标准**:
- ACTION_WHITELIST 包含 pause_task、resume_task、mark_task_blocked、quarantine_task
- 每个 action 有对应执行逻辑（handler）
- facts-check.mjs 通过（action_count 同步更新，DEFINITION.md 同步）
- 4 个单元测试通过
**非目标**: 不修改 quarantine 模块的底层实现，不添加新的 HTTP 端点，不修改现有 action 行为
