# PRD: Dashboard 内容制作页面

## 需求
在 Dashboard 中新增"内容制作"页面，员工可以：
1. 选择内容类型（content_type）
2. 输入关键词和要求
3. 一键发起 content-pipeline
4. 实时查看 pipeline 6 步进度
5. 查看历史 pipeline 列表和状态

## 成功标准

1. [ARTIFACT] content feature manifest 注册到 coreFeatures
2. [ARTIFACT] ContentPipelinePage 组件存在且可渲染
3. [BEHAVIOR] 点击"发起制作"调用 POST /api/brain/pipelines 创建任务
4. [BEHAVIOR] 页面轮询 GET /api/brain/pipelines 显示 pipeline 列表
5. [BEHAVIOR] 点击 pipeline 查看 6 步进度（GET /api/brain/pipelines/:id/stages）
6. [PRESERVE] 现有 Dashboard 页面不受影响
7. [GATE] 所有现有测试通过
