# Task Card: Brain API 图片匹配 — 同时扫描 dan-koe 和 dankoe 前缀

## 任务信息
- **Branch**: cp-03300752-0d35b20c-32d6-4bc5-9b99-26264e
- **类型**: fix
- **文件**: packages/brain/src/routes/content-pipeline.js

## 问题描述
图片扫描时，`topic` 由 keyword 转换而来（连字符保留），导致文件名前缀为 `dankoe-` 的图片无法被 `dan-koe-` 前缀匹配到。

## 解决方案
在 `filter` 中同时检查两种前缀：
1. `topic-`（原始，保留连字符）
2. `topicNoDash-`（去除所有连字符）

## 成功标准
- image filter 同时使用 `topic-` 和 `topicNoDash-` 两种前缀匹配
- `dan-koe` 关键词能匹配 `dan-koe-*.png` 和 `dankoe-*.png` 两种文件
