# Task Card: 更新 credentials SKILL.md — 1Password 优先流程

## 需求（What & Why）
**功能描述**: 将 credentials skill 的凭据管理流程从"直接写 ~/.credentials/"改为"1Password 优先，sync 到本地"。
**背景**: 当前 SKILL.md 指引是直接存 ~/.credentials/，导致新凭据经常只在本地，1Password 里缺失。正确流程是 1Password 是 SSOT，~/.credentials/ 只是缓存。
**不做什么**: 不改 bash-guard、不动 1password.env bootstrap 逻辑。

## 成功标准

1. [ARTIFACT] credentials SKILL.md 不再包含"cat >> ~/.credentials/"的旧存储指引
2. [BEHAVIOR] SKILL.md 包含"1Password 先存，sync-credentials.sh 同步"的流程说明
3. [GATE] CI 全部通过
