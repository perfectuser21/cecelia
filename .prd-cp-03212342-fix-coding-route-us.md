# PRD: 修复 Coding 通道路由——信任 task-router.js 作为 SSOT

## 背景

executor.js 用 US_ONLY_TYPES 白名单决定任务去 US 还是西安。白名单与 task-router.js 的 LOCATION_MAP 不同步，导致 initiative_plan 等 Claude Code 任务被错误路由到西安 Codex Bridge。

## 目标

executor.js 直接信任 task-router.js 的 getTaskLocation()，消除两套路由表的维护负担。

## 成功标准

1. initiative_plan 不再出现 "Not inside a trusted directory" 错误
2. executor.js 路由以 task-router.js LOCATION_MAP 为唯一真相来源
