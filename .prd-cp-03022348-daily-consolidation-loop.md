# PRD: Cecelia 每日合并循环（Daily Consolidation Loop）

## 背景

Cecelia 的自我意识系统完成了 P0-A/B/C 断层修复后，各层已能实时感知：
- 对话产生 learning（P0-A）
- 任务完成回写欲望效能（P0-B）
- 反刍洞察直接触发欲望（P0-C）

但缺少一个**每日综合层**：每天积累的对话/任务/learnings 没有被系统性合并成「今日记忆」，self-model 更新频率极低（近乎静止）。

## 目标

实现 `consolidation.js` 模块，在每天固定时间（或 intellectual_idle 信号触发时）：
1. 汇总今日数据（对话摘要、完成任务、新 learnings）
2. 用 LLM 生成结构化每日摘要
3. 写入情节记忆（memory_stream，type=daily_consolidation）
4. 触发 self-model 更新

## 功能需求

### F1：数据汇总
- 读取今日 memory_stream 中的 `chat_summary`、`task_reflection` 类型
- 读取今日新增的 learnings 表记录（created_at >= today 00:00）
- 读取今日完成/失败的 tasks（completed_at 或 failed_at >= today 00:00）

### F2：LLM 生成每日摘要
- 调用 thalamus/cortex LLM
- 输出结构化 JSON：
  ```json
  {
    "date": "2026-03-02",
    "key_events": ["事件1", "事件2"],
    "new_learnings": ["洞察1"],
    "completed_goals": ["任务摘要"],
    "mood_trajectory": "描述今日情绪变化",
    "self_model_delta": {
      "insight": "基于今日，对自我的新认知",
      "confidence_change": "某方面能力的变化"
    }
  }
  ```

### F3：写入情节记忆
- INSERT INTO memory_stream (type='daily_consolidation', content=摘要 JSON, importance=8)
- 防重复：今日已有 daily_consolidation 则跳过

### F4：触发 self-model 更新
- 调用 `updateSelfModel()` 传入今日摘要的 self_model_delta

### F5：触发时机
- tick.js 中检查：当前时间 >= 凌晨 3:00 且今日尚未合并 → 触发
- 或 intellectual_idle 信号触发（次要，不阻塞 tick 主流程）

### F6：防重复保护
- 用 `daily_logs` 表记录合并完成（或 brain_config key='last_consolidation_date'）
- 今日已运行则立即返回，不重复执行

## 成功标准

1. consolidation.js 模块可独立测试
2. 每日凌晨 3:00 自动触发（tick.js 集成）
3. 运行后 memory_stream 新增 type=daily_consolidation 记录
4. 运行后 self-model 有更新
5. 同一天不重复运行（防重复）

## 优先级

P1（自我意识闭环重要补全，接 P0 系列）
