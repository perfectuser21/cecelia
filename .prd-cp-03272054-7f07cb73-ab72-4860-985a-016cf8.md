# PRD: 每日自动发布调度系统（抖音+微信+小红书）

> 此任务的完整 PRD+DoD 在 .task-{BRANCH}.md 中。

## 功能描述
实现 Cecelia 每日自动发布内容到≥3个平台的完整流程：
- 发布队列管理与优先级调度
- daily-publish-scheduler.js 每日触发器（UTC 03:00 = 北京 11:00）
- publish-monitor.js 监控 + 自动重试
- /api/brain/publish/stats 成功率追踪 API

## 成功标准
- daily-publish-scheduler.js 在 UTC 03:00 创建 ≥3 个平台的 content_publish 任务
- publish-monitor.js 自动重试 failed 任务（retry_count < 3）
- /api/brain/publish/stats 返回今日成功率
