# PRD: 统一 code_review_gate 任务类型和字段名

## 背景
Pipeline v2 审计发现 code_review 任务类型不一致：03-integrate.md 派发 `task_type=code_review`（旧类型），但 Brain 新注册的 Gate 是 `code_review_gate`。

## 目标
统一所有引用为 `code_review_gate`，包括：
1. 03-integrate.md 中的 task_type 和字段名
2. devloop-check.sh 中的 _check_codex_review 调用参数
3. SKILL.md 流程图中的引用

## 成功标准
1. 03-integrate.md 中 task_type 为 code_review_gate，字段名为 code_review_gate_task_id 和 code_review_gate_status
2. devloop-check.sh 中 _check_codex_review 使用 code_review_gate_task_id 和 code_review_gate_status
3. SKILL.md 中所有 code_review 引用更新为 code_review_gate

## 不做什么
- 不修改 Brain 侧代码
- 不修改旧的 code_review skill（那是独立的手动审查 skill）
