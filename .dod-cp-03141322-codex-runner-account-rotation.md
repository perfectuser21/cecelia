# DoD：Codex Runner 账号轮换

## 验收清单

- [x] runner.sh 读取 `CODEX_HOMES`（冒号分隔），若未设置则降级到单一 `CODEX_HOME`
  Test: manual:bash -c "grep -c 'CODEX_HOMES' packages/engine/runners/codex/runner.sh"

- [x] runner.sh 在检测到 `Quota exceeded` 时切换到下一个账号，而非直接失败
  Test: manual:bash -c "grep -c 'Quota exceeded' packages/engine/runners/codex/runner.sh"

- [x] 所有账号都耗尽时，任务才真正失败（输出明确错误信息）
  Test: manual:bash -c "grep -c '所有账号' packages/engine/runners/codex/runner.sh"

- [x] 账号切换时输出日志（账号序号、新 CODEX_HOME 路径）
  Test: manual:bash -c "grep -c '账号.*切换\|切换.*账号' packages/engine/runners/codex/runner.sh"

- [x] runner.sh 版本号更新到 v2.3.0
  Test: manual:bash -c "grep -c 'v2.3.0' packages/engine/runners/codex/runner.sh"
