# PRD - quarantine last_error 记录修复

**需求来源**: Brain 自动调度 / agent_ops 领域
**功能描述**: 修复隔离任务 last_error 字段始终为 "Unknown" 的问题，在 cecelia-run / executor 失败回调中正确捕获并写入真实错误信息。
**涉及文件**:
- `packages/brain/src/executor.js` — 失败回调处理
- `packages/brain/src/cecelia-run.js` 或 `scripts/cecelia-run.sh` — run 脚本错误捕获
- `packages/brain/src/routes/` — 相关 API 路由（若有）

**非目标**: 不修改隔离区判定逻辑，不改变 quarantine_info 其他字段结构
