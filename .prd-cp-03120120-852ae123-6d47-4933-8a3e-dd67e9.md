# PRD - 验证 PR #874 billing pause 熔断端到端路径

**需求来源**: Brain 自动调度（task_id: 852ae123-6d47-4933-8a3e-dd67e98e6eb5，type: verification）
**功能描述**: 编写并执行验证脚本，确认 quota_exhausted → setBillingPause → tick 熔断 → requeue 完整路径在运行时正确工作
**涉及文件**: packages/brain/tests/integration/billing-pause-e2e.test.js（新建）
**成功标准**: 见下文
**非目标**: 不修改 billing pause 核心逻辑代码，不修改 tick.js / executor.js

## 验证目标

PR #874 已合并（commit 8c3a2a006），Brain 已重启（PR #876）。

需要验证以下完整路径：
1. `/api/brain/billing-pause` 端点可用，初始 active=false
2. quota_exhausted 回调 → setBillingPause active=true
3. tick 执行时 dispatch 被跳过（billing pause 生效）
4. 清除 pause → quota_exhausted 任务自动 requeue 为 queued

## 成功标准

1. [ARTIFACT] 验证测试文件存在 packages/brain/tests/integration/billing-pause-e2e.test.js
2. [BEHAVIOR] billing-pause API 返回 `{ success: true, active: false }` 初始状态
3. [BEHAVIOR] 触发 quota_exhausted 回调后，billing-pause active 变为 true
4. [BEHAVIOR] 触发 tick 后日志/响应中出现 billing pause 熔断信号（dispatch 被跳过）
5. [BEHAVIOR] 调用 DELETE /api/brain/billing-pause 后，任务状态从 quota_exhausted requeue 为 queued

## 异常处理

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| Brain 未运行 | 测试跳过，输出明确错误 | P2 | brain_not_running |
| billing-pause 端点不存在 | 测试失败，报告 404 | P1 | billing_pause_api_missing |
