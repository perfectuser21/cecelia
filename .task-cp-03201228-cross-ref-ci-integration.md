# Task Card: cross-ref-registry 检查器实现 + CI L2 接入

## 需求（What & Why）

**功能描述**: 实现 scripts/check-cross-ref.mjs 跨文件引用一致性检查器，并接入 CI L2 Consistency Gate。

**背景**: cross-ref-registry.yaml 已定义了 4 条 SSOT→消费者引用关系规则，但没有检查器来验证。PR #1167 暴露的步骤命名不一致问题就是因为缺少这个检查。

**不做什么**: 不改 registry.yaml 的规则定义，只实现检查器 + CI 集成。

## 成功标准

1. [ARTIFACT] scripts/check-cross-ref.mjs 存在且能执行
2. [BEHAVIOR] check-cross-ref.mjs 在全一致时 exit 0
3. [BEHAVIOR] CI L2 新增 cross-ref-check job
4. [GATE] CI 全通过

## 验收条件（DoD）

- [x] [ARTIFACT] check-cross-ref.mjs 存在
  Test: manual:node -e "require('fs').accessSync('scripts/check-cross-ref.mjs');process.exit(0)"
