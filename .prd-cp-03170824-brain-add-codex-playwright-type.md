# PRD: Brain DB — 补充 codex_playwright task type

## 背景

PR #998 已合并，向 task-router.js 注册了 `codex_playwright` 任务类型，
但遗漏了对应的 DB migration，导致 tasks 表的 check constraint 仍不允许此类型，
Brain 创建 `codex_playwright` 任务时报错：
`tasks_task_type_check constraint violation`

## 目标

补一条 migration（154），将 `codex_playwright` 加入 tasks.task_type check constraint，
同时更新 selfcheck.js EXPECTED_SCHEMA_VERSION 到 '154'。

## 成功标准

- Brain API 能成功创建 task_type=codex_playwright 的任务
- selfcheck 通过（schema version >= 154）
- CI 全绿
