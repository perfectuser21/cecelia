# PRD: OKR 飞轮 Brain 侧 — okr-closer.js + tick 集成

## 背景

当前 OKR 系统（okr_initiatives / okr_scopes / okr_projects）缺少完成检测飞轮机制。旧系统（projects 表）有 initiative-closer.js，新 okr_* 表尚无对应实现，导致递进式飞轮断链。

## 需要实现

1. 新建 `packages/brain/src/okr-closer.js`：
   - `checkOkrInitiativeCompletion(pool)`：检测 okr_initiatives 下所有 tasks 完成 → 更新 initiative status=completed → 创建 okr_scope_plan 任务
   - `checkOkrScopeCompletion(pool)`：检测 okr_scopes 下所有 initiatives 完成 → 更新 scope status=completed → 创建 okr_project_plan 任务
   - `checkOkrProjectCompletion(pool)`：检测 okr_projects 下所有 scopes 完成 → 更新 project status=completed
2. 在 `tick.js` 集成三个检测函数
3. 在 `task-router.js` 添加新路由：okr_initiative_plan / okr_scope_plan / okr_project_plan → /decomp
4. 在 `thalamus.js` ACTION_WHITELIST 添加新 okr 完成事件动作

## 成功标准

- okr-closer.js 文件存在且包含三个检测函数
- tick.js 调用 okr-closer.js 的三个函数
- task-router.js 包含三个新 task_type 路由
- thalamus.js ACTION_WHITELIST 包含 okr_initiative_completed
