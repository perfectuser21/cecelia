# PRD: Brain API 图片匹配 — 同时扫描 dan-koe 和 dankoe 前缀

## 背景
content-pipeline.js 图片扫描时，topic 由 keyword 转换而来（连字符保留），导致文件名前缀为 `dankoe-` 的图片无法被 `dan-koe-` 前缀匹配到。

## 功能描述
在 image filter 中同时检查 `topic-` 和 `topicNoDash-` 两种前缀，使 dan-koe 也能匹配 dankoe-*.png。

## 文件
`packages/brain/src/routes/content-pipeline.js`

## 成功标准
- image filter 同时使用 `topic-` 和 `topicNoDash-` 两种前缀匹配
- `dan-koe` 关键词能匹配 `dan-koe-*.png` 和 `dankoe-*.png` 两种文件
