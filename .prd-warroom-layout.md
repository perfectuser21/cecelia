# PRD: War Room 改为横向多列 dashboard 布局

## 背景

GTDWarRoom 当前是竖向单列滚动布局（`max-w-4xl space-y-6`），在宽屏下浪费空间，且用户反馈"上下刷不动"（容器高度限制导致滚动失效）。

## 目标

将 War Room 改为横向多列 dashboard 布局，充分利用宽屏空间，各列独立滚动。

## 成功标准

- War Room 页面显示为：顶部 Vision 横幅 + 下方三列（Objectives+KR / Tasks / 备用列）
- 页面撑满容器高度（h-full），不依赖外层滚动
- 各列独立 overflow-y-auto 滚动，互不干扰
- 窄屏（<768px）降级为单列布局
