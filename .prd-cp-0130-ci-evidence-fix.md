# PRD: Release-check Evidence 生成修复

## 背景

release-check CI job 运行 L3 回归测试（run-regression.sh release）时，C8-001 RCI 检查失败。
原因是 release-check job 做了 fresh checkout，没有 test job 生成的 .quality-evidence.*.json 文件。

## 问题分析

1. test job 生成 evidence 文件
2. release-check job 做 fresh checkout，文件不存在
3. run-regression.sh 的 C8-001 检查找不到文件，报错

## 修复方案

在 release-check job 的 L3 Regression Tests 之前添加 Generate Evidence 步骤。
复用 ci/scripts/generate-evidence.sh 生成 evidence 文件。

## 验收标准

- [x] release-check job 包含 Generate Evidence 步骤
- [x] C8-001 RCI 检查能找到 evidence 文件
- [x] develop → main PR CI 通过
