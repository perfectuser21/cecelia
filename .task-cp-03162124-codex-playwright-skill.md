# Task Card: codex_playwright 任务类型

## 需求（What & Why）

**功能描述**: 新增 `codex_playwright` task_type，让 Brain 能派发 Playwright 自动化任务到西安 M4 的 Codex agent，Codex 用大模型探索并写出能跑通的 Playwright .cjs 脚本，脚本通过 CDP 控制西安 PC（100.97.242.124:19225）。

**背景**:
- 现有 `codex_qa`/`codex_dev` 都走西安 M4，但没有 Playwright 自动化能力
- 现有发布脚本（weibo、小红书）手写 CDP 繁琐，Playwright 封装更友好
- 目标：Codex 用大模型一次性探索出可用脚本，后续直接 `node script.cjs` 跑，不再调 LLM

**两阶段工作流**:
- Phase 1（探索）: Codex + 大模型写 Playwright .cjs，反复测试直到跑通
- Phase 2（执行）: 直接运行已保存的 .cjs，不调 LLM

**不做什么**:
- 不安装 Playwright 到 M4（本 PR 只做代码注册，安装是运维操作）
- 不修改现有 CDP 脚本（保持向后兼容）
- 不做 Phase 2 调度逻辑（本期只做 Phase 1 基础设施）
