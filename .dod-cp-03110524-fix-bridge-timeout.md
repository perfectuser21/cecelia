# DoD: 修复 cecelia-bridge Math.min 截断

## 功能验收

- [x] `cecelia-bridge.js` line 74 的 `Math.min` 上限从 `BRIDGE_TIMEOUT_MS`(120s) 改为 `MAX_BRIDGE_LLM_TIMEOUT_MS`(600s)
  Test: manual:bash -c "grep -c 'CECELIA_BRIDGE_MAX_TIMEOUT_MS' packages/brain/scripts/cecelia-bridge.js"

- [x] 新增 `CECELIA_BRIDGE_MAX_TIMEOUT_MS` 环境变量，默认 600000ms（10 分钟），作为安全上限
  Test: manual:bash -c "grep -c 'MAX_BRIDGE_LLM_TIMEOUT_MS' packages/brain/scripts/cecelia-bridge.js"

- [x] 未传 timeout 时行为不变：使用 `BRIDGE_TIMEOUT_MS`（默认 120s）
  Test: manual:bash -c "grep -c 'timeout || BRIDGE_TIMEOUT_MS' packages/brain/scripts/cecelia-bridge.js"

## 测试验收

- [x] timeout=300000 时 bridge 不截断（期望 Math.min(300000, 600000) = 300000）
