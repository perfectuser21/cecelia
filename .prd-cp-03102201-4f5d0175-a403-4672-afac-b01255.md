## PRD - feat(brain): 对比报告 Notion 推送端点

**需求来源**: Brain 自动调度任务
**功能描述**: 新增 `POST /api/brain/projects/compare/report/push` 端点，支持将项目对比报告推送到 Notion
**涉及文件**:
- `packages/brain/src/project-compare.js` — 新增 `pushCompareReportToNotion` 函数
- `packages/brain/src/routes/task-projects.js` — 注册推送路由
- `packages/brain/src/__tests__/project-compare-push.test.js` — 新增单元测试

**非目标**: 不支持除 Notion 之外的其他推送目标

### 端点规范

```
POST /api/brain/projects/compare/report/push
Body: {
  project_ids: string[],         // 必填，至少 2 个项目 UUID
  destination: "notion",          // 当前只支持 "notion"
  format?: "markdown" | "json",   // 默认 "markdown"
  notion_parent_id?: string       // Notion 父页面 ID，缺省读 env.NOTION_COMPARE_PARENT_ID
}
```

响应（成功）：
```json
{ "success": true, "notion_url": "https://notion.so/...", "page_id": "..." }
```

响应（失败）：
```json
{ "success": false, "error": "...", "code": "NOTION_CONFIG_MISSING" }
```

### 实现细节

**`packages/brain/src/project-compare.js`** — 新增导出函数：
```js
export async function pushCompareReportToNotion({ project_ids, format, notion_parent_id })
```

内部流程：
1. 调用 `generateCompareReport({ project_ids, format: markdown })` 生成 Markdown
2. 读取 `process.env.NOTION_API_KEY`，若缺失抛出 `{ status: 400, code: NOTION_CONFIG_MISSING }`
3. 读取 `notion_parent_id`（参数优先，否则 `NOTION_COMPARE_PARENT_ID` 环境变量），若两者均缺失抛出 400
4. 调用 Notion API `POST /pages` 创建新页面
5. 返回 `{ success: true, notion_url, page_id }`

复用 `notion-sync.js` 的 `notionRequest` 工具函数。

### 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| NOTION_API_KEY 缺失 | 返回 400 + code: NOTION_CONFIG_MISSING | P2 | notion_config_missing |
| notion_parent_id 缺失 | 返回 400 + 错误说明 | P2 | notion_parent_missing |
| project_ids 少于 2 | 返回 400 + 参数校验错误 | P3 | invalid_project_ids |
| destination 非 notion | 返回 400 + 不支持提示 | P3 | unsupported_destination |
| Notion API 调用失败 | 透传 error.message，返回 500 | P1 | notion_api_error |

## 成功标准

1. `POST /api/brain/projects/compare/report/push` 端点存在，`destination=notion` 时创建 Notion 页面并返回 `notion_url`
2. `NOTION_API_KEY` 或 `notion_parent_id` 缺失时返回有意义的 400 错误
3. 所有新增单元测试通过
4. Brain CI 绿（无新增失败）
