# Task Card: Pipeline Integrity Gate — CI pipeline 安全属性元检查

## 需求（What & Why）
**功能描述**: 为 CI pipeline 新增安全属性元检查测试，确保 pipeline 安全不变量不会因代码变更而静默退化
**背景**: /dev 工作流的安全性依赖多个不变量（fail-closed、无上限孤儿处理、gate 链完整性、seal 格式），这些属性曾存在 bug，修复后缺乏回归测试保护
