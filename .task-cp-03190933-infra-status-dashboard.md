# Task Card: Dashboard 机器状态页面（Fleet Monitor）

## 需求（What & Why）
**功能描述**: 在 Dashboard 的 /system/infra 页面新增 "Fleet" 标签页，展示所有 7 台设备（美国 Mac mini M4、美国 VPS、香港 VPS、西安 Mac mini M1、西安 Mac mini M4、西安 PC、NAS）的实时状态，包括 CPU 负载、内存、磁盘、在线状态。

**背景**: 当前 /system/infra 只监控本机（美国 Mac mini），缺乏对整个基础设施集群的全局视图。用户需要一个统一面板查看所有设备健康状况。

**不做什么**: 不做历史趋势图、不做告警通知、不做远程操作。

## 成功标准

1. [ARTIFACT] Brain 新增 `/api/brain/infra-status/servers` API 端点，返回所有设备状态
2. [BEHAVIOR] 访问 /system/infra → Fleet 标签页显示 7 台设备的状态卡片（CPU/内存/磁盘/在线）
3. [BEHAVIOR] 离线设备显示为 offline 状态（SSH 超时 5s 即判离线）
4. [GATE] CI 全部通过
