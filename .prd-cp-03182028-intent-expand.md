# PRD: intent-expand skill

## 目标

实现 `/intent-expand` skill，供 Brain 派发的 `intent_expand` 任务执行。

Skill 沿 Task → Project → KR → OKR → Vision 层级链条查询 Brain API，将层级上下文与原始 PRD 合并，生成 enriched PRD。

## 背景

Brain 现已支持 `intent_expand` task_type（`task-router.js` 中的 `LOCATION_MAP` 已包含），但缺少对应的执行 skill 文件。

## 成功标准

1. `packages/engine/skills/intent-expand/SKILL.md` 存在，包含可执行的 Step 1-4 流程
2. Skill 能通过 Brain API 查询层级上下文（`/api/brain/tasks/:id`、`/api/brain/projects/:id`、`/api/brain/goals/:id`）
3. enriched PRD 格式清晰：包含 Vision/OKR/KR/Project 上下文 + 补全后的需求 + 原始 PRD
4. 输出写入 `.enriched-prd-<branch>.md` 文件
