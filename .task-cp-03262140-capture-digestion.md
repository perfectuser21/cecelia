# Task Card: feat(capture) — Capture Digestion 基础设施（PR1: 建表+API）

## 需求（What & Why）

**功能描述**: 为 Capture Digestion Pipeline 建立数据库基础设施和 API 层。capture_atoms 表存放 AI 拆解后的原子事件，events 表存放生活事件。

**背景**: captures 表（PR #1597）已就位，用户输入的原始内容进入 inbox。下一步需要 capture_atoms 表来存放拆解结果，以及 events 表作为第 6 条路由目标。

**不做什么**: Brain digestion job（AI 拆解）、Review UI（前端）、自动路由逻辑。

## 成功标准

1. capture_atoms 表存在，含 capture_id/content/target_type/target_subtype/suggested_area_id/status/routed_to_table/routed_to_id/confidence 字段
2. life_events 表存在，含 name/date/event_type/location/people/area_id/description/capture_atom_id 字段
3. GET /api/capture-atoms 返回 200 + JSON 数组
4. POST /api/capture-atoms 创建 atom 返回 201
5. PATCH /api/capture-atoms/:id confirm/dismiss 返回 200
6. GET /api/life-events 返回 200 + JSON 数组
7. POST /api/life-events 创建 event 返回 201
8. Brain 迁移文件已创建（199_capture_atoms_events.sql）
