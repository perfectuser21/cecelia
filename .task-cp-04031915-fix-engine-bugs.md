# Task Card: 修复 Engine 3 个 Bug

## 需求（What & Why）
**功能描述**: 修复 packages/engine 中通过深度代码审计发现的 3 个真实 bug（P0×2 + P1×1）
**背景**: 代码审计发现 cleanup 路径查找失败、凭据占位符误判、worktree cleanup 命令失效
**不做什么**: 不修改 P2 级别小问题（路径前缀、sed 兼容、jq 错误处理）

## 成功标准

> [ARTIFACT] 三处代码修改正确落地
> [BEHAVIOR] devloop-check 能找到 cleanup.sh
> [BEHAVIOR] hook-utils 不误判 PLACEHOLDER 格式为真实凭据
> [BEHAVIOR] worktree-manage cleanup 命令不再以 exit 1 失败

3. `packages/engine/skills/dev/scripts/worktree-manage.sh` — cmd_cleanup 内联实现，移除对 worktree-gc.sh 的依赖

**受影响函数**:
