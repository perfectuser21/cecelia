# Task Card: 建 system-registry（总表 + 分系统详情）

## 需求（What & Why）
**功能描述**: 将单体 feature-registry.yml（38 feature、779行）拆分为模块化结构：system-registry.yml 总表 + features/*.yml 分系统详情
**背景**: v14.0.0 删除了旧 devgate 体系，feature-registry.yml 虽已恢复但引用了不存在的 CI 层（L3/L4）。需要重建为准确、可维护的模块化 registry，为后续 CI registry-lint 做基础
**不做什么**: 不写 CI registry-lint job（下个任务），不改现有代码，不新增 feature

## 成功标准
> system-registry.yml 作为总表索引所有系统，features/*.yml 按系统拆分详情，所有 code_path 指向真实存在的文件
