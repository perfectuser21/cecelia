# Task Card: Dev Pipeline Codex 独立审查加固

## 需求（What & Why）
**功能描述**: 修复 /dev pipeline 中 Codex 独立审查从未真正生效的 5 个问题
**背景**: spec_review 和 code_review_gate 设计上是独立第三方审查（防止"说A做0.3A"），但实际上执行者是 claude（非 codex）、prompt 只有标题（无内容）、资源池未隔离、slot 数字不一致、skill 依赖不存在的 PR diff。
**不做什么**: 不改 devloop-check 的超时逻辑、不改 Brain 任务状态机、不改 CI 流程

## 成功标准

1. [BEHAVIOR] spec_review/code_review_gate/prd_review/initiative_review 由 Codex CLI 执行
2. [BEHAVIOR] spec_review prompt 包含完整 Task Card 文件内容
3. [BEHAVIOR] code_review_gate prompt 包含完整 git diff origin/main..HEAD
4. [BEHAVIOR] review 任务走独立 /tmp/codex-review-locks/ 池（MAX=2），不占动态池
5. [BEHAVIOR] Brain AUTO_DISPATCH_MAX=10，cecelia-run MAX_CONCURRENT=10，总计12
6. [BEHAVIOR] code-review-gate SKILL.md 用 git diff 替代 gh pr diff
7. [GATE] CI 全部通过
