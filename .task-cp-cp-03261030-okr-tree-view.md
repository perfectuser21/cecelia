# Task Card: OKR 全树视图

## 需求（What & Why）
**功能描述**: 扩展 GET /api/brain/okr/tree API，返回 objectives→key_results→okr_projects→okr_scopes→okr_initiatives→tasks 完整嵌套结构。前端新增 /okr/tree 页面，可展开/收起每层节点，点击节点显示状态/进度。

**背景**: 当前 /api/brain/okr/tree 只返回 Vision→Objectives→KRs 三层，缺少 Projects→Scopes→Initiatives→Tasks 四层。需要完整树形视图来追踪 OKR 全链路进度。

**不做什么**: 不改 schema，复用现有表（visions, objectives, key_results, okr_projects, okr_scopes, okr_initiatives, tasks）。

## 成功标准

- GET /api/brain/okr/tree 返回完整 7 层嵌套 JSON（Vision→Objective→KR→Project→Scope→Initiative→Task）
- 前端 /okr/tree 页面可展开/收起每层节点，展示状态和进度信息
- 复用现有数据库表，不改 schema
