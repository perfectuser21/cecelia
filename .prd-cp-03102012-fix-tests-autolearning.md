# PRD：修复 focus/pr-progress 测试隔离 + auto-learning error_message

## 背景

L4 brain-integration CI 当前 baseline 125，其中：
- `focus.test.js`：16 个失败（mock 隔离问题）
- `pr-progress.test.js`：7 个失败（mock 隔离问题）
- `auto-learning.js`：SELECT 缺 `error_message` 字段，导致自动学习无法分析失败原因

## 目标

1. 修复 focus.test.js 和 pr-progress.test.js 的 mock/模块隔离问题，让这 23 个测试在 L4 中通过
2. 在 auto-learning.js 的 SELECT 查询中加入 error_message 字段
3. L4 baseline 从 125 降至 102（或更低）

## 验收条件

- focus.test.js 所有测试本地通过
- pr-progress.test.js 所有测试本地通过

## 成功标准
