# PRD: CI Gate — 假 DoD Test 检测

branch: cp-03231442-ci-gate-fake-dod-test
created: 2026-03-23

## 背景

verify-step.sh 已有基础假测试检测（echo/ls/cat），但缺少 `grep|wc` 等模式。
更重要的是 CI 完全没有对应检查——假测试能通过本地 verify-step 后直接混入 CI。

## 目标

1. verify-step.sh：扩展假测试正则，加入 `grep.*|.*wc`、`wc -l`、`printf` 等模式
2. 创建共享脚本 `packages/engine/scripts/devgate/check-fake-dod-tests.cjs`
3. CI L1 新增 `fake-dod-test-check` job，使用共享脚本扫描任务卡

## 成功标准

- verify-step.sh 能检测出 `grep | wc -l` 型假测试
- CI L1 有 fake-dod-test-check job，stage 2+ 触发
- 共享脚本可独立运行，exit 1 on fake，exit 0 on clean
