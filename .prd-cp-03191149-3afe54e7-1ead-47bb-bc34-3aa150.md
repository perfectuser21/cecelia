# PRD: fix(brain): status.js 缺失 IDEMPOTENCY_TTL/ALLOWED_ACTIONS 导入

## 背景

`/api/brain/status` 每次调用都返回错误：`IDEMPOTENCY_TTL is not defined`

## 根因

`packages/brain/src/routes/status.js` 第68行使用 `IDEMPOTENCY_TTL / 1000`，第70行使用 `Object.keys(ALLOWED_ACTIONS)`，但 import 第6行只导入了函数，没有导入这两个常量。

两个常量都已在 `shared.js` 中定义并导出。

## 修复方案

在 status.js 的 import 语句中添加 `IDEMPOTENCY_TTL` 和 `ALLOWED_ACTIONS`。

## 成功标准

- `curl localhost:5221/api/brain/status` 返回完整 JSON 不报错
- `action_constraints.idempotency_ttl_seconds` 值为 300
- `action_constraints.allowed_actions` 列表非空
