# PRD: Tick Watchdog 测试覆盖

## 背景
Brain 自动调度的 [P0] drain 自杀修复 + tick watchdog 任务。代码改动已在 PR #1267 合并（drain 改用 postDrainCooldown、TICK_AUTO_RECOVER_MINUTES=5、startTickWatchdog）。本次补充缺失的 tick watchdog 行为测试覆盖。

## 功能描述
为 `startTickWatchdog()` / `stopTickWatchdog()` 添加单元测试，覆盖：
1. drain/alertness source 禁用 tick 时 watchdog 自动恢复
2. manual source 禁用 tick 时 watchdog 不自动恢复
3. 幂等性（重复调用不启动多个定时器）
4. stopTickWatchdog 停止后不再触发
