## PRD - feat(brain): historical_learnings schema 扩展

**需求来源**: Brain 自动调度任务
**功能描述**: 为 historical_learnings 表新增三个字段（category、root_cause_hash、occurrence_count），为后续去重归并和聚合检测奠定数据基础
**涉及文件**:
- packages/brain/migrations/NNN_add_learnings_fields.sql（新增 migration）
- packages/brain/src/selfcheck.js（EXPECTED_SCHEMA_VERSION 更新）
- packages/brain/src/actions.js（addLearning 函数签名扩展）
- packages/brain/src/cortex.js 或 thalamus.js（调用 addLearning 的地方可能需更新）

**非目标**: 现有数据回填、UI 展示变更

## 成功标准

- migration 文件存在且 SQL 可执行
- learnings 表新增 root_cause_hash VARCHAR(64) 和 occurrence_count INTEGER DEFAULT 1
- selfcheck EXPECTED_SCHEMA_VERSION 同步更新为 '143'
- addLearning() 导出函数支持 category 参数，按 root_cause_hash 去重并递增 occurrence_count

### 异常处理 & 报警规范

N/A - 仅 schema 变更和函数签名扩展，无新 IO 操作
