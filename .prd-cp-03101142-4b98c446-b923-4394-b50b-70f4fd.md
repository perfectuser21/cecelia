## PRD - 快手批量发布接入新 API 方案

**需求来源**: Brain 自动调度 — [P0 KR1] 快手发布 API 接通（OAuth + 图文上传）

**成功标准**:

  - 保留旧脚本作为备用（不删除）

**非目标**: 不修改实际 API 调用逻辑，不删除旧脚本
