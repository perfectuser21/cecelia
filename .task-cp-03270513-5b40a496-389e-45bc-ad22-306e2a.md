# Task Card: War Room 横向多列 Dashboard 布局

## 需求（What & Why）
**功能描述**: 将 `apps/api/features/gtd/pages/GTDWarRoom.tsx` 从竖向滚动改为横向多列布局：Vision 顶部横幅（全宽） + 下方三列（左：Objectives+KR / 中：进行中 Tasks / 右：空备用列），各列独立 overflow-y-auto 滚动。页面容器用 h-full flex-col 撑满父级高度。

**背景**: 当前 War Room 是竖向单列 `max-w-4xl mx-auto`，信息密度低，无法同屏查看目标和任务。改为横向多列后，左右信息并排可读性更好，适合作战室"一眼全局"的使用场景。

**不做什么**: 不改变数据获取逻辑、不改变组件接口、不新增 API 调用。

- [ARTIFACT] `apps/api/features/gtd/pages/GTDWarRoom.tsx` 不含 `max-w-4xl`、`space-y-6`（旧竖向布局类名），包含 `flex-1 flex` 三列结构
- [GATE] CI 全部通过

## 实现方案
**要改的文件**: `apps/api/features/gtd/pages/GTDWarRoom.tsx`

**Scope 锚定**:
- 外层容器: `p-6 max-w-4xl mx-auto space-y-6` → `h-full flex flex-col px-4 pt-4 pb-2 gap-3 min-h-0`
