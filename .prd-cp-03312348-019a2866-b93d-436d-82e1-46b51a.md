# PRD: [SelfDrive] 内容选题自动化 - MVP

## 背景
topic-selector.js 已存在但 LLM 路由指向 codex（当前不可用），导致每日自动选题无法实际执行；同时缺少查看选题候选库的 API，无法感知选题结果。

## 目标
修复 AI 选题 LLM 调用失败问题，并新增 GET /api/brain/topics 路由，使内容生成具备最小可用的自动化选题能力。

## 不做什么
- 不改动 content_topics 表集成
- 不改动 UI/Dashboard
- 不改动 topic_selection_log 表结构
- 不改动 topic-selector.js（无硬编码 codex）

## 成功标准

- GET /api/brain/topics 返回 HTTP 200，支持按日期过滤
- POST /api/brain/pipelines/trigger-topics 触发后 LLM 调用不报错
- llm-caller.js 支持 fallback 机制
- topic-selector.js 使用 callLLM('cortex', ...) 不硬编码 codex
