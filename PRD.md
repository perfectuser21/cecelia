# PRD: 合并审查 Skill 为 4 个 Codex Gate

## 背景

当前有 5 个审查任务类型互相重叠：decomp_review、prd_audit、cto_review、code_quality、initiative_verify。
需要合并为 4 个清晰的 Gate Skill 定义。

## 成功标准

- 创建 4 个新 SKILL.md：prd-review、spec-review、code-review-gate、initiative-review
- 每个 SKILL.md 包含：版本号、触发条件、输入参数、输出格式、审查维度
- 不修改现有 Skill 文件
- 语言全部使用简体中文
