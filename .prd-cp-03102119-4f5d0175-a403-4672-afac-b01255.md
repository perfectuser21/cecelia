## PRD - feat(dashboard): 对比报告 Notion 推送端点

**需求来源**: Brain 自动调度任务
**功能描述**: 新增 `POST /api/brain/projects/compare/report/push` 端点，支持将项目对比报告推送到 Notion 页面
**涉及文件**:
- `packages/brain/src/project-compare.js` — 新增 `pushCompareReportToNotion` 导出函数
- `packages/brain/src/routes/task-projects.js` — 注册新路由
- `packages/brain/src/__tests__/project-compare-push.test.js` — 新建单元测试文件

## 成功标准

1. `POST /api/brain/projects/compare/report/push` 端点存在，`destination=notion` 时创建 Notion 页面并返回 `notion_url`
2. `NOTION_API_KEY` 或 `notion_parent_id` 缺失时返回有意义的 400 错误
3. 所有新增单元测试通过
4. Brain CI 绿（无新增失败）

**非目标**: 不支持 notion 以外的 destination；不修改现有 `/compare/report` 端点

### 接口规范

#### 请求
```
POST /api/brain/projects/compare/report/push
Body: {
  project_ids: string[],         // 必填，至少 2 个项目 UUID
  destination: "notion",          // 当前只支持 "notion"
  format?: "markdown" | "json",   // 默认 "markdown"
  notion_parent_id?: string       // Notion 父页面 ID，缺省读 env.NOTION_COMPARE_PARENT_ID
}
```

#### 响应（成功）
```json
{ "success": true, "notion_url": "https://notion.so/...", "page_id": "..." }
```

#### 响应（失败）
```json
{ "success": false, "error": "...", "code": "NOTION_CONFIG_MISSING" }
```

### 实现流程（pushCompareReportToNotion）

1. 调用 `generateCompareReport({ project_ids, format: 'markdown' })` 生成 Markdown
2. 读取 `process.env.NOTION_API_KEY`，若缺失抛出 `{ status: 400, code: 'NOTION_CONFIG_MISSING' }`
3. 读取 `notion_parent_id`（参数优先，否则 `NOTION_COMPARE_PARENT_ID` 环境变量），若两者均缺失抛出 400
4. 调用 Notion API `POST /pages` 创建新页面：
   - `parent.page_id = notion_parent_id`
   - `properties.title` = `项目对比报告 YYYY-MM-DD`
   - `children` = 将 Markdown 拆分为 Notion blocks（段落块，每段最多 2000 字符）
5. 返回 `{ success: true, notion_url, page_id }`

### 单元测试用例

- 正常路径：mock `generateCompareReport` + `fetch`，断言返回 `{ success: true, notion_url, page_id }`
- NOTION_API_KEY 缺失 → status 400, code NOTION_CONFIG_MISSING
- notion_parent_id 两者均缺失 → status 400
- project_ids 少于 2 → status 400
- destination 非 notion → status 400
- Notion API 返回错误 → 透传 error.message

### 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| NOTION_API_KEY 缺失 | 返回 400 NOTION_CONFIG_MISSING | P3 | notion_config_missing |
| notion_parent_id 缺失 | 返回 400 | P3 | notion_parent_missing |
| Notion API 调用失败 | 透传错误，返回 500 | P2 | notion_api_error |
| project_ids 不足 | 返回 400 | P3 | invalid_project_ids |
| generateCompareReport 失败 | 透传错误 | P2 | compare_report_error |
