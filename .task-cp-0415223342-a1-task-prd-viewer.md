# Task Card: A1 — Dashboard Task PRD Viewer

## 需求

用户痛点：跟 Claude 对话讨论 PRD → Brain 里 description 字段存了 → 用户没有友好入口浏览。
解决：浏览器 `http://localhost:5211/tasks/<task-uuid>/prd` 即可看 PRD 全文。

Day 2 Epic A 第一个 PR。Plan 原估 3h，探查后降至 1.5h（Brain GET endpoint 已存在）。

## 实现方案

**新建**：`apps/dashboard/src/pages/tasks/TaskPrdPage.tsx` + `.test.tsx`
**改**：`apps/dashboard/src/App.tsx`（向 DynamicRouter 传 children Route）

**TaskPrdPage 关键点**：
- `useParams<{id:string}>()` 拿 task id
- `useEffect` fetch `/api/brain/tasks/${id}`
- 状态机：loading / error / loaded
- 渲染：header（title + status badge + priority + 时间 + PR 链接）+ body（description 用 `<pre>` 含 `whitespace-pre-wrap`）
- 错误：404 / 网络错显示用户友好提示

**App.tsx 集成**：
- DynamicRouter 已支持 children prop
- 用 `lazy` + `Suspense` 异步加载 TaskPrdPage
- 加 `<Route path="/tasks/:id/prd" element={<TaskPrdPage />} />` 作为 children
- 不需要 navigation 显示（只通过 URL 进入）

## 不做
- react-markdown（保 scope；`<pre>` 渲染足够）
- 任务列表 / 侧边栏入口
- 编辑能力（只读 viewer）
- Brain 端改动（GET endpoint 已有）
