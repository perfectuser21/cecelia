# Task Card: Migration 182 — 新 OKR 表补充运营列

## 需求（What & Why）

**功能描述**: 为新 OKR 表（visions/okr_objectives/key_results/okr_projects/okr_scopes/okr_initiatives）补充运营列，使这些表具备与旧 goals/projects 表同等的业务字段支持。

**背景**: Migration 181 已建立 goals/projects→新表的双向同步触发器，但新表缺少 description、priority、progress、starvation_score、completed_at、last_dispatch_at 等运营列，导致 ~50 个业务文件无法迁移到新表。本 PR 补充这些列并更新触发器，解锁后续代码迁移。

**不做什么**:
- 不修改任何业务 .js 代码文件
- 不添加 embedding/notion_* 列
- 不删除旧表
