# Task Card: CI 接入 DeepSeek PR 自动 Review

## 需求（What & Why）

**功能描述**：在 Cecelia monorepo 的 GitHub CI 中新增 PR 自动 review workflow，使用 `hustcer/deepseek-review@v1` action 通过 OpenRouter 接入 DeepSeek V3 模型，对每个 PR 的代码变更自动生成 AI review 评论。

**背景**：当前 PR review 完全依赖人工，AI 辅助 review 可以提前发现逻辑问题、安全风险、DoD 符合度问题，减少 review 往返次数。

**不做什么**：
- 不修改现有 CI workflow（L1-L4）
- 不要求 review 通过才能合并（advisory only）
- 不配置 GitHub secret（需用户在 repo settings 配置 OPENROUTER_API_KEY）

## 成功标准

1. `.github/workflows/pr-review.yml` 文件存在
2. workflow 在 PR opened/synchronize 时触发
3. 使用 `hustcer/deepseek-review@v1`，配置正确的 OpenRouter base-url 和模型
4. sys-prompt 要求用中文 review，关注逻辑问题、安全风险、DoD 符合度
