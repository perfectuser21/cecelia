# Task Card: 扩展所有 Brain 器官的 allowed_models 白名单

## 需求（What & Why）

**功能描述**: 在 `packages/brain/src/model-registry.js` 中，为 thalamus/cortex/mouth/memory/rumination/narrative 这 6 个大脑器官的 `allowed_models` 白名单加入新注册的 codex 和 openai 模型 ID。

**背景**: PR #1422 已成功注册了 6 个 codex/* 模型和 5 个 gpt-5.4-* 模型，但只更新了 `reflection` 器官的白名单。用户在 Dashboard `/brain-models` 页面尝试将 thalamus 切换为 `codex/gpt-5.4` 时，收到报错：`Model codex/gpt-5.4 is not allowed for agent thalamus`。

**不做什么**: 不修改 executor 层器官（dev/qa/codex_qa/codex_playwright/architect/strategy_session/intent_expand）；不修改 recommended_model；不修改 llm-caller.js。

## 成功标准

1. [ARTIFACT] thalamus/cortex/mouth/memory/rumination/narrative 的 `allowed_models` 均包含 `gpt-5.4-mini`、`gpt-5.4`、`codex/gpt-5.4-mini`、`codex/gpt-5.4`
2. [BEHAVIOR] 调用 `isModelAllowedForAgent('thalamus', 'codex/gpt-5.4')` 返回 `true`
3. [GATE] Brain 测试全部通过（npm test）
