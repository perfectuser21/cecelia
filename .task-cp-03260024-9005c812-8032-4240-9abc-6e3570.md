# Task Card: fix(gtd): full-tree 过滤 archived objectives

**背景**: GTD OKR 页面展示 Area → Objective → KR 两层结构，archived objectives 应当从视图中隐藏。

**不做什么**: 不修改 KR/Project/Scope/Initiative 的过滤逻辑（本次只聚焦 objectives 层）。

3. [GATE] CI 全部通过
