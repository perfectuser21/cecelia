## 实现方案
**要改的文件**: `packages/brain/src/cortex.js`
**Scope 锚定**: 仅修改 analyzeDeep 函数及提取的子函数，不改变其他任何代码。

**重构策略**：
将 analyzeDeep（216 行）拆解为 8 个子函数 + 1 个主函数：
1. `_buildBaseContext(event, thalamusDecision)` → 构建基础 context
2. `_enrichContextWithHistory(context)` → async，历史决策注入
3. `_enrichContextWithSystemStatus(context)` → async，系统状态注入
4. `_enrichContextWithLearnings(context, event)` → async，历史经验注入
5. `_enrichContextWithAnalyses(context, event)` → async，历史分析注入
6. `_enrichContextWithSelfModel(context)` → async，自我认知注入
7. `_enrichContextWithCrossTaskPatterns(context)` → async，跨任务失败模式
8. `_processLLMResponse(event, prompt)` → async，LLM 调用 + 后处理
9. `analyzeDeep(event, thalamusDecision)` → 主函数仅 orchestrate（约 30 行）
