# PRD: 删除 /dev SKILL.md 中废弃的"多 Feature 支持"章节

## 背景

`packages/engine/skills/dev/SKILL.md` 中有一节 `## 多 Feature 支持（可选）`，描述了 `/dev continue` 命令和 multi-feature 状态机。这个概念已废弃——多 PR 编排在 Initiative 层（/architect skill）中处理，不在 /dev 层。

保留此章节会误导使用者认为 /dev 支持 multi-feature 状态机。

## 目标

1. 删除 SKILL.md 中 `## 多 Feature 支持（可选）` 整个章节
2. 在删除位置添加一行指向 /architect skill 的注释
3. Engine 版本 bump + changelog 记录

## 成功标准

1. SKILL.md 不再包含 `## 多 Feature 支持` 标题和相关内容
2. SKILL.md 包含指向 /architect skill 的注释
3. Engine 版本成功 bump 并同步到所有版本文件
4. CI 通过
