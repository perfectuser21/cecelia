# PRD: intent-expand 和 Learning 检索集成到 Pipeline Stage 1

## 背景
Agent 执行任务时缺少"更大目标"的上下文，也不知道历史踩坑记录。需要在 Stage 1 Spec 流程中自动补充这些信息。

## 功能描述
1. 在 01-spec.md 的 Task Card 生成之前新增 intent-expand 步骤，从 Brain API 读取 KR/Initiative/Project 上下文
2. 在 Task Card 生成之后新增 Learning 检索步骤，搜索 docs/learnings/ 推荐相关历史 Learning

## 成功标准
1. 01-spec.md 包含 "## 1.1.5 上下文补充（intent-expand）" 章节
2. 01-spec.md 包含 "## 1.3.5 相关 Learning 检索" 章节
3. 章节顺序正确（1.1.5 在 1.2 前，1.3.5 在 1.3 前）
4. CI 通过
