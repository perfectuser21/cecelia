## 实现方案

**要改的文件**:
- `scripts/generate-ops-summary.mjs`（新建）

**核心 SQL 查询**:
```sql
-- 每日完成数
SELECT DATE(completed_at AT TIME ZONE 'Asia/Shanghai') as day,
       COUNT(*) as completed
FROM tasks WHERE status='completed'
  AND completed_at > NOW() - INTERVAL '8 days'
GROUP BY day ORDER BY day;

-- 高频失败类型 Top3
SELECT task_type, COUNT(*) as cnt
FROM tasks WHERE status='failed'
  AND updated_at > NOW() - INTERVAL '8 days'
GROUP BY task_type ORDER BY cnt DESC LIMIT 3;

-- Initiative 完成数（projects 表）
SELECT COUNT(*) FROM projects
WHERE type='initiative' AND status='completed'
  AND updated_at > NOW() - INTERVAL '8 days';
```

**Brain DB 连接**: 读取 Brain API（`localhost:5221/api/brain/tasks`）而非直连 DB，避免绕过 Brain 架构。

**飞书推送**: `import { sendFeishu } from '../packages/brain/src/notifier.js'`

**输出目录**: `~/claude-output/`（已有 9998 端口 HTTP 服务）
