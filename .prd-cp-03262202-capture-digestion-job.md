# PRD: feat(capture) — Brain Digestion Job（PR2: AI 拆解逻辑）

## 背景

capture_atoms 表和 CRUD API 已就位（PR #1601）。现在需要 Brain tick 中增加 digestion job，自动扫描 inbox captures，调 LLM 拆解为原子事件写入 capture_atoms。

## 目标

1. 新增 `capture-digestion.js` — Brain 后台 digestion job
2. 在 tick.js 中注册调用
3. 扫描 status=inbox 的 captures，调 LLM 拆解
4. 每个 atom 含 target_type/target_subtype/confidence/ai_reason
5. 拆解后 capture status 从 inbox → done

## 成功标准

- capture-digestion.js 文件存在于 packages/brain/src/
- tick.js 中包含 runCaptureDigestion 调用
- runCaptureDigestion 函数导出且可调用
- LLM prompt 包含 6 种 target_type 定义
- 拆解结果写入 capture_atoms 表
- capture status 更新为 done
