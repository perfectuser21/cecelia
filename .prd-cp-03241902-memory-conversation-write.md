# PRD: 记忆系统修复 — 对话写入 memory_stream（带情绪标签+显著性评分）

Brain Task ID: aee7996c-65c0-4e78-9bec-6d61c5f96f9e

## 背景

Cecelia 的记忆系统最关键的输入管道缺陷：对话写入 memory_stream 时无 salience_score/emotion_tag，
且 memory-retriever 从 cecelia_events（无 embedding）读取对话历史，无法语义检索。

## 目标

1. 为 memory_stream 表新增 salience_score 和 emotion_tag 列（migration 186）
2. orchestrator-chat.js 写入 source_type='conversation_turn' 并携带 salience_score/emotion_tag
3. memory-retriever 对话检索改用 memory_stream（带 embedding 语义搜索）

## 成功标准

- memory_stream 表有 salience_score 和 emotion_tag 列
- 对话后 memory_stream 有 conversation_turn 类型记录，带 salience_score + emotion_tag
- salience_score 按纠正→0.9/决策→0.8/疑问→0.6/普通→0.3 规则计算
- emotion_tag 从 working_memory[emotion_state] 读取，失败为 null
- 写入失败不影响主流程
- memory-retriever loadConversationHistory 从 memory_stream 读取，空时 fallback
