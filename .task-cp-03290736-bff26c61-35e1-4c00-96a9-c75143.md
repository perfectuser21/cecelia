# Task Card: Playwright Evaluator 层 — 端到端行为验证

## 需求（What & Why）

**功能描述**：在 /dev Stage 3（CI 之后）新增 Playwright Evaluator，对照 Task Card 的 [BEHAVIOR] DoD 条目逐条执行验证，覆盖 Brain API 基础端点。评估结果以详细报告形式反馈给主 agent。

**不做什么**：
