# PRD: 修复 Codex runner.sh 自动加载 CODEX_API_KEY

## 背景

codex-bin v0.114.0 使用 `CODEX_API_KEY` 环境变量（优先级高于 `OPENAI_API_KEY`）。
当前 runner.sh 未设置此变量，导致所有 Codex 任务执行时返回 401 Unauthorized。

## 目标

runner.sh 自动从 `~/.credentials/openai.env` 加载 OpenAI API key 并导出为 `CODEX_API_KEY`。

## 成功标准

- runner.sh 配置区新增 CODEX_API_KEY 加载逻辑
- 若环境变量已存在则直接使用（用户可覆盖）
- 若 credentials 文件存在则读取 OPENAI_API_KEY 字段
- 若两者均不存在则输出警告（不 exit，允许用户自行配置）
- 版本升为 v2.1.0
