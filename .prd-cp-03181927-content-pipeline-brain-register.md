# PRD: 注册5个 content-* task_type 到 Brain skillMap + executor 路由

## 背景
内容工厂 Pipeline 需要5个新 task_type：content-pipeline / content-research / content-generate / content-review / content-export。Brain executor 目前不认识这些类型。

## 目标
在 skillMap 和 executor 路由中注册5个新类型，Brain 能正确派发这些任务给对应的 skill/agent。

## 验收标准

## 成功标准
1. skillMap 中5个新 task_type 均有对应 skill 路径
2. executor 收到这些类型时不报 "unknown task_type" 错误
3. brain-register CI 检查通过
