# PRD: OKR PR11 — 清理旧表残留收尾

## 背景

PR10 合并后仍有 4 处旧表（goals/projects）实际查询残留，旧表无法删除。

## 目标

1. `packages/brain/src/actions.js` — updateGoal 最终 fallback `UPDATE goals SET` 删除，改为 `return { success: false, error: 'Goal not found' }`
2. `packages/brain/src/planner.js` — 任务创建校验 project_id 时先查旧 `projects` 表，改为只查新表（okr_initiatives/okr_scopes/okr_projects UNION ALL）
3. `packages/brain/src/services/memory-service.js` — getDetail 中查 `projects` 和 `goals` 旧表，迁移到新 OKR 表
4. `packages/brain/src/project-compare.js` — getCompareMetrics 和 generateCompareReport 查询 `projects`，迁移至新表 UNION ALL

## 成功标准

- `grep -rn "FROM goals|FROM projects|UPDATE goals|UPDATE projects|INSERT INTO goals|INSERT INTO projects" packages/brain/src/*.js packages/brain/src/services/*.js` 不返回生产代码文件结果（notion-full-sync.js 除外）
- 所有现有测试通过
