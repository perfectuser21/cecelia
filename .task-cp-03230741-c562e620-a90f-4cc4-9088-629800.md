# Task Card: CI 覆盖率门禁扩展 + L4 task-lifecycle E2E

## 需求（What & Why）

**功能描述**:
1. 扩展 L3 CI 覆盖率门禁：当前 `coverage-delta` job 只对 `feat` PR 生效，`fix:/refactor:` PR 可绕过。需将门禁扩展至 fix/refactor 类型。
2. 新增 L4 task-lifecycle E2E 测试：当前 GoldenPath 只测 Brain 启动 + 空 tick，不覆盖完整任务生命周期（创建 → dispatch → agent 接收 → 完成）。

**背景**: CI 质量保障体系中，覆盖率门禁存在漏洞——bug 修复 PR 也应保持测试覆盖，否则修复引入的变更行没有覆盖保障。

**不做什么**: 不修改覆盖率阈值，不修改 feat PR 现有逻辑，不引入需要实际 Brain 运行的集成测试（质量测试须 CI-safe）。

## 成功标准

1. [ARTIFACT] `.github/workflows/ci-l3-code.yml` 中 `coverage-delta` job 的条件包含 fix 和 refactor 类型
2. [BEHAVIOR] `packages/quality/src/task-lifecycle.test.ts` 存在，测试任务从创建到完成的状态机
3. [GATE] 所有现有 CI 全部通过
