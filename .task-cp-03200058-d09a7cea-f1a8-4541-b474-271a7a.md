# Task Card: [SelfDrive] 修正 Scanner 对内嵌能力的识别逻辑

## 需求（What & Why）
**功能描述**: 修改 Scanner 的探测机制，添加对 Brain 内嵌能力的识别（如通过 Brain API 查询内嵌服务状态、熔断器运行状态等），准确区分真实孤岛和已运行的内嵌能力。

**背景**: 当前 Scanner 未能正确识别在 Brain 进程中嵌入运行的能力（分支保护 Hooks、CI DevGate、看门狗等），误判为孤岛。需要修正后重新诊断，为后续孤岛激活决策提供准确的系统状态基线。

**不做什么**: 不改变 Scanner 的基本探测框架，只增强其对 Brain 内嵌能力的识别准确性。

## 成功标准
> [ARTIFACT] 产出物 / [BEHAVIOR] 运行时行为 / [GATE] 门禁

1. [ARTIFACT] Scanner 代码已修改，增加 Brain API 查询逻辑
2. [BEHAVIOR] 能正确识别 Brain 内嵌的分支保护 Hooks、CI DevGate、看门狗等能力
3. [BEHAVIOR] 准确区分真实孤岛和已运行的内嵌能力
4. [GATE] 所有现有测试通过，CI 全部通过
