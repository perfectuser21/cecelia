**背景**: migration 177-182 建立7张新表+同步触发器，PR1-PR7 已完成大部分迁移，本 PR 处理剩余17个文件。

- `goals WHERE type='area_okr'` → `objectives`（name 同 title）
- `goals WHERE type IN ('area_kr','global_kr','key_result')` → `key_results`
- `goals WHERE type='vision'` → `visions`
- `projects WHERE type='project'` → `okr_projects`（name→title, deadline→end_date）
- `projects WHERE type='scope'` → `okr_scopes`（name→title, parent_id→project_id）
- `projects WHERE type='initiative'` → `okr_initiatives`（name→title, parent_id→scope_id）
