## PRD - 知乎文章发布 API 接通

**需求来源**: Brain 自动调度任务 [P1 KR1] task-id: ddef213e-1e2d-4f87-9e6a-2c26b0f43ece
**功能描述**: 新增 publish-zhihu-api.cjs，对标 publish-kuaishou-api.cjs 的"CDP Cookie + in-browser fetch"模式，取代当前 CDP UI 自动化方案，更稳定不受页面改版影响。

## 背景

当前 publish-zhihu-article.cjs 使用 CDP 控制 Windows Chrome UI（点击、填表），脆弱且受页面改版影响。
对标 kuaishou-publisher/scripts/publish-kuaishou-api.cjs：
- 快手：CDP 提取 Cookie → HTTP API 直接调用
- 知乎：CDP 连接 Chrome → in-browser fetch 调用知乎内部 API（避免复杂 CSRF 计算）
