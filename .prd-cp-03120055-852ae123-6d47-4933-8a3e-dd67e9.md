## PRD - 验证 PR #874 billing pause 熔断端到端路径

**需求来源**: Brain 自动调度任务 - 验证 billing pause 熔断功能
**功能描述**: 新增端到端验证脚本，验证 quota_exhausted → setBillingPause → tick 熔断 → requeue 完整链路
**涉及文件**: packages/brain/scripts/verify-billing-pause.sh（新增验证脚本）
**成功标准**: 脚本可重复执行，验证所有 5 个步骤均通过
**非目标**: 修改现有 billing pause 实现逻辑

## 成功标准

- [ ] Step 1：/api/brain/billing-pause 返回 { active: false }
- [ ] Step 2：POST /api/brain/execution-callback (quota_exhausted) 成功
- [ ] Step 3：quota_exhausted 回调后 billing pause active=true
- [ ] Step 4：手动触发 tick 后 dispatch 被熔断（reason=billing_pause）
- [ ] Step 5：billing pause 清除后，任务自动 requeue 为 queued

### 异常处理 & 报警规范

N/A - 无 IO 操作，纯验证脚本
