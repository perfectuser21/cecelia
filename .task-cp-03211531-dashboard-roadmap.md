# Task Card: Dashboard Roadmap 页面（OKR 进度 + Now/Next/Later + Cecelia 思考）

## 需求（What & Why）
**功能描述**: 在 Dashboard 中替换旧的 RoadmapView 页面，新建一个实时展示 OKR 进度、Now/Next/Later 项目分布、Cecelia SelfDrive 思考、Agent 状态的 Roadmap 页面。
**背景**: 现有 RoadmapView 展示的是 Project/Feature/PRD 旧视图，不符合 OKR 体系。需要一个能实时反映系统状态的新 Roadmap 页面。
