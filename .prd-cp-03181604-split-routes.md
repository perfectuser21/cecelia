# PRD: Split routes.js into sub-route files

## 背景
`packages/brain/src/routes.js` 有 12,613 行，是一个不可维护的巨型文件。
需要拆分为 10 个独立子路由文件。

## 目标
将 routes.js 拆分为：
- `routes.js` — ≤200 行纯聚合器
- `routes/shared.js` — 共享工具函数
- `routes/status.js`, `routes/tasks.js`, `routes/tick.js`
- `routes/actions.js`, `routes/execution.js`, `routes/goals.js`
- `routes/analytics.js`, `routes/brain-meta.js`, `routes/ops.js`

## 约束
- 纯搬迁，不改任何 handler 逻辑
- 所有现有测试必须通过

## 成功标准
- `wc -l packages/brain/src/routes.js` 输出 ≤ 200
- `ls packages/brain/src/routes/` 有 ≥ 9 个新文件
- `cd packages/brain && npm test` 无 FAIL
