## PRD - CI 三处门禁漏洞修复

**需求来源**: CI 审查发现三处漏洞需修复
**功能描述**: 修复 3 个 CI workflow 文件中的独立问题，加强门禁严格性

**涉及文件**:
- `.github/workflows/ci-l3-code.yml` — l3-passed gate 对 workflows-l3 的 skipped 豁免漏洞
- `.github/workflows/ci-l4-runtime.yml` — DEFINITION.md 改动不应触发 L4（两处）
- `.github/workflows/ci-l2-consistency.yml` — Impact Check 缺少 devgate 脚本路径

**成功标准**

1. ci-l3-code.yml 的 l3-passed gate 对 workflows-l3 的检查，`skipped` 不再被视为通过
2. ci-l4-runtime.yml 的 push 和 PR 事件两处 changes 检测，均不含 `DEFINITION.md`
3. ci-l2-consistency.yml 的 Impact Check CORE_PATHS 包含 `packages/engine/scripts/devgate/` 和 `packages/engine/scripts/ci-evolution-check.mjs`

**非目标**: 不修改 Engine 版本号（仅 .github/workflows/ 变更，不涉及 engine 代码）

### 异常处理 & 报警规范

N/A - 无 IO 操作（纯 CI 配置文件修改）
