# Task Card: feat PR 测试覆盖强制检查

## 需求（What & Why）

**功能描述**: 在 `ci-l3-code.yml` 新增 `test-coverage-required` job，对 feat 类型 PR 强制要求包含测试文件改动。若无测试文件改动则 fail，有则 pass 并输出找到的测试文件。

**背景**: 当前 feat 类型 PR 只要老测试绿了就能通过 CI，没有验证新功能是否带测试。`detect-commit-type` job 已有 `should_run_l3` 输出（feat/feat! → true，fix/docs → false）。

**不做什么**: 不修改现有 CI 流程，不改变 fix/docs/chore 类型 PR 的行为，不引入新依赖。

## 成功标准

1. [ARTIFACT] `ci-l3-code.yml` 新增 `test-coverage-required` job，needs `detect-commit-type`，if `should_run_l3 == 'true'`
2. [BEHAVIOR] feat 类型 PR 无测试文件改动时，`test-coverage-required` job fail 并输出清晰错误信息
3. [BEHAVIOR] feat 类型 PR 有测试文件改动时，`test-coverage-required` job pass 并输出找到的文件列表
4. [ARTIFACT] Engine 版本从 12.89.0 bump 到 12.90.0（5个文件同步）
5. [ARTIFACT] `feature-registry.yml` 添加 12.90.0 changelog 条目
6. [GATE] CI 全部通过（engine-ci.yml 绿）

## 验收条件（DoD）

- [x] [ARTIFACT] `ci-l3-code.yml` 中存在 `test-coverage-required` job
  Test: manual:bash -c "grep -c 'test-coverage-required' .github/workflows/ci-l3-code.yml"

- [x] [ARTIFACT] job 有正确的 `needs` 和 `if` 条件，引用 detect-commit-type
  Test: manual:bash -c "grep -A5 'test-coverage-required:' .github/workflows/ci-l3-code.yml | grep -c 'detect-commit-type'"

- [x] [BEHAVIOR] job 的测试文件检测逻辑正确：有测试文件（.test.ts）时应通过，无测试文件时应失败
  Test: manual:node -e "const TEST_PATTERNS=/\.test\.ts$|\.spec\.ts$|\.test\.js$|\.spec\.js$/; const withTest=['src/foo.ts','src/foo.test.ts']; const noTest=['src/foo.ts','src/bar.ts']; const found=withTest.filter(f=>TEST_PATTERNS.test(f)); const notFound=noTest.filter(f=>TEST_PATTERNS.test(f)); if(found.length===0) throw new Error('有测试文件时应能检测到'); if(notFound.length>0) throw new Error('无测试文件时不应检测到'); console.log('✅ 检测逻辑正确');"

- [x] [ARTIFACT] job 包含无测试文件时的错误提示（feat PR 必须包含测试文件改动）
  Test: manual:bash -c "grep -c 'feat PR 必须包含测试文件改动' .github/workflows/ci-l3-code.yml"

- [x] [ARTIFACT] Engine 版本 5 个文件同步更新到 12.90.0
  Test: manual:node -e "const fs=require('fs'); const files=['packages/engine/VERSION','packages/engine/package.json','packages/engine/.hook-core-version','packages/engine/regression-contract.yaml']; const ok=files.every(f=>fs.readFileSync(f,'utf8').includes('12.90.0')); if(!ok) throw new Error('版本未同步'); console.log('✅ 版本同步');"

- [x] [ARTIFACT] feature-registry.yml 包含 12.90.0 changelog 条目
  Test: manual:bash -c "grep -c '12.90.0' packages/engine/features/feature-registry.yml"

- [x] [GATE] CI 配置文件存在且包含新 job
  Test: manual:node -e "const fs=require('fs'); const c=fs.readFileSync('.github/workflows/ci-l3-code.yml','utf8'); if(!c.includes('test-coverage-required')) throw new Error('job not found'); console.log('✅ job exists');"

- `packages/engine/package.json` — bump 12.89.0 → 12.90.0
