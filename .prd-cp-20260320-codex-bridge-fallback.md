# PRD: Codex Bridge 断连时降级到本机执行

## 背景
`packages/brain/src/executor.js` 的 `triggerCeceliaRun()` 函数在调用西安 Codex Bridge 失败时，直接返回 error，任务回到 queued 状态。没有降级策略——不会尝试在美国本机 cecelia-bridge 执行，导致 Codex Bridge 网络断开时所有非 US_ONLY 任务都无法执行。

## 成功标准
- Codex Bridge 调用失败后，自动降级到本机 cecelia-bridge 执行
- 只对非 US_ONLY_TYPES 的任务执行降级
- 降级时输出明确日志，包含 task_type 和错误信息
