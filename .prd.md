# PRD: Executor 注册 Run 到 Core Dashboard

## 背景
Brain executor dispatch 任务后，前端 /cecelia/runs 页面看不到执行状态，因为 executor 只写 tasks 表，不调 Core 的 /api/cecelia/runs API。

## 功能描述
在 executor.js 的 triggerCeceliaRun 中，dispatch 成功后调用 Core API POST /api/cecelia/runs 注册 run，让前端实时可见。

## 成功标准
- [ ] dispatch 后前端 /cecelia/runs 能看到新的 run
- [ ] run 包含 PRD 标题、项目名、分支信息
- [ ] 不影响原有 dispatch 逻辑
