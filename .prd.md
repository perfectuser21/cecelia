Brain 的 `tick.js` 会自动派发 `codex_qa` 类型任务，任务描述中包含脚本路径：
```
/home/xx/perfect21/cecelia/packages/quality/scripts/run-codex-immune.sh
```

但是这个脚本目前不存在，需要创建。

## 功能描述

创建 `run-codex-immune.sh` 脚本，用于执行 Cecelia 的 Codex 免疫检查。

- [ ] 脚本文件创建在正确路径

- [ ] 脚本能成功执行基本的免疫系统检查

## 技术细节

### 脚本路径
```
/home/xx/perfect21/cecelia/packages/quality/scripts/run-codex-immune.sh
```

### 功能需求

1. **健康检查**
   - 检查 Brain API (localhost:5221) 是否可达
   - 查询免疫系统状态

2. **数据检查**
   - 查询 failure_signatures 表
   - 查询 absorption_policies 表
   - 查询 policy_evaluations 表

3. **日志输出**
   - 创建日志目录 `packages/.codex-immune-logs/`
   - 生成时间戳格式日志文件
   - 记录检查结果

4. **错误处理**
   - API 不可达时返回错误
   - 数据库连接失败时返回错误
   - 所有错误都记录到日志
