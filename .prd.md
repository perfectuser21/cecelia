## PRD - 实现执行状态实时展示组件

**需求来源**: Brain 自动调度 — TaskMonitor 页面当前为占位组件，需要实现真实的执行状态实时展示
**功能描述**: 将 TaskMonitor.tsx 从占位组件升级为真实的执行状态实时展示组件。展示当前所有 Cecelia 运行中和最近的任务执行状态，包括步骤进度、checkpoint 详情、实时轮询更新。复用已有的 agents.api.ts 数据层和 CeceliaRun 类型定义。

**涉及文件**:
- `frontend/src/features/core/execution/pages/TaskMonitor.tsx` (重写 — 从占位改为真实组件)

**成功标准**:
- [ ] TaskMonitor 页面展示运行中任务的实时状态（5s 轮询）
- [ ] 显示任务列表，包含项目名、分支名、状态、进度条
- [ ] 运行中任务突出显示当前步骤和进度
- [ ] 支持点击查看任务详情（跳转到 RunDetail）
- [ ] 显示统计摘要（运行中/完成/失败数量）
- [ ] TypeScript 编译通过
- [ ] 复用已有的 shared 组件（StatusBadge, LoadingState, ErrorState）

**非目标**:
- 不实现 WebSocket/SSE（保持现有轮询机制）
- 不修改后端 API
- 不创建新的 API 层（复用 agents.api.ts）
- 不修改路由配置
