# 模型配置升级 — Agent 级别模型选择 API
## 功能描述
为每个 LLM Agent 提供单独选择模型的能力，通过 PATCH API 直接修改 active profile 的 config JSONB。
## 成功标准
- GET /api/brain/model-profiles/models 返回完整注册表
- PATCH /api/brain/model-profiles/active/agent 更新成功
- 无效参数返回明确错误
## 范围
- 模型注册表（静态数据）
- Agent 定义表（白名单、固定 provider 约束）
- PATCH API 更新单个 agent 的模型
