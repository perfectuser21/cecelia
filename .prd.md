# PRD：将 Vivian 路由从 HK 改为 US

## 背景

Vivian (微微安) 是 OKR 拆解质检员，负责审查秋米的拆解产出。
她的 SKILL.md 已迁移到 US VPS 的 `cecelia-workflows/agents/vivian/`，
模型改为 `claude-haiku-4-5-20251001`（Anthropic），不再使用 MiniMax。

但 Brain 里还有两处旧配置将她路由到 HK：
1. `task-router.js`：`decomp_review → hk`
2. `model-registry.js`（在 executor.js 内）：allowed_models 是 MiniMax

## 变更范围

### 1. task-router.js（第 51 行）
```js
// 改前
'decomp_review': 'hk',
// 改后
'decomp_review': 'us',
```

### 2. model-registry.js（decomp_review 条目）
```js
// 改前
{
  id: 'decomp_review',
  name: '拆解审查 Vivian',
  allowed_models: ['MiniMax-M2.5-highspeed', 'MiniMax-M2.5'],
  recommended_model: 'MiniMax-M2.5-highspeed',
  fixed_provider: 'minimax',
}
// 改后
{
  id: 'decomp_review',
  name: '拆解审查 Vivian',
  allowed_models: ['claude-haiku-4-5-20251001'],
  recommended_model: 'claude-haiku-4-5-20251001',
  // 移除 fixed_provider
}
```

## 验收标准

- `getTaskLocation('decomp_review')` 返回 `'us'`
- model-registry 中 decomp_review 无 fixed_provider 字段
- 现有测试全部通过
