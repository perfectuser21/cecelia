# PRD: Quality API Extensions

## 目标
为 Quality Monitor v2 添加后端 API 支持。

## 功能需求
- GET /api/queue - 获取任务队列
- GET /api/worker - 获取 Worker 状态
- GET /api/runs/:runId/evidence - 获取 run 的 evidence 文件列表
- GET /api/runs/:runId/logs/:filename - 下载 evidence 文件

## 成功标准
- [ ] API 端点正确返回数据
- [ ] 错误处理完善
- [ ] 与前端集成成功
