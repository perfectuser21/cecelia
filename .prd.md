# PRD: Slot 分配优化 — 预留缩减 + 滑动窗口 + 安全水位

## 功能描述
优化 slot-allocator 和 executor 的资源管理策略：
1. 用户预留缩减（USER_RESERVED_BASE 2->1, USER_PRIORITY_HEADROOM 2->1）
2. checkServerResources 加滑动窗口平滑（CPU 5次平均, MEM 3次最高值）
3. effectiveSlots 乘 0.85 安全水位
4. applySlotBuffer 非对称（快刹车-3/慢恢复+1）

## 成功标准
- USER_RESERVED_BASE = 1, USER_PRIORITY_HEADROOM = 1
- checkServerResources 使用滑动窗口平滑 CPU/MEM 读数
- effectiveSlots 乘以 SAFETY_MARGIN(0.85)
- applySlotBuffer 下降-3/上升+1
- 所有现有测试通过
