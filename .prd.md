# PRD: CI/CD 自动部署

## 需求来源

develop 分支 PR 合并后需要手动在 US VPS 上运行 brain-deploy.sh，无自动化机制，导致代码积压不上线。

## 功能描述

在 `.github/workflows/ci.yml` 中新增 `deploy` job：
- 触发条件：push to develop
- 依赖：brain-test 和 facts-check 通过
- 执行：SSH 到 US VPS → git pull + brain-deploy.sh
- secrets：DEPLOY_SSH_KEY / DEPLOY_HOST / DEPLOY_USER

## 成功标准

develop merge 后 GitHub Actions 自动触发部署，Brain 生产版本自动升级。
