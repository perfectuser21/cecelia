# PRD: 清理模型注册表

## 背景
模型注册表中包含已废弃/旧版模型，需要清理。

## 功能描述
1. 删除旧版 Anthropic 模型（Sonnet 2025, Opus 2025）
2. 删除 MiniMax M2.1 Fast（不存在的模型）
3. 保留 M2.1 但去掉 deprecated 标记
4. 更新所有 AGENTS 的 allowed_models 和 recommended_model 为新版 4.6 ID

## 成功标准
- GET /api/brain/model-profiles/models 只返回当前有效模型
- AGENTS 引用全部指向 4.6 版本
- npm test 通过
