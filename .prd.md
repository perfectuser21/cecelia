# PRD: Dev Pipeline Codex 独立审查加固

## 背景
spec_review 和 code_review_gate 设计上是独立第三方审查，但实际上：
- 执行者是 claude（非 codex）
- prompt 只有标题（无内容）
- spawn 参数用的是 claude 的 flag
- git diff 命令缺 ..HEAD
- SKILL.md 用 gh pr diff（Stage 2 无 PR）

## 目标
修复以上 5 个问题，让 Codex 独立审查真正生效。

## 成功标准

1. triggerCodexReview 使用 codex exec 而非 claude CLI，并回调 Brain verdict
2. spec_review prompt 含完整 Task Card 内容，code_review_gate prompt 含 git diff
3. review 任务走独立资源池（/tmp/codex-review-locks/ MAX=2），不影响 dev 并发
4. code-review-gate SKILL.md 改用 git diff origin/main..HEAD（Stage 2 无 PR）
5. CI 全部通过
