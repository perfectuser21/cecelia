# PRD: Worker 编辑 API

## 功能描述
- 新增 PUT /api/brain/staff/workers/:workerId 接口
- 支持更新 worker 的 skill 字段（写入 workers.config.json）
- 支持更新 worker 的 model 配置（写入 active model profile 的 model_map）

## 成功标准
- PUT /api/brain/staff/workers/:workerId 返回 200
- skill 变更持久化到 workers.config.json
- model 变更持久化到 active profile
- 测试通过，build 无错误
