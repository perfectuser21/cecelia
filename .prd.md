# PRD: PipelineOutputPage 重构为作品主页（四 Tab）

## 背景
现有 PipelineOutputPage 展示内容单薄，无法满足内容运营对"一个作品的全生命周期"可视化需求；需要将内容生成、发布、数据三条线聚合在同一页面。

## 成功标准
- PipelineOutputPage 组件包含四个 Tab（Summary / 生成记录 / 发布记录 / 数据记录）
- 访问 Pipeline 详情页时，默认展示 Summary Tab，可切换至其他三个 Tab
- 生成记录 Tab 展示封面图、卡片图片、文案预览，并显示 pipeline 执行阶段信息
- 发布记录 Tab 展示 8 个平台各自的发布状态（已发布/定时/未发布/失败），无数据时显示 mock 占位
- 数据记录 Tab 展示各平台播放/点赞/评论/收藏数据，无数据时显示 mock 占位
- 页面整体样式采用深色背景（#07050f）、紫色渐变标题，符合 Justin Welsh 预览页风格
- /content-factory 路由路径保持不变，ContentFactory 页面原有数据加载逻辑不断
- Summary Tab 展示总曝光、总互动、平台数、互动率四项汇总指标
- 生成记录 Tab 调用 Brain API /pipelines/:id/output 与 /stages 端点获取数据
- 所有现有测试通过（GATE），产出物已创建，行为符合预期，CI 全部通过，文件包含正确内容
