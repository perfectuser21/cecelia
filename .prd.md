# PRD: Voice Queue Management Tools

## 需求背景

配合 cecelia-workspace/apps/core 的队列管理 API，为 Realtime Voice 添加 3 个工具函数，让用户可以通过语音控制任务队列。

## 功能需求

### Realtime Voice Tools

在 Realtime Config 中注册 3 个工具函数：

1. `get_queue` - 查看任务队列
2. `execute_now` - 插队执行任务（模糊匹配任务名）
3. `pause_task` - 暂停任务（模糊匹配任务名）

### 工具执行逻辑

实现工具调用的后端逻辑，调用 Core 的队列管理 API。

## 技术方案

**修改文件**：
- `src/api/orchestrator_routes.py` - 在 /realtime/config 添加 tools 定义
- `src/api/orchestrator_routes.py` - 在 /realtime/tool 添加执行逻辑

**API 调用**：
- 工具执行时调用 Core API: `http://localhost:5211/api/orchestrator/queue`
- 使用 httpx 发送请求

## 验收标准 (DoD)

- [ ] Realtime Config 包含 3 个工具定义
- [ ] 工具执行调用 Core API
- [ ] 返回友好的文本反馈

## 依赖

- cecelia-workspace/apps/core 的队列管理 API (cp-voice-orchestrator-integration 分支)
