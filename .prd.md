## PRD - CeceliaPage V2 Phase 2: 骨架重写

**需求来源**: CeceliaPage V2 设计方案 Phase 2（docs/design-cecelia-page-v2.md）
**功能描述**: 将 CeceliaPage 从被动聊天窗口重写为"有意识的管家指挥室"，新布局包含：
1. **Ambient Glow** — 页面边缘根据 alertness 等级变色脉动的呼吸光晕
2. **Pulse Strip** — 顶部状态条（警觉状态、tick 倒计时、运行中/排队任务数、今日费用）
3. **Consciousness Flow** — 左侧意识流区域（聊天 + 简报卡片 + 事件卡片混合）
4. **Activity Panel** — 右侧活动面板（运行中 Agent、集群状态、今日统计）
5. **useCeceliaWS hook** — WebSocket 封装，替代 setInterval 轮询
6. **Briefing Card** — 页面打开时自动展示简报卡片（调用 GET /api/brain/briefing）

**背景**: Phase 1（PR #551）已完成 Brain 端工作：WebSocket 新增 4 事件 + briefing API。本 Phase 实现前端骨架。

**涉及文件**:
- `apps/api/features/` 下新建/修改前端组件
- 修改 CeceliaPage 布局
- 新建 useCeceliaWS hook
- 新建各子组件（AmbientGlow, PulseStrip, ConsciousnessFlow, ActivityPanel, BriefingCard）

**成功标准**:
1. CeceliaPage 有 Ambient Glow + Pulse Strip + 双栏布局
2. WebSocket 连接 /ws 并接收事件（替代轮询）
3. 页面打开自动显示简报卡片
4. Activity Panel 显示运行中任务列表
5. 聊天功能保持可用
6. build 成功

**非目标**:
- 不做 DecisionCard 操作按钮（Phase 3）
- 不做记忆浮现（Phase 3）
- 不做动画打磨（Phase 4）
- 不改 Brain 后端代码
