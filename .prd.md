# PRD: Brain 部署流程可观测性补齐

## 背景
Brain deploy webhook (`POST /api/brain/deploy`) 是 fire-and-forget 设计，无状态追踪。

## 目标
补齐 deploy webhook 可观测性，使 CI 能感知实际部署结果。

## 成功标准
- GET /api/brain/deploy/status 端点存在并返回 status 字段
- 部署状态包含 running/success/failed 三态
- deploy.yml 加入 poll 验证步骤确认部署结果
- 单元测试覆盖 deploy status 状态流转
