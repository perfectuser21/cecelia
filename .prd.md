# PRD: 冻结 Python Brain 重叠路由

## 背景
Node.js Brain 已迁移到 cecelia-semantic-brain/brain/ 并运行在 5221 端口。Python Brain 仍然保留了重叠的 /api/brain/* 路由（focus, tick, goals, actions, queue），这些应该返回 410 Gone。

## 功能描述
将 Python FastAPI 中 state_routes.py 的所有 /api/brain/* 端点替换为 410 Gone 响应，指向 Node.js Brain。

## 成功标准
- [ ] 所有 /api/brain/* Python 端点返回 410
- [ ] Python 语义能力端点不受影响（/health, /fusion, /parse 等）
- [ ] 主入口 main.py 使用 frozen router
