# PRD: Fix UUID Serialization in Focus API

## 功能描述

修复 `/api/brain/focus/summary` 端点返回 500 错误的问题：
- objective_id 返回 UUID 对象而不是字符串
- key_results 中的 id 也是 UUID 对象
- Pydantic 验证失败导致 500 错误

## 成功标准

- [ ] /api/brain/focus/summary 返回 200
- [ ] objective_id 是字符串类型
- [ ] key_results[].id 是字符串类型
