# PRD: content pipeline executor 接通 Claude CLI 调用

## 背景

content-pipeline-executors.js 中 4 个阶段未调用 Claude CLI，配置的 prompt 被忽略。

## 成功标准

- executeCopywriting 使用 generate_prompt 调用 claude CLI，产出文案，rerun 时注入 review_feedback
- executeCopyReview 使用 review_prompt 调用 claude CLI，返回 rule_results 和 review_passed
- executeGenerate 使用 image_prompt 调用 claude CLI，产出图片描述
- executeImageReview 使用 image_review_prompt 调用 claude CLI，返回 review_passed 和 issues
- 修改后的文件 node 语法检查通过（node --check 验证）

## 修改范围

`packages/brain/src/content-pipeline-executors.js`
