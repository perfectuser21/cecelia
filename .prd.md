# PRD: 修复 ContentFactory + PipelineOutputPage UI 状态问题

## 背景
修复内容工厂前端4个UI状态问题，提升用户体验。

## 功能描述
1. ContentFactory pipeline 列表条目改为 Link 导航，支持点击进入详情页
2. PipelineOutputPage 在 pipeline 未完成时每 5s 自动轮询更新状态
3. StatusBadge 新增 pending → 待执行 中文映射
4. ContentFactory 列表在有进行中 pipeline 时每 5s 自动刷新
5. PipelineOutputPage GenerationTab 展示全部 6 个阶段（含 pending 占位）

## 成功标准
- ContentFactory pipeline 列表条目渲染为 Link 组件，href 指向 /content-factory/:id
- 有 in_progress/queued pipeline 时 ContentFactory 每 5s 自动轮询
- PipelineOutputPage 在 pipeline 非 completed/failed 时每 5s 轮询
- StatusBadge 包含 pending → 待执行 映射
- GenerationTab 始终显示全部 6 个阶段（基于 PIPELINE_STAGE_ORDER）
