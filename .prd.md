# PRD: 添加 dept_heartbeat 到 model-profile.js

## 背景

部门主管（repo-lead）架构设计中，Cecelia 每 5 分钟通过 heartbeat 触发各部门主管运行。
部门主管使用 MiniMax-M2.5-highspeed 模型。

需要在 FALLBACK_PROFILE 中注册 `dept_heartbeat` task type，使其：
1. 映射到 MiniMax-M2.5-highspeed 模型
2. 固定使用 minimax provider

## 目标

在 `brain/src/model-profile.js` 的 `FALLBACK_PROFILE` 中添加：

```javascript
// executor.model_map 里加：
dept_heartbeat: { anthropic: null, minimax: 'MiniMax-M2.5-highspeed' },

// executor.fixed_provider 里加：
dept_heartbeat: 'minimax',
```

## 验收标准

- [ ] model_map 包含 dept_heartbeat 条目
- [ ] fixed_provider 包含 dept_heartbeat 条目
- [ ] npm test 通过
- [ ] PR 合并到 develop
