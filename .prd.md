# PRD: 添加 Orchestrator API

## 背景

Cecelia Orchestrator 状态存储在 `~/runtime/state.json`，需要 API 暴露给前端。

## 目标

在 FastAPI 中添加 `/orchestrator` 路由组。

## 范围

- `src/api/orchestrator_routes.py` (新建)
- `src/api/main.py` (注册 router)

## 验收标准

1. GET `/orchestrator` 返回完整状态
2. GET `/orchestrator/health` 返回健康状态
