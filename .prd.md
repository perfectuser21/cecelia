# PRD: 修复前端路径配置

## 问题

服务器找不到前端文件：
```
ENOENT: no such file or directory, stat 'apps/core/frontend/dist/index.html'
```

## 原因

- 服务器编译位置：`apps/core/dist/dashboard/server.js`
- 当前配置：`join(__dirname, '../../frontend/dist')` → `apps/core/frontend/dist`（不存在）
- 实际前端位置：`apps/dashboard/frontend/dist`

## 修复

修改 `server.ts` 中的 `frontendPath`：
```typescript
join(__dirname, '../../../dashboard/frontend/dist')
```

## 成功标准

1. 访问 `/brain` 能正常加载页面
2. 不再报 ENOENT 错误
