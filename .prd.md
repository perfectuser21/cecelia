# PRD: Brain Linux->macOS 全面平台适配

## 功能描述
创建 packages/brain/src/platform-utils.js 统一平台适配层，修复所有 P0/P1 的 Linux-only 兼容性问题。
Brain 从 Ubuntu VPS 迁移到 Mac mini (Darwin) 后，大量 Linux-only 代码在 macOS 上静默失败。
CTO 审计发现 20+ 个兼容性问题，需要系统性修复。

## 成功标准
- 所有 Linux-only 系统调用在 macOS 上有对应实现或降级处理
- detectUserSessions() 在 macOS 上正确检测用户 claude 进程
- sampleCpuUsage() 在 macOS 上返回有意义的 CPU 使用率
- checkServerResources() swap 检测在 macOS 上正常工作
- PHYSICAL_CAPACITY 在 Mac mini 16GB 上算出合理值 (<=10)
- pgrep claude 计数在 macOS 上正确
- cluster.js 进程信息在 macOS 上可用
- zombie-sweep 进程扫描在 macOS 上正确
- vps-monitor CPU/网络统计在 macOS 上正确
- 现有测试全部通过

## 范围
- P0: slot-allocator detectUserSessions, executor sampleCpuUsage/checkServerResources/pgrep/PHYSICAL_CAPACITY, cluster.js /proc 依赖
- P1: zombie-sweep ps 格式, vps-monitor top/netstat, executor dmesg

## 不改动
- watchdog.js (已适配)
- 版本号 (auto-version.yml 处理)
