# PRD: 修复 capability-scanner skillStats recent_30d 缺失

## 背景

`capability-scanner.js` 的 `scanCapabilities` 在 PR #1431 完成圈复杂度重构后，
发现 `skillStats` SQL 查询缺少 `recent_30d` 字段，导致通过 `run_events` 追踪的
技能在 `health.usage_30d` 统计中始终为 0，能力活跃度被低估。

## 成功标准

1. `skillStats` 查询新增 `recent_30d` 计算列（30 天内运行次数）
2. 新增 `failing` 状态场景测试（success_rate < 30%）
3. 所有现有测试继续通过
