# PRD: 快手视频发布脚本（publish-kuaishou-video.cjs）

## 背景

快手 publisher 当前只支持图文（publish-kuaishou-api.cjs），缺少视频发布能力。
本 PR 新增 publish-kuaishou-video.cjs，与图文方案对齐使用 CDP Cookie + HTTP API 方式。

## 目标

实现快手视频内容自动化发布，支持视频文件上传、标题、标签、封面设置。

## 功能描述

- 复用 kuaishou-publisher 现有 CDP Cookie 提取逻辑
- 调用快手内部视频上传 API
- 支持参数：--video（视频路径）--title --tags --cover（封面图路径，可选）
- 脚本位置：packages/workflows/skills/kuaishou-publisher/scripts/publish-kuaishou-video.cjs

## 成功标准

- publish-kuaishou-video.cjs 文件存在且可执行
- 支持 --video / --title / --tags / --cover 参数
- 发布成功返回作品 ID
- 错误时输出可读的失败原因
- SKILL.md changelog 更新到 1.4.0
