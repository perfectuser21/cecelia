# PRD - 修复深度审计 HIGH 级问题

## 需求来源

深度审计（5 Agent 并行）发现的 HIGH 严重性问题，可能导致功能失败或安全风险。

## 功能描述

### 1. CI 配置修复
- 删除 ci.yml 中不存在的 npm scripts（lint, format:check）

### 2. Hooks 安全加固
- branch-protect.sh: cd 失败处理
- pr-gate-v2.sh: cd 失败处理

### 3. 脚本安全加固
- run-regression.sh: 命令注入防护

## 成功标准

- CI 不调用不存在的脚本
- cd 失败时脚本安全退出
- 命令执行使用参数数组而非字符串
