# PRD: Quality API Trigger Endpoints

## 目标
为 Quality Monitor v2.1 添加触发操作的 API 端点。

## 功能需求
- POST /api/trigger/runQA - 触发 QA 检查
- POST /api/trigger/syncNotion - 同步 Notion
- POST /api/trigger/healthCheck - 健康检查
- DELETE /api/queue/clear - 清空队列
- POST /api/worker/restart - 重启 Worker

- [ ] API 端点正确返回数据
- [ ] 错误处理完善
- [ ] 与前端集成成功
