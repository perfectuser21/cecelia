# PRD: Executor Callback 集成测试

## 背景
Executor 完成任务后 POST callback 到 Brain（POST /api/brain/execution-callback），Brain 更新 task 状态到 DB。这条链路从未被真实测试过，现有测试全部 mock 了 DB 层。

## 目标
验证 callback → DB 状态更新的完整链路，确保 callback 路由正确持久化结果到 PostgreSQL。

## 成功标准
- tests/integration/callback.integration.test.ts 存在，含 ≥3 个测试用例
- 使用真实 pg.Pool，不 mock 任何内部模块
- 覆盖 success/failure/result payload 三种场景
- skipIf 机制确保无 DB 环境自动跳过（不影响 L3）
