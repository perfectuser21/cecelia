# PRD: executeResearch NotebookLM web search

## 背景
内容 pipeline 的 research 阶段使用 NotebookLM 进行信息收集。
当前实现只对空 notebook 直接 ask，导致返回空内容或无意义回答。

## 需求
修复 executeResearch，让其按以下顺序操作：
1. `notebooklm use {notebookId}` — 切换 notebook
2. 清空现有 sources（notebook 复用）
3. `notebooklm source add-research "{keyword}" --mode deep --no-wait` — 搜索 web
4. `notebooklm research wait --timeout 300 --import-all` — 等待并导入所有结果
5. `notebooklm ask "{prompt}" --json` — 查询汇总
6. 清空 sources（下次复用）
