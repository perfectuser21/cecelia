# Task Card: 内容工厂 Pipeline Executor 全链路实现

## 需求（What & Why）
**功能描述**: 为内容工厂的 4 个阶段实现 executor，让 pipeline 从"创建子任务但不执行"变为"自动执行每个阶段"：
1. content-research executor：调用 notebooklm CLI 拉取调研素材
2. content-generate executor：基于素材生成图文文案 + 长文
3. content-review executor：自动检查品牌关键词命中和禁用词
4. content-export executor：归档 + manifest.json + 在线预览页

**背景**: orchestrator 状态机已完整，但子任务创建后没有 executor 执行，全靠手动 callback。

**不做什么**:
- 不改 /share-card skill
- 不改 tick/scheduler 调度逻辑
- 不做八平台发布对接

## 成功标准
1. [ARTIFACT] content-pipeline-executors.js 包含 4 个 export function
2. [BEHAVIOR] research executor 能调用 notebooklm ask 命令
3. [BEHAVIOR] review executor 能检查品牌关键词和禁用词
4. [ARTIFACT] export executor 生成 manifest.json
5. [GATE] 不破坏 Brain 现有功能
