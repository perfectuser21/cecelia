# Task Card: [选题闭环] 基于数据驱动的AI自动选题引擎

## 需求（What & Why）
**功能描述**: 实现内容反馈→选题的完整闭环：(1)分析近7日高ROI内容特征注入选题Prompt；(2)LLM抽取「AI一人公司」垂类热点话题；(3)生成3-5个选题候选+理由；(4)推荐给Alex或2小时后自动进入内容队列。
**背景**: 核心框架（topic-selector/topic-suggestion-manager/scheduler/routes）已实现，但缺少：近7日 content_analytics ROI数据直接注入选题Prompt（只有周粒度feedback），以及手动触发端点（测试/Debug用）。
**不做什么**: 不改变现有调度逻辑；不修改DB schema；不改 topic-heat-scorer 的周粒度逻辑。

## 成功标准
- 近7日高ROI内容特征（平台/互动率）注入 generateTopics Prompt
- POST /api/brain/topics/generate 手动触发端点可用
- 测试文件覆盖核心纯函数
