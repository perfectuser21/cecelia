# Task Card: 知识模块页面修复 — UI 暗色风格 + 查看链接修复

## 需求（What & Why）

**功能描述**:
1. 将 `KnowledgeModules.tsx` 和 `KnowledgeModuleDetail.tsx` 改为暗色风格，与 `InstructionBook.tsx` 保持视觉一致（`#0d0d0d` 背景、深色卡片、`#3467D6` 蓝色强调）
2. 修复 `KnowledgeModuleDetail.tsx` 中"在 Dashboard 中查看"按钮的 URL：从相对路径改为绝对 URL `http://38.23.47.81:9998/{output_url}`（HTML 知识页托管在 port 9998）

**背景**: 知识模块功能（PR #1620/1625）实现了页面和 API，但 UI 用了白色/浅色主题，与 InstructionBook 说明书的深色风格不一致。"查看知识页"按钮用相对路径无法加载文件（文件在 port 9998，不是 dashboard 同域）。

**不做什么**: 不改动路由注册、不改动 Brain API 端点、不改动 BACKLOG.yaml 数据

## 成功标准

1. [ARTIFACT] `KnowledgeModules.tsx` 主背景使用 `#0d0d0d`（与 InstructionBook 对齐）
2. [ARTIFACT] `KnowledgeModuleDetail.tsx` 查看按钮 URL 使用 `http://38.23.47.81:9998/`
3. [BEHAVIOR] 点击"在 Dashboard 中查看"时用 `window.open` 在新标签页打开绝对 URL
4. [GATE] CI 全部通过
