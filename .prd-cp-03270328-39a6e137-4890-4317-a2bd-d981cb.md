# PRD: Knowledge Modules 页面接入 Dashboard

## 背景
将西安生成的 86 个知识页接入 Dashboard，用原生组件渲染。

## 功能描述
在 Dashboard 新增 `/knowledge/modules` 页面，展示所有知识模块卡片（brain/engine/system/workflows 四个分组），支持点击查看模块详情。

## 成功标准
- `/knowledge/modules` 页面展示所有模块卡片（brain/engine/system/workflows 四个分组）
- 点进任意模块可看到详情：标题、摘要、关键数字、来源文件
- Brain API `GET /api/brain/knowledge/modules` 返回结构化数据（从 BACKLOG.yaml 读）
- 不用 iframe，全部用 Dashboard 原生组件渲染
