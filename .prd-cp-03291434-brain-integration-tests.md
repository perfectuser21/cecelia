# PRD: Brain 关键路由集成测试补全

**任务 ID**: d1029612-8719-43e9-88da-aa5e540ce23f
**分支**: cp-03291434-brain-integration-tests
**优先级**: P1

## 背景

Brain 路由目前只有单元测试（mock DB），几乎没有集成测试。`/context` 路由曾有 SQL bug（已修复），但无测试保护，未来变更可能再次引入 regression。

CI 的 `brain-unit` job 已有真实 PostgreSQL，`integration-test-required` 门禁要求 Brain feat PR 必须含集成测试。

## 目标

给以下 4 个关键路由补充集成测试（使用真实 PostgreSQL，非 mock DB）：

- `GET /api/brain/health`
- `GET /api/brain/tasks`
- `GET /api/brain/context`（最重要：曾有 SQL bug）
- `GET /api/brain/okr/current`

## 技术方案

- 测试文件：`packages/brain/src/__tests__/integration/critical-routes.integration.test.js`
- 使用 supertest + express 构建测试 app
- 使用真实 pg.Pool 连接 DB（DB_DEFAULTS）
- beforeAll 插入测试数据，afterAll 清理
- mock 外部服务（tick.js、circuit-breaker.js）避免副作用

## 成功标准

- 集成测试文件存在于 `packages/brain/src/__tests__/integration/` 目录
- 覆盖 4 个核心路由的端到端 HTTP 行为
- CI L3 `integration-test-required` 门禁通过
- 所有测试在 `brain-unit` job（真实 DB）中通过
