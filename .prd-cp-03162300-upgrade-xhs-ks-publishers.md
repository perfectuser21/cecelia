# PRD: 升级小红书和快手发布脚本（CDP UI自动化 + 音乐选择）

## 背景

本会话已验证快手 CDP UI 方案（/tmp/ks-publish-v4.js）。小红书现有脚本用旧方案（DOM.setFileInputFiles 触发不了 Vue 上传），且端口写错（19225→应为19224）。两个平台发布时都没有选择背景音乐。文件复制到 Windows 走 xian-mac 中转。

## 目标

把小红书和快手发布脚本升级为生产可用的纯自动化脚本，与抖音发布器保持一致。

## 成功标准

1. XHS 脚本端口改为 19224，使用 Input.dispatchMouseEvent + Page.fileChooserOpened 上传图片
2. XHS 脚本含 SCP 步骤（通过 xian-mac 跳板），含音乐选择
3. KS 脚本（publish-kuaishou-image.cjs）替换为 CDP UI 方案，含 SCP 步骤，含音乐选择
4. 两个脚本都支持 `--content <dir>` 无人值守运行
5. 音乐选择失败时降级跳过（不阻断发布）
