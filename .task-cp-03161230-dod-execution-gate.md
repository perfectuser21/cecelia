# Task Card: DoD Execution Gate

## 需求（What & Why）
**功能描述**: 在 CI L1 中新增 `dod-execution-gate.sh` 脚本，实际执行 Task Card 中所有 `[BEHAVIOR]` 条目后跟的 `Test:` 命令，让 CI 不仅验证格式，还验证行为是否真实工作。

**背景**: 当前 CI L1 的 `check-dod-mapping.cjs` 只检查 DoD Test 字段的**格式**（非空、有正确前缀），从不执行命令。开发者可以写出格式正确但逻辑永远失败的 Test，CI 也会绿灯放行。新功能可能根本没有工作。

**不做什么**:
- 不执行需要运行服务的命令（curl、chrome 等）
- 不修改现有的 check-dod-mapping.cjs（格式检查保留）
- 不执行 [ARTIFACT] 或 [GATE] 条目的 Test（只执行 [BEHAVIOR]）

## 成功标准
> [ARTIFACT] 产出物 / [BEHAVIOR] 运行时行为 / [GATE] 门禁

1. [ARTIFACT] `packages/engine/scripts/devgate/dod-execution-gate.sh` 文件存在且可执行
2. [ARTIFACT] CI L1 `dod-check` job 新增执行 `dod-execution-gate.sh` 的 step
3. [BEHAVIOR] 脚本对包含 `manual:bash -c "grep ..."` 类 Test 的 Task Card，执行命令并在成功时返回 0
4. [BEHAVIOR] 脚本对 `manual:curl`、`manual:chrome:screenshot` 类标记为 DEFERRED（输出提示但不报错）
5. [BEHAVIOR] 脚本对 `Test: tests/path/to/file.ts` 类，检查文件是否存在
6. [BEHAVIOR] 任何 bash 命令执行失败时 exit 1，并输出失败的条目内容
7. [GATE] CI L1 所有现有检查仍通过

## 验收条件（DoD）

- [x] [ARTIFACT] `dod-execution-gate.sh` 存在于正确路径且可执行
  Test: manual:bash -c "stat packages/engine/scripts/devgate/dod-execution-gate.sh && bash packages/engine/scripts/devgate/dod-execution-gate.sh --help 2>&1 || true"

- [x] [ARTIFACT] CI L1 `dod-check` job 包含对 `dod-execution-gate.sh` 的调用
  Test: manual:bash -c "grep -c 'dod-execution-gate' .github/workflows/ci-l1-process.yml"

- [x] [BEHAVIOR] 脚本包含识别 `[BEHAVIOR]` 条目和执行 `manual:bash` 命令的逻辑
  Test: manual:bash -c "node -e \"const s=require('fs').readFileSync('packages/engine/scripts/devgate/dod-execution-gate.sh','utf8'); process.exit(s.includes('[BEHAVIOR]') && s.includes('manual:bash') ? 0 : 1)\""

- [x] [BEHAVIOR] `manual:curl` 和 `manual:chrome` 类命令被标记为 DEFERRED 跳过
  Test: manual:bash -c "node -e \"const s=require('fs').readFileSync('packages/engine/scripts/devgate/dod-execution-gate.sh','utf8'); process.exit(s.includes('DEFERRED') ? 0 : 1)\""

- [x] [BEHAVIOR] `Test: tests/` 类条目检查文件存在性的逻辑存在于脚本中
  Test: manual:bash -c "node -e \"const s=require('fs').readFileSync('packages/engine/scripts/devgate/dod-execution-gate.sh','utf8'); process.exit(s.includes('tests/') ? 0 : 1)\""

- [x] [GATE] 所有现有测试通过
  Test: tests/quality-system/run-all.sh
