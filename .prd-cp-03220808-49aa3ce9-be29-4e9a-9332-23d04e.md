# PRD: Content Factory 批量生成

**Task ID**: 49aa3ce9-be29-4e9a-9332-23d04e691165
**Branch**: cp-03220808-49aa3ce9-be29-4e9a-9332-23d04e

## 背景

Content Factory 需要支持批量并发生成多条内容，当前只有单条生成接口。需要新增批量端点，支持 2-20 条内容同时排队生成。

## 目标

新增 `POST /api/brain/pipelines/batch` 端点，支持 AI 批量内容生成任务。

## 功能要求

1. 接受 `items` 数组，每项包含 `content_type` 和可选 `context`
2. `items` 数量 < 2 返回 400 错误
3. `items` 数量 > 20 返回 400 错误
4. 无效的 `content_type` 返回 400 错误
5. 合法请求返回 202 Accepted，附带 pipeline_ids 数组

## 成功标准

- `POST /api/brain/pipelines/batch` 端点存在并可调用
- items < 2 返回 400 Bad Request
- items > 20 返回 400 Bad Request
- 无效 content_type 返回 400 Bad Request
- `packages/brain/src/__tests__/content-pipeline-routes.test.js` 包含 ≥14 个测试
