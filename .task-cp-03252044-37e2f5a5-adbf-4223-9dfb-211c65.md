## 实现方案

**要改的文件**:
- `apps/api/src/task-system/goals.js` — GET / 改为 fetch Brain /api/brain/goals
- `apps/api/src/task-system/projects.js` — GET / 改为 fetch Brain /api/brain/projects
- `apps/api/src/task-system/areas.js` — 删除 ORDER BY sort_order，改为 ORDER BY created_at
- `apps/api/src/task-system/routes.js` — import areasRouter + router.use('/areas', areasRouter)

**数据映射**:
- goals: type `area_kr` → `kr`; 补充 priority='P2', weight=1.0（Brain 不返回这些字段）
- projects: title → name; type='initiative'; 补充 priority='P2', parent_id=null, execution_mode=null（Brain 有 progress 字段）
