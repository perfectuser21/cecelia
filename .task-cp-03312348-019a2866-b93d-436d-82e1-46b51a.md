# Task Card: [SelfDrive] 内容选题自动化 - MVP

## 需求（What & Why）
**功能描述**: 修复 AI 选题 LLM 调用失败问题，并新增 GET /api/brain/topics 路由，使内容生成具备最小可用的自动化选题能力。
**背景**: topic-selector.js 已存在但 LLM 路由指向 codex（当前不可用），导致每日自动选题无法实际执行；同时缺少查看选题候选库的 API，无法感知选题结果。
**不做什么**: 不改动 content_topics 表集成；不改动 UI/Dashboard；不改动 topic_selection_log 表结构。

## 成功标准

1. [ARTIFACT] GET /api/brain/topics 路由存在并已注册到 routes.js
2. [BEHAVIOR] GET /api/brain/topics 返回 topic_selection_log 中的今日及历史选题记录
3. [BEHAVIOR] POST /api/brain/pipelines/trigger-topics 触发后，LLM 调用实际成功（不再因 codex profile 失败），topic_selection_log 写入新记录
4. [PRESERVE] topic-selection-scheduler 每日 UTC 01:00 调度逻辑不变
5. [GATE] CI 全部通过
