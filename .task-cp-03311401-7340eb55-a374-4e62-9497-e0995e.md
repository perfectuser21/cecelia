# Task Card: 社媒热点数据接入接口

## 需求（What & Why）
**功能描述**: 在 Brain 新增 GET /api/brain/social/trending 端点，查询 TimescaleDB 社媒采集数据，供内容生成系统消费。支持 platform 和 limit 查询参数过滤结果，TimescaleDB 不可达时降级返回空数组。

**背景**: Cecelia 内容生成系统需要消费社媒热点数据，数据存储在 TimescaleDB（统一视图 v_all_platforms，各平台原始表 raw_data_douyin/raw_data_kuaishou 等，字段含 title/views/likes/comments/platform/scraped_at）。当前 Brain 没有对应 API 端点，内容系统无法消费热点数据。

**不做什么**: 不写入数据、不做数据采集、不修改 TimescaleDB schema、不做数据聚合分析、不暴露原始表直接查询接口。

## 成功标准
1. [ARTIFACT] packages/brain/src/routes/social-trending.js 文件存在，且在 routes.js 中完成注册
2. [BEHAVIOR] GET /api/brain/social/trending 返回 HTTP 200 及 JSON 数组，每条记录含 platform 字段；支持 ?platform=xxx 和 ?limit=N 查询参数
3. [BEHAVIOR] TimescaleDB 不可达时，接口返回 HTTP 200 及空数组（不报 500）
4. [GATE] CI 全部通过
