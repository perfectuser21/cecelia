# PRD: Brain 关键路由集成测试补全

## 背景
Brain CI 目前只有 mock DB 的单元测试，无集成验证。`/api/brain/context` 刚修复了 SQL bug 但无测试保护。

## 功能描述
给 Brain 最关键的 4 个路由补集成测试（用真实 PostgreSQL），解决 CI 只有单元测试、无集成验证的根本缺口。

## 目标路由
- GET /api/brain/context（刚修复的 SQL bug，无测试保护）
- GET /api/brain/tasks
- GET /api/brain/okr/current
- GET /api/brain/health

## 技术方案
- 使用 supertest + 真实 PostgreSQL 连接（非 mock db.js）
- CI brain-unit job 已有 PostgreSQL 服务
- 测试文件放 packages/brain/src/routes/__tests__/

## 成功标准
- 4 个核心路由有集成测试（真实 DB）
- CI 跑通这些测试
- /context 不再无测试保护（SQL 变更会被捕获）
