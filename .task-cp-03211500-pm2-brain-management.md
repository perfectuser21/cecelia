# Task Card: PM2 进程管理 + 日志轮转

## 需求（What & Why）
**功能描述**: 配置 PM2 进程管理器管理 Brain 进程，实现崩溃自动重启、日志自动轮转、磁盘空间监控，保证 Brain 24/7 稳定运行。
**背景**: Brain 是 Cecelia 系统核心，需要稳定运行。当前缺少进程管理，崩溃后无自动恢复；日志文件无轮转策略可能占满磁盘。
**不做什么**: 不改动 Brain 核心逻辑代码，不改动 tick loop，不改动 server.js 业务逻辑。

## 成功标准
> [ARTIFACT] 产出物 / [BEHAVIOR] 运行时行为 / [GATE] 门禁

2. [ARTIFACT] packages/brain/scripts/setup-pm2.sh 文件存在且可执行
3. [ARTIFACT] scripts/disk-check.sh 磁盘空间监控脚本存在且可执行
4. [BEHAVIOR] ecosystem.config.cjs 导出合法的 PM2 配置（apps 数组、autorestart=true、max_memory_restart 设定）
5. [GATE] CI 全部通过
