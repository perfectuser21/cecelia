# PRD: account_usage_cache 新增 seven_day_sonnet_resets_at

## 背景
Brain 的 account_usage_cache 表缺少 seven_day_sonnet scope 的独立 resets_at 字段。
Anthropic API 对 five_hour、seven_day、seven_day_sonnet 三个窗口各自返回独立的 resets_at，
但当前代码只存了前两个，丢掉了 sonnet 的。

## 目标
新增 seven_day_sonnet_resets_at 列，使 /llm-quota 能显示 Sonnet 独立的重置时间。

## 成功标准

1. DB migration 新增 seven_day_sonnet_resets_at 列
2. account-usage.js upsertCache 存储 seven_day_sonnet_resets_at 字段
3. getAccountUsage 返回 seven_day_sonnet_resets_at 字段
4. fallback 默认值包含 seven_day_sonnet_resets_at: null
5. CI 全部通过
