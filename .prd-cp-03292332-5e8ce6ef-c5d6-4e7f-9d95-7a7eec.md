# PRD: Pipeline Integrity Gate — CI pipeline 安全属性元检查

## 背景
/dev 工作流依赖一系列 pipeline 安全属性（fail-closed、无上限孤儿处理、gate 链完整性、seal 格式）。这些属性曾存在 bug，修复后缺乏回归测试保护。

## 功能描述
新增 Pipeline Integrity Gate CI 测试文件 `packages/engine/tests/pipeline-integrity.test.ts`，验证所有 pipeline 安全不变量：
1. fail-closed 属性：API 错误时必须 exit 1，不能放行
2. 反模式扫描（N次后放行）：已删除孤儿计数超限放行逻辑
3. gate 链完整性：devloop-check.sh 包含 4 个必要 gate 条件
4. 关键文件存在性：pipeline 安全文件必须存在
5. seal 格式校验：divergence_count 门禁机制存在

## 成功标准
- pipeline-integrity.test.ts 创建并通过所有测试
- CI 全部通过
