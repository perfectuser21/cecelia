## PRD - feat: GET /api/brain/projects/compare — KR进度 + 趋势数据

**需求来源**: Brain 自动调度任务
**功能描述**: 新增 GET /api/brain/projects/compare 端点，补充 KR 进度和历史趋势数据，基于 PR #783（POST /api/brain/projects/compare/report）扩展实现
**涉及文件**:
- `packages/brain/src/project-compare.js` — 新增 getCompareMetrics 函数
- `packages/brain/src/routes.js` — 新增 GET /projects/compare 路由
- `packages/brain/src/__tests__/project-compare.test.js` — 追加测试块

## 成功标准
- GET /api/brain/projects/compare?ids=uuid1,uuid2 返回 200 且含 projects 数组
- projects[0].kr.progress 是整数（或 kr 为 null 当无关联 KR 时）
- projects[0].trend 是数组，每项含 week（如 "2026-W10"）和 completed（整数）
- ids 少于2个时返回 400
- 现有 generateCompareReport 的测试仍全通过
- Brain CI（brain-unit）通过

**非目标**: 不修改现有 generateCompareReport 函数，不做 UI 展示

### 异常处理 & 报警规范

N/A - 纯 API 查询，无外部 IO 操作
