# Task Card: KnowledgeModules UI 重设计 + 查看链接修复

## 需求（What & Why）
**功能描述**: 修复 KnowledgeModules.tsx 和 KnowledgeModuleDetail.tsx 两个问题：
1. 将两个组件从亮色（white/gray Tailwind）风格统一改为暗色风格（对齐 InstructionBook 的 `#0d0d0d` 背景色系）
2. 知识页"查看"链接改为绝对 URL（`http://38.23.47.81:9998/{output_url}`），使用 `window.open` 在新标签打开，而非通过 navigate 跳转相对路径

**背景**: 当前 KnowledgeModules 页面使用亮色 Tailwind 样式，与系统其他页面（InstructionBook）风格不一致。查看链接跳转到内部 `/knowledge/view` 路由也可能有问题。

**不做什么**: 不修改 API 调用逻辑，不修改路由，不修改数据结构。

## 成功标准

1. [ARTIFACT] KnowledgeModules.tsx 背景色改为 `#0d0d0d`，卡片/面板为 `#141414`，文字为 `#fff`/`#888`/`#ccc`
2. [ARTIFACT] KnowledgeModuleDetail.tsx 背景色改为暗色，与 InstructionBook 风格一致
3. [BEHAVIOR] KnowledgeModuleDetail 中"在 Dashboard 中查看"按钮点击后以 `window.open('http://38.23.47.81:9998/'+output_url, '_blank')` 打开
4. [GATE] CI 全部通过
