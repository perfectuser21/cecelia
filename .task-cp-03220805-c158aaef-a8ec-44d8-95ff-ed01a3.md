# Task Card: KR3-T1 内容模板梳理

## 背景

Cecelia 内容工厂系统已有 `content-types/` 目录，支持通过新增 YAML 文件扩展内容类型，无需修改代码。当前仅有 `solo-company-case.yaml` 一个样本，缺乏完整的内容模板体系。

## 目标

在 `packages/brain/src/content-types/` 下新增 5 个内容类型 YAML：
- **long-form** — 长文（深度文章）
- **short-form** — 短文（图文/短内容）
- **video-script** — 视频脚本（抖音/快手竖屏）
- **promo-copy** — 推广文案（产品/服务推广）
- **data-analysis** — 数据分析报告

## 成功标准

- [ ] [ARTIFACT] 5 个 YAML 文件均存在于 `packages/brain/src/content-types/`
  Test: manual: node -e "const fs=require('fs');['long-form','short-form','video-script','promo-copy','data-analysis'].forEach(t=>{if(!fs.existsSync('packages/brain/src/content-types/'+t+'.yaml')){console.error('缺失:',t);process.exit(1);}});console.log('OK');"

- [ ] [BEHAVIOR] 所有文件通过 YAML 解析且必填字段完整
  Test: manual: node -e "const yaml=require('js-yaml');const fs=require('fs');const req=['content_type','images','template','review_rules','copy_rules'];let ok=true;['long-form','short-form','video-script','promo-copy','data-analysis'].forEach(t=>{const d=yaml.load(fs.readFileSync('packages/brain/src/content-types/'+t+'.yaml','utf8'));req.forEach(f=>{if(!d[f]&&d[f]!==0){console.error('缺字段:',t,f);ok=false;}});if(d.content_type!==t){console.error('content_type不一致:',t,'vs',d.content_type);ok=false;}});if(!ok)process.exit(1);console.log('ALL OK');"

- [ ] [BEHAVIOR] `content-type-registry.js` 能成功加载所有 5 个新类型
  Test: manual: node --input-type=module -e "import {loadAllContentTypes} from './packages/brain/src/content-types/content-type-registry.js';const r=await loadAllContentTypes();const names=r.map(t=>t.name);['long-form','short-form','video-script','promo-copy','data-analysis'].forEach(n=>{if(!names.includes(n)){process.stderr.write('未加载:'+n+'\n');process.exit(1);}});console.log('OK:',names.length,'个类型');"

- [ ] [GATE] 所有内容类型通过 `content-type-validator.js` 校验
  Test: manual: node --input-type=module -e "import {validateAllContentTypes} from './packages/brain/src/content-types/content-type-validator.js';const result=await validateAllContentTypes();const failed=result.results.filter(r=>!r.valid);if(failed.length>0){failed.forEach(f=>process.stderr.write('校验失败:'+f.name+'\n'));process.exit(1);}console.log('全部通过，共',result.results.length,'个类型');"
