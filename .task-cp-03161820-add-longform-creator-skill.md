# Task Card: longform-creator Skill

## 需求（What & Why）
**功能描述**: 创建 `longform-creator` skill，用于生成公众号长文内容（文字 HTML + 配图），并保存到 NAS。
**背景**: 用户需要将内容创作和发布分离。创作（longform-creator）→ 保存到 NAS → 发布（wechat-publisher）从 NAS 读取。
**图片规格**: 封面 900×383 (2.35:1)，正文配图 1080×810 (4:3 横版)，深色科技风格。
**不做什么**: 不负责发布（那是 wechat-publisher 的职责），不负责具体内容生成（skill 是指令文档）。

4. [ARTIFACT] NAS 路径和目录结构明确
5. [GATE] CI 全部通过
