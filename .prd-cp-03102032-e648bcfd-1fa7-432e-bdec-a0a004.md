## PRD - feat(dashboard): ProjectCompare 补充 KR 达成率 + 周趋势迷你图

**需求来源**: Brain 自动调度任务
**功能描述**: PR #808 实现了 GET /api/brain/projects/compare 端点（返回 kr_progress + trend 字段），PR #810 实现了 ProjectCompare 前端页面，但前端仅调用 POST /compare/report 端点，完全未展示 KR 达成率和历史趋势数据。需要在 ProjectCompare 页面集成 KR 达成率和近 4 周趋势迷你图。
**涉及文件**: `apps/api/features/planning/pages/ProjectCompare.tsx`（仅此一个文件）
**成功标准**:
1. GET /api/brain/projects/compare 被正确调用（与 POST report 并行）
2. 每个 ProjectCard 显示 KR 进度条和百分比
3. 每个 ProjectCard 显示近 4 周趋势迷你图
4. 无外部图表依赖新增
5. 无 TypeScript 报错
6. kr 为 null 时 UI 优雅降级
**非目标**: 不修改其他文件，不引入 recharts 等图表库

### 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| GET /compare 接口失败 | 降级：不显示 KR 和趋势区块 | P3 | compare_kr_fetch_error |
| kr 字段为 null | 显示「暂无关联 KR」 | P3 | - |
| trend 数组为空 | 显示「暂无趋势数据」 | P3 | - |
