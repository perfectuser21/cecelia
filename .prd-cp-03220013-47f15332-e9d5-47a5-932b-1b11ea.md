# PRD: branch-protect 新增 .dod.md 归属校验

## 功能描述
在 `packages/engine/hooks/branch-protect.sh` 中新增检查：若当前 worktree 的项目根目录存在 `.dod.md` 文件，且其 frontmatter 中有 `branch:` 字段，则校验该字段值是否与当前 git 分支一致。不一致时输出 WARNING 提示（非阻断）。

## 成功标准

1. branch-protect.sh 增加 .dod.md branch 字段校验
2. 校验不通过时输出警告信息
3. 不阻断正常工作流（仅 warning，不 exit 1）
