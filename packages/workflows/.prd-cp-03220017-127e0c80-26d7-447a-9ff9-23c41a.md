# PRD: brain-register 新增 task_type 时验证 skill 文件存在

## 背景
来自知识反刍 Batch 1。当前 brain-register SKILL.md 的 B 类（Executor 任务类型）流程
只引导修改 executor.js，但未验证对应的 SKILL.md 是否存在。
若缺少 SKILL.md，Brain dispatch 后 Claude Code 无法加载 skill，导致任务执行失败。

## 目标
1. 在 brain-register SKILL.md 的 B 类步骤中新增 SKILL.md 存在性验证说明
2. 新建 CI DevGate 脚本，自动检查 PR 中新增的 task_type 对应 SKILL.md 是否存在
3. 将 CI 检查集成到 brain-l2 job

## 成功标准
- `packages/engine/scripts/devgate/check-skill-file-exists.mjs` 已创建
- brain-register SKILL.md B 类步骤新增 check-skill-file-exists 验证引用
- ci-l2-consistency.yml brain-l2 job 调用新脚本
- CI 全部通过
