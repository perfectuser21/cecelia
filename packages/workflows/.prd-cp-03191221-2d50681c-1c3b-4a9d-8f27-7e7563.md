# PRD - 小红书视频发布脚本（publish-xiaohongshu-video.cjs）

## 背景
小红书 publisher 当前只支持图文（publish-xiaohongshu-image.cjs），缺少视频发布能力。本 PR 新增 publish-xiaohongshu-video.cjs，与图文方案对齐使用 CDP WebSocket 直连 Windows PC。

## 目标
实现小红书视频内容自动化发布，支持视频文件上传、标题、标签、封面。

## 技术方案
- 复用 xiaohongshu-publisher 现有 CDP WebSocket 架构
- CDP 端口：19225（小红书视频专用）
- Windows PC Tailscale: 100.97.242.124
- 发布流程：进入发布页 → 选择视频 → 填写信息 → 发布
- 支持参数：--video --title --tags --cover（可选）
- 脚本位置：packages/workflows/skills/xiaohongshu-publisher/scripts/publish-xiaohongshu-video.cjs

## 验收标准
- 成功上传视频到小红书创作者中心
- 标题/标签写入正确
- 发布成功，可在创作中心看到作品
- SKILL.md changelog 更新到 1.3.0

## 成功标准
- 任务完成
