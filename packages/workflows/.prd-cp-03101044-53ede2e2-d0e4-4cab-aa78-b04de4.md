## PRD - [P1 KR1] 公众号发布 API 接通

**需求来源**: Brain 自动调度 — 完成平台发布矩阵（与快手、微博并列）
**功能描述**: 实现微信公众号（WeChat Official Account）图文发布接口，通过官方 API 认证，发布图文（草稿+发布）
**涉及文件**:
- `packages/workflows/skills/wechat-publisher/SKILL.md`（新建）
- `packages/workflows/skills/wechat-publisher/REQUIREMENTS.md`（新建）
- `packages/workflows/skills/wechat-publisher/scripts/publish-wechat-article.cjs`（新建）
- `packages/workflows/skills/wechat-publisher/scripts/check-wechat-token.cjs`（新建）
- `~/.credentials/wechat.env`（新建，凭据文件，不进 git）

## 成功标准
- [ ] 读取 `~/.credentials/wechat.env` 中的 WECHAT_APPID + WECHAT_APPSECRET
- [ ] 自动获取 / 刷新 Access Token（有效期 7200s，缓存到 `/tmp/wechat_token.json`）
- [ ] 支持上传封面图片（通过 `media/upload_img` API）
- [ ] 支持创建草稿文章（`draft/add` API）
- [ ] 支持提交草稿发布（`freepublish/submit` API）
- [ ] check-wechat-token.cjs 能验证当前 Token 有效性（exit 0 / exit 1 / exit 2）
- [ ] SKILL.md 文档完整，包含使用方式和架构说明

**非目标**:
- 不实现视频发布
- 不实现评论管理
- 不集成 N8N（后续单独 PR）
- 不实现 CDP 浏览器自动化（官方 API 方案）

### 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| APPID/APPSECRET 缺失 | 退出并打印清晰错误信息 | P2 | wechat_missing_credentials |
| Access Token 获取失败（网络/凭据错误） | 重试 3 次，仍失败则 exit 1 | P1 | wechat_token_failure |
| 封面图上传失败 | 打印错误，继续尝试发布（无图封面） | P2 | wechat_image_upload_failure |
| 草稿创建失败 | exit 1，打印 errcode + errmsg | P1 | wechat_draft_failure |
| 发布提交失败 | exit 1，打印 errcode + errmsg | P1 | wechat_publish_failure |
