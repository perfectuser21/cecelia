# PRD: 修复 Sprint Contract Gate — Evaluator 必须先独立生成测试计划再比对

## 背景

诊断发现 Sprint Contract Gate 已形同虚设：
- CRG：12次审查中11次 stats 全零（{blocker:0,warning:0,info:0}），100% PASS
- spec_review：18次审查中12次 independent_test_plans 数组为空（Evaluator 没独立工作就直接通过）
- 两个 seal 文件的 `reviewer_model` 字段均为 `undefined`（字段从未被写入）

根本原因：缺少"必须有所发现"的最低压力规则，Evaluator 可以零观察通过。

## 修复范围

1. CRG (code-review-gate/SKILL.md v1.6.0 → v1.7.0)：添加 reviewer_model 字段；添加全零统计强制输出至少1条 info 规则
2. spec_review (spec-review/SKILL.md v1.5.0 → v1.6.0)：添加 reviewer_model 字段；添加 independent_test_plans.length = 0 时 exit 2 规则
3. 01-spec.md (v3.1.0 → v3.2.0)：Sprint Contract Gate PASS 后额外验证 plans.length > 0
4. Engine 版本 bump：13.68.0 → 13.69.0（6个文件）
