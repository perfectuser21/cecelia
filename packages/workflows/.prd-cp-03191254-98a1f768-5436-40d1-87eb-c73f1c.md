# PRD: 图文三平台 NAS 批量发布统一调度

## 背景
头条已有 Mac mini scheduler.sh 支持从 NAS 读取内容批量发布。知乎/公众号缺少 NAS 批量调度，三平台无法统一操作。

## 目标
创建统一批量发布脚本，从 NAS 日期目录读取内容，一键发布到知乎+公众号+头条三平台。

## 成功标准
- bash batch-publish-image-text.sh --date 2026-03-19 --dry-run 输出内容列表并退出 0
- 每个 post-* 子目录包含 platforms.txt（值：zhihu,wechat,toutiao 任意组合）
- 根据 platforms.txt 依次调用各平台发布脚本
- 输出统计报告：{ total: N, success: { zhihu: N, wechat: N, toutiao: N }, failed: [...] }
