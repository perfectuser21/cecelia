## PRD - 知乎文章发布 API 接通

**需求来源**: Brain 自动调度任务 — [P1 KR1] 知乎文章发布 API 接通
**功能描述**: 实现 `packages/workflows/skills/zhihu-publisher/scripts/publish-zhihu-article.cjs`，对标 publish-to-toutiao.py 的 CDP 自动化模式，通过 CDP 连接 Windows PC Chrome（端口 19229）自动发布知乎专栏文章。

**涉及文件**:
- `packages/workflows/skills/zhihu-publisher/scripts/publish-zhihu-article.cjs` — 主发布脚本（新建）
- `packages/workflows/skills/zhihu-publisher/scripts/__tests__/publish-zhihu-article.test.cjs` — 单元测试（新建）

**成功标准**:
- [ ] 脚本可接受 `--content <dir>` 参数，读取目录中 title.txt / content.txt / cover.jpg
- [ ] 通过 CDP 连接 Windows PC 19229 端口，导航到 https://zhuanlan.zhihu.com/write
- [ ] 自动填写标题、正文（知乎 draft-js/quill 编辑器）
- [ ] 如有 cover.jpg 则上传封面图
- [ ] 点击发布按钮完成发布
- [ ] `--dry-run` 模式：只打印参数，不实际连接
- [ ] 单元测试通过（escapeForJS、isLoginError、isPublishSuccess 等纯函数）

**非目标**:
- 不实现批量发布（batch 脚本留后续）
- 不实现视频发布

## 架构说明

```
Mac mini (本机)
    ↓ HTTP GET :19229/json (获取 tabs)
    ↓ WebSocket CDP
Windows PC (100.97.242.124:19229)
    ↓ Chrome 已登录知乎
知乎专栏发布成功
```

内容目录结构：
```
~/.zhihu-queue/{date}/
├── article-{id}/
│   ├── title.txt   — 标题（必需，60字以内）
│   ├── content.txt — 正文 HTML 或纯文本（必需）
│   └── cover.jpg   — 封面图（可选）
```

## 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| CDP 连接失败 | 报错退出，打印排查命令 | P1 | zhihu_cdp_connect_failed |
| 知乎未登录（跳转登录页） | 报错退出，提示重新登录 | P1 | zhihu_session_expired |
| 标题输入框未找到 | 截图后报错退出 | P1 | zhihu_title_input_not_found |
| 正文编辑器未找到 | 截图后报错退出 | P1 | zhihu_editor_not_found |
| 发布按钮未找到 | 截图后报错退出 | P1 | zhihu_publish_btn_not_found |
| 发布超时（60s内未跳转） | 截图后警告，不失败 | P2 | zhihu_publish_timeout |
