## 查重使用方法

创建前在这里搜"你要解决的问题"关键词，而不是搜名字。

例：
- 想做"自动发微博" → 搜"微博" → 找到 `/weibo-publisher` → 不需要新建
- 想做"图文发布到多个平台" → 搜"多平台" → 无结果 → 可以新建

**有相似但不完全一样** → 给已有 skill 加参数，不要新建。
**完全没有** → 用 `/skill-creator` 创建，完成后把新 skill 加到本文件。
