# /create - 创建内容

根据主题或灵感创建新内容。

## 触发方式

```
/create <主题或想法>
/create 深度帖 AI时代的个人竞争力
/create 短贴 关于信息过载
```

## 内容类型

| 类型 | 说明 | 保存位置 |
|------|------|----------|
| 深度帖 | 完整论述，有结构 | content/deep-posts/ |
| 短贴 | 精炼观点 | content/short-posts/ |
| 文章 | 长文，多段落 | content/articles/ |

## 创作原则

### 1. 先定受众
- 这篇写给谁？
- 他们关心什么？
- 他们的痛点是什么？

### 2. 核心观点
- 一句话说清楚你要表达什么
- 观点要有锐度，不要平庸

### 3. 结构清晰
- 深度帖：痛点 → 观点 → 论证 → 金句
- 短贴：钩子 → 观点 → 行动

### 4. 金句收尾
- 最后一句要值得转发

## 使用示例

```
/create 深度帖 为什么大多数人用不好AI
```

执行：
1. 确认主题和类型
2. 参考已有爆款的结构
3. 生成初稿
4. 提供优化建议
5. 保存到对应目录

## 输出格式

```markdown
---
title: "为什么大多数人用不好 AI"
date: 2026-01-28
type: 深度帖
theme: 创作系统
status: 草稿
is_hit: false
is_premium: false
---

[正文内容]
```

## 参考素材

创作时可参考：
- content/ 下的已有内容
- 用户提供的灵感/素材
- 指定的写作风格
