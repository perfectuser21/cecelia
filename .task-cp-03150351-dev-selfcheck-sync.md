# Task Card: /dev 步骤文件 sync + 自检命令

## 需求（What & Why）
**功能描述**: 
1. 同步 packages/engine/skills/dev/steps/ 的 4 个新步骤文件（02-code、03-prci、04-learning、05-clean）到 ~/.claude/skills/dev/steps/
2. 在关键步骤加硬性自检命令，让 Claude 跑完一步后必须自己验证格式正确，否则不能继续

3. [ARTIFACT] 02-code.md 的 DoD 区块包含假测试检测命令（grep echo.*exit）
