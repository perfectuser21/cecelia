# PRD: 修复 088 migration ADD CONSTRAINT 不幂等

## 背景

CI L3 smoke.test.js 在 migration 023 修复后仍然失败：

```
[FAIL] 088_progress_ledger.sql: constraint "chk_progress_ledger_status" for relation "progress_ledger" already exists
[smoke] Server exited with code 1
```

## 问题

`088_progress_ledger.sql` 中 4 个 `ALTER TABLE ... ADD CONSTRAINT` 语句没有幂等保护。PostgreSQL 不支持 `ADD CONSTRAINT IF NOT EXISTS`，必须用 `DO $$ BEGIN ... EXCEPTION WHEN duplicate_object THEN NULL; END $$` 包裹。

## 目标

为 088 migration 中所有 `ADD CONSTRAINT` 语句加幂等保护，使 migration 可安全重复执行。

## 成功标准

1. [BEHAVIOR] 088 migration 可重复执行，不因 "already exists" 报错
2. [ARTIFACT] 所有 ADD CONSTRAINT 均用 DO $$ EXCEPTION WHEN duplicate_object 包裹
3. [GATE] CI L3 smoke.test.js 通过
