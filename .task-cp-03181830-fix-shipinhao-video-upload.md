# Task Card: 修复视频号 iframe 视频上传

## 需求

视频号发布页面使用嵌套 iframe 结构，直接操作 page 对象无法找到上传按钮。
需要实现正确穿透 iframe 的视频上传脚本。

## 成功标准

- 脚本文件存在于 `packages/workflows/skills/shipinhao-publisher/scripts/publish-shipinhao-video.cjs`
- 使用 `frameLocator` 或 `page.frames()` 穿透 iframe 层级
- 支持 `--title`、`--video`、`--desc`、`--dry-run` 参数
- Windows SSH 文件传输 + CDP Playwright 自动化上传
