# Task Card: engine 垃圾代码清理

## 需求（What & Why）
**功能描述**: 删除 engine 中的死文件和垃圾代码，减少维护负担：
1. 删除 `packages/engine/regression-contract.yaml.bak`（v13.7.5 旧备份，当前 v14.x）
