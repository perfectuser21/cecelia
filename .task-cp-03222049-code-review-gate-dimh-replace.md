# Task Card: code-review-gate 维度H升级 — 替代性内容未删旧内容=blocker

## 需求（What & Why）

**功能描述**: 在 code-review-gate 维度H 新增 blocker 级检查：AI 改动内容时新增了替代性描述但未删除旧描述，视为 blocker。同时在 Stage 2 加编辑规则约束。

**背景**: AI 默认行为是追加不是替换。改 A→B 时只加 B，A 还在。多轮后 A/B/C 全在，全是旧信息。维度H 目前只检测 warning 级的"引用已删除路径"，无法拦截"加新不删旧"的核心问题。

**不做什么**: 不改 CI 脚本、hooks 代码、其他 step 文件。
