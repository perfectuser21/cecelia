# PRD: conversation-digest — Brain 自动读取 Claude Code 对话日志

## 背景

用户与 Claude Code 的规划对话目前完全没有被捕获。对话日志以 .jsonl 格式写在 ~/.claude-account1/projects/{slug}/ 下，Brain 在同一台机器，可以直接读取。

不依赖 LLM 记忆（prompt 注入），而是机械地读取 artifact，由 Brain 异步分析。

## 成功标准

1. [ARTIFACT] 新增 DB 表 conversation_log_cursors（tracking 哪些文件/行已处理）
2. [ARTIFACT] 新增 packages/brain/src/conversation-digest.js 模块
3. [BEHAVIOR] Brain tick 每轮检测新日志内容，发现满足阈值的 session 后触发分析
4. [ARTIFACT] Cortex 分析结果写入 conversation_captures + decisions 表
5. [ARTIFACT] selfcheck.js EXPECTED_SCHEMA_VERSION = 196
