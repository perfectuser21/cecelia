# Task Card: OKR PR12 — 清除旧表最终依赖 + DROP goals/projects

## 需求（What & Why）

**功能描述**: 清除 `goals`、`projects`、`project_kr_links` 三张旧表的所有生产代码引用，迁移 `goal_evaluations.goal_id` FK，最终 DROP 三张旧表，完成 OKR 新表链路的收尾。

**背景**: 新 OKR 表链路已完整（visions→objectives→key_results→okr_projects→okr_scopes→okr_initiatives→tasks），旧表 goals/projects 仍被部分生产代码引用，且存在大量下游表的 FK 约束（tasks.goal_id、goal_evaluations.goal_id、reflections.project_id 等），导致无法直接 DROP。本 PR 分步清除所有阻碍并执行 DROP。

**不做什么**:
- 不修改 OKR 新表结构（okr_* 表）
- 不修改 tasks.goal_id 列（该列继续存在，仅 FK 约束从 goals→key_results 迁移）
- 不改业务逻辑，只改旧表引用为新表等效查询
