# PRD: Pipeline Security Bug Fix

## 背景

修复 /dev 工作流中4个 P0/P1 安全漏洞。

## 功能描述

修复以下安全漏洞：
1. `packages/engine/lib/devloop-check.sh` seal文件只检查存在性→加内容校验（verdict字段非空、divergence_count字段非空）
2. `packages/engine/hooks/stop-dev.sh` fallback路径降级不安全→改为exit 2拒绝降级
3. `packages/engine/lib/lock-utils.sh` flock不可用时静默返回0→改为硬错误返回1
4. `packages/engine/lib/devloop-check.sh` PR合并前无二次确认→合并前重新check mergeable+CI

## 成功标准

- seal 文件 verdict 字段为空时 devloop-check.sh 阻断
- seal 文件 divergence_count 字段缺失时 devloop-check.sh 阻断
- devloop-check.sh 未加载时 stop-dev.sh 以 exit 2 拒绝降级
- flock 不可用时 acquire_dev_mode_lock 返回 1（失败）
- PR 合并前重新检查 mergeable + CI 状态
