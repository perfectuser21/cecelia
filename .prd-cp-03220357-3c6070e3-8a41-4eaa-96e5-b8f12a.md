# PRD: intent-expand集成Stage1 + Learning检索

## 背景
任务来自 Brain 自动调度（agent_ops 领域）。
Stage 1 Spec 流程的 `fetch-task-prd.sh` 脚本缺少两个关键功能：
1. 查询任务关联的 KR/Initiative/OKR 战略上下文（intent-expand），用于丰富 PRD 内容
2. 检索 docs/learnings/ 推荐相关历史 Learning，帮助 Agent 避免重复踩坑

01-spec.md 已有这两个功能的文档章节（1.1.5 和 1.3.5），但 fetch-task-prd.sh 未实现。

## 功能描述
1. 在 `fetch-task-prd.sh` 中新增 `fetch_intent_context` 函数，查询 goal_id/project_id 链路（KR → OKR → Vision），将战略上下文注入生成的 PRD
2. 新增 `search_related_learnings` 函数，从任务标题提取关键词，搜索 `docs/learnings/`，推荐相关历史 Learning 文件

## 成功标准
- [ ] fetch-task-prd.sh 包含 `fetch_intent_context` 函数
- [ ] fetch-task-prd.sh 包含 `docs/learnings` 搜索逻辑
- [ ] PRD 在有上下文时包含"战略上下文"章节
