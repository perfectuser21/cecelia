# PRD: AI 热点选题引擎 + 每日自动排期（Topic Selector v1）

## 背景

Initiative「选题+文案自动化（AI驱动）」目标：每日输出 ≥ 10 个可用选题，人工介入时间 = 0。
当前内容流水线已支持 4 阶段执行，但触发点依赖人工输入 keyword，缺少自动选题层。

## 需求

1. `packages/brain/src/topic-selector.js`（新建）— 调用 Claude API 生成选题
2. `packages/brain/src/topic-selection-scheduler.js`（新建）— 每日 UTC 01:00 触发，创建 ≥10 个 content-pipeline tasks
3. `packages/brain/migrations/162_add_topic_selection_log.sql`（新建）— 选题日志表
4. `packages/brain/src/tick.js`（修改）— 集成每日选题调度器
5. `packages/brain/src/__tests__/topic-selection-scheduler.test.js`（新建）— vitest 单元测试

## 成功标准

- Brain 每天北京时间 09:00 自动创建 ≥ 10 个 content-pipeline tasks
- 每个 task 含 `payload.pipeline_keyword`、`payload.trigger_source: daily_topic_selection`
- 当天已触发时不重复创建
- `topic_selection_log` 表记录每次选题
- 所有 vitest 单元测试通过

## 范围边界

- 不修改 content-pipeline-executors.js
- 不做前端 UI
- 不集成第三方热点 API（MVP 用 Claude 推断）
