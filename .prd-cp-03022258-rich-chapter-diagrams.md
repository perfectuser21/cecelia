# PRD：各章 SVG 配图深度重绘——全细节 + 风险标注

## 背景

说明书手风琴布局已完成（PR #319），但展开章节后的 SVG 图太简单：
- 感知层只画了几个信号，实际有 16 个感知信号
- 意识核心没有清楚展示三层架构（L0/L1/L2）
- 其他章节图也需要完整化

## 需求

每章展开时，SVG 配图应该是量身定制的完整图，包含该章节的所有核心细节。

### 第 1 章 外界接口
- 双通道：飞书/WebSocket 对话通道 + 定时器 Tick 通道
- orchestrator-chat → 丘脑 快速路径
- tick → executor 执行路径
- ⚠️ 风险：接口断开=大脑失联

### 第 2 章 感知层
- 全部 16 个感知信号（从代码读取实际信号名）
- 按类型分组
- ⚠️ 风险：信号缺失=感知盲区

### 第 3 章 意识核心
- 三层：L0 脑干（代码）/ L1 丘脑（Haiku）/ L2 皮层（Sonnet）
- 决策流向：事件 → L1 分类 → L0 执行 or L2 深析
- ⚠️ 风险：LLM 超时=决策瘫痪

### 第 4 章 行动层
- 完整执行链：任务队列 → 资源检查 → 选 Agent → Bridge → Claude
- 熔断机制（并发限制、失败隔离）
- ⚠️ 风险：Bridge 超时=任务积压

### 第 5 章 自我演化
- 四模块：learning / memory / self-model / self-report
- 数据流：经验 → 学习 → 记忆 → 自我认知 → 行为调整
- ⚠️ 风险：学习循环断裂=停止进化

## 技术要求
- 修改 `apps/api/features/brain/pages/SuperBrain.tsx` 的 ChapterDiagram 组件
- SVG 高度 300-400px，宽度自适应
- 暗色主题，⚠️ 用橙色显示
