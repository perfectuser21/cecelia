# PRD: 补充 INFRA 能力元数据，减少 capability-scanner 白名单依赖

## 背景
PR #1463 为 capability-scanner 添加了 INFRA_DEPLOYED_CAPABILITIES 白名单（9个能力）。但这些能力在 capabilities 表中缺少 related_skills/key_tables 元数据，导致扫描器完全依赖 JS 硬编码集合。

## 目标
为 5 个有间接证据的 INFRA 能力补充 DB 元数据（migration 182），使其无需依赖白名单即可被自主检测：
- branch-protection-hooks → related_skills=['dev']
- ci-devgate-quality → related_skills=['dev']
- brain-deployment → related_skills=['dev']
- cecelia-dashboard → key_tables=['tasks']
- zenithjoy-dashboard → key_tables=['tasks']
