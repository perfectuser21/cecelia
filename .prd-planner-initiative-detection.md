## PRD - 增强 Planner 识别有 Initiative 无 Task 的 KR 并自动派发

**需求来源**: Brain 自动调度任务
**功能描述**:
当前 Planner 没有识别出"有 Initiative 但没有 Task"的 KR 的信号，
导致某些 KR（如 KR4）长期无法派发任务。需要增强 Planner 的识别逻辑，
自动发现需要拆解的 Initiative 并生成 initiative_plan 任务。

**涉及文件**:
- `packages/brain/src/planner.js` - 核心 Planner 逻辑
- `packages/brain/src/server.js` - 新增监控端点
- `packages/brain/tests/planner.test.js` - 单元测试

**成功标准**:
1. scoreKRs() 能识别有 Initiative 无 Task 的 KR，给该 KR +15 分
2. selectTargetProject() 优先选择需要拆解的 Initiative
3. generateInitiativePlanTask() 生成正确的 initiative_plan 任务
4. 监控端点 GET /api/brain/planner/initiatives-without-tasks 可用
5. Brain CI 通过，单元测试覆盖率 >= 80%

**非目标**:
- 不修改其他调度逻辑
- 不改变现有 Task 的处理流程
