# PRD: memory-retriever salience 加权召回（PR11）

## 背景

当前 memory-retriever.js 按 importance + 时间衰减召回记忆，但 PR9 引入的 salience_score 字段未被用于加权。导致 IN filter 识别的高价值消息（洞察/决定/纠正）出去时没有优先权。

## 目标

修改 memory-retriever.js，在召回评分公式中加入 salience_score 权重，使高 salience 的记忆在 MMR/排序中获得加分。

## 成功标准

- memory-retriever.js 召回评分公式中包含 salience_score 权重
- salience_score IS NULL 时安全降级为 0（向后兼容）
- 新增测试验证高 salience 记录排序靠前
- 不改变接口签名
