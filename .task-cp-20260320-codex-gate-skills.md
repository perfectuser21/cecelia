# Task Card: 合并审查 Skill 为 4 个 Codex Gate

## 需求（What & Why）
**功能描述**: 合并 5 个重叠的审查任务类型（decomp_review、prd_audit、cto_review、code_quality、initiative_verify）为 4 个清晰的 Gate Skill 定义。
**背景**: 当前审查 Skill 职责重叠，需要整合为统一的 4 阶段 Gate 流水线。
**不做什么**: 不修改现有 Skill 文件，不修改 Brain 代码，只创建新 Skill 定义。

## 成功标准

1. [ARTIFACT] 创建 prd-review/SKILL.md（合并 decomp-check + prd_audit）
2. [ARTIFACT] 创建 spec-review/SKILL.md（合并 dod_verify + cto_review 单 PR 部分）
3. [ARTIFACT] 创建 code-review-gate/SKILL.md（合并 code_quality + /simplify）
4. [ARTIFACT] 创建 initiative-review/SKILL.md（合并 initiative_verify + cto_review 整体部分）
5. [GATE] 所有 SKILL.md 包含版本号、触发条件、输入参数、输出格式、审查维度
