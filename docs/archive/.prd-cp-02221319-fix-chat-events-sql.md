# PRD: 修复 orchestrator-chat 事件记录 SQL

## 功能描述

修复 orchestrator-chat.js 中 recordChatEvent 的 SQL 列名错误。

## 成功标准

- 对话事件正确写入 cecelia_events 表
- 端到端测试通过

## 问题

cecelia_events 表结构：`(id, event_type, source, payload, created_at)`

orchestrator-chat.js 使用了错误的列名：
- `type` → 应为 `event_type`
- `data` → 应为 `payload`（jsonb）
- 缺少 `source` 列

## 修复

修改 `recordChatEvent()` 的 INSERT 语句。
