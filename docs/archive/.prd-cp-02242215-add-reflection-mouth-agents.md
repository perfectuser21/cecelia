# PRD: 添加 reflection 和 mouth 到 model-registry

## 功能描述
在 Brain 的 model-registry.js 中添加 `reflection`（L3 反思层）和 `mouth`（嘴巴）两个 brain-layer agents，使配置页可以正确保存这两层的模型选择。

## 背景
- Config 页尝试保存 `reflection`/`mouth` agent 时 API 报 "Unknown agent" 错误
- 嘴巴层已改用 MiniMax API，但 model-registry 没有 `mouth` agent 定义
- reflection 层使用 Anthropic API，但没有 `reflection` agent 定义

## 成功标准
- [ ] PATCH /api/brain/model-profiles/active/agent 接受 agent_id=reflection
- [ ] PATCH /api/brain/model-profiles/active/agent 接受 agent_id=mouth
- [ ] mouth 的 allowed_models 包含 MiniMax 选项
- [ ] 现有 thalamus/cortex agent 行为不变
- [ ] 现有测试全部通过
