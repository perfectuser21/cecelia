## PRD - v1.65.0 渐进验证循环（Progressive Validation）

**需求来源**: OKR 渐进式拆解与验证系统 — PR 3/4
**功能描述**: Project 完成后对比预期，收集执行数据，审查后调整后续 Project 计划
**涉及文件**:
- `brain/src/progress-reviewer.js` — 新文件，渐进验证核心逻辑
- `brain/src/initiative-closer.js` — Project 完成时触发审查
- `brain/src/routes.js` — execution-callback 处理 plan_adjustment

**成功标准**:
1. reviewProjectCompletion 收集完成数据并对比时间预算
2. shouldAdjustPlan 判断是否需要调整后续 Project
3. createPlanAdjustmentTask 创建 Vivian 审查任务
4. initiative-closer Project 完成时自动触发审查
5. execution-callback 处理 plan_adjustment verdict

**非目标**:
- 不修改秋米 prompt（PR 4）
- 不修改 decomp_reviews 表结构（PR 2 已创建）
