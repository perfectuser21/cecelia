# Learning: 社媒热点 API 接入 TimescaleDB

**Branch**: cp-03311325-7340eb55-a374-4e62-9497-e0995e  
**Date**: 2026-03-31

### 根本原因

Brain 现有路由只连接 Brain PostgreSQL（cecelia DB），缺乏访问 HK VPS 上 TimescaleDB 社媒数据的能力，导致内容 pipeline 无法感知社媒热点趋势。

### 解决方案

新增独立 pg.Pool 连接 TimescaleDB（通过 TIMESCALE_* 环境变量配置），查询 `v_all_platforms` 视图，降级策略为连接失败时返回空数组（不报错）。
