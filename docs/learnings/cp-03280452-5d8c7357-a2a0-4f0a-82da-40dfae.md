# Learning: CI L2 扩展 — Brain 新文件强制登记知识文档

## 变更摘要

扩展 `ci-l2-consistency.yml` 的 `brain-l2` job，新增 PR-only Impact Check step，
监控 `packages/brain/src/*.js` 新增文件，强制同步更新 `.agent-knowledge/brain.md` 或 `DEFINITION.md`。

### 根本原因

Brain 新增源文件（如新模块 `xxx.js`）时没有门禁机制确保知识文档同步更新，
导致 `.agent-knowledge/brain.md` 和 `DEFINITION.md` 落后于实际代码状态，
AI Agent 对系统组件的感知会失真，增加后续开发出错概率。

### 下次预防

- [ ] 新增任何 Brain 模块时，同步在 `.agent-knowledge/brain.md` 的对应章节描述职责
- [ ] 如果是核心架构变更，同步更新 `DEFINITION.md`
- [ ] 本 CI check 会自动提醒：失败信息中包含 Fix 提示，按提示更新即可
