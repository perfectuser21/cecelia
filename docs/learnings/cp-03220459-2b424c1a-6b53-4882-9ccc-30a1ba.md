## 背景

### 根本原因

- [ ] 添加新测试时，检查同文件中是否有结构相似的路径（matchAll 模式比 indexOf 更适合多次出现的场景）
- [ ] `devloop-check-gates.test.ts` 的 stop-dev.sh describe block 应保持 spec/code_review 对称结构

在 `stop-dev.sh — 15min 超时自动 pass 降级机制` describe block 中新增测试：
- 使用 `matchAll(/-gt 900/g)` 找到所有超时块（比 indexOf 更健壮）
