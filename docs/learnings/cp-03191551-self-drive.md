# Learning: 自驱引擎

## 分支
`cp-03191551-self-drive`

## 变更摘要
新增 self-drive.js，每 12 小时读取 probe + scan 体检数据，调 LLM 分析优先级，自动创建任务。

### 根本原因

Cecelia 有了"眼睛"（Probe + Scanner），能看到自己哪里通哪里断、哪些能力是孤岛。但看到之后不会行动——还是等用户下指令。

缺少的是"自驱力"：看到问题 → 分析优先级 → 自己创建任务 → 自己执行。
