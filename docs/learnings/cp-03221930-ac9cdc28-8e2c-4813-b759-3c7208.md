# Learning: 蒸馏文档 SELF_MODEL/USER_PROFILE 更新频率调整

## 任务摘要
在 `packages/brain/server.js` 的 Layer 2 蒸馏文档初始化块中，增加了 `SELF_MODEL`（每24h）和 `USER_PROFILE`（每6h）的定时刷新调度。

## 实现要点
- `refreshSelfModel` 和 `refreshUserProfile` 函数已在 `distilled-docs.js` 中存在，只需在 server.js 中导入并调用
- 与现有 WORLD_STATE 模式保持一致：setTimeout 首次触发 + setInterval 定期触发
- SELF_MODEL 延迟 15s 首次触发（比 WORLD_STATE 的 10s 晚），USER_PROFILE 延迟 20s 首次触发，避免启动时集中压力

### 根本原因
Layer 2 蒸馏文档层的设计意图是让三类动态文档（WORLD_STATE/SELF_MODEL/USER_PROFILE）都能自动刷新，但最初实现时只给 WORLD_STATE 加了 cron，另外两个函数虽然存在但没有被排程。
