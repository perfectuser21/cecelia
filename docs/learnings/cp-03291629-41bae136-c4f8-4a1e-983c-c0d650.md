# Learning: 移除 /dev Stage 3 Playwright Evaluator

## 任务描述
从 /dev pipeline 的 Stage 3 移除 Playwright Evaluator 步骤（条件 4.5），简化 Stage 3 执行路径。

### 根本原因
Playwright Evaluator 层增加了 CI 复杂度，且在 CI 环境（无浏览器）下难以可靠运行。决定改为 post-merge 触发。

### 下次预防
- [ ] 移除一个步骤时，需同时检查：对应测试文件（tests/scripts/）、regression-contract.yaml RCI 条目、feature-registry.yml changelog
- [ ] devloop-check.sh 条件顺序变更后，验证条件间无逻辑断层（1→1.5→2→2.5→2.6→2.7→3→4→5）
- [ ] 删除功能的测试文件，否则 CI 测试失败
