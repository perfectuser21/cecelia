**Branch**: cp-03270407-39a6e137-4890-4317-a2bd-d981cb
**Date**: 2026-03-27

## 功能总结

在 KnowledgeHome.tsx 中添加第5张卡片「深度知识页」，指向 `/knowledge/modules` 页面（由 PR #1620 实现）。

### 根本原因

PR #1620 实现了 /knowledge/modules 页面和 Brain API，但 PR 范围仅聚焦于路由和页面组件，遗漏了 KnowledgeHome.tsx 导航入口的同步更新。
功能路由已通过 `index.ts` 的 `navItem` 配置注册到侧边栏，说明导航链接存在，但 Knowledge Hub 主页（KnowledgeHome）缺少对应入口卡片。
根本原因是新功能 PR 的验收清单没有包含「父级 Home 页入口更新」，导致已上线功能的可发现性降低。
