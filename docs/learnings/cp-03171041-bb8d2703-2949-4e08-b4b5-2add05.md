## 清理 Engine RCI 垃圾条目 + 新增健康守护脚本（2026-03-17）

### 根本原因

本次开发流程顺畅，CI 0 次失败，本地验证 0 次失败。唯一的非显而易见问题：

root 目录 `package-lock.json` 中有 `packages/engine` 条目，版本号停留在 12.84.0（远低于当前 12.93.x），容易在 Engine 版本 bump 时遗漏。
原因：root package-lock.json 是 monorepo workspace 引用，其中每个 workspace 包都有独立的 version 字段，但日常版本 bump 只更新 `packages/engine/` 内的文件，忘记同步 root 锁文件。
