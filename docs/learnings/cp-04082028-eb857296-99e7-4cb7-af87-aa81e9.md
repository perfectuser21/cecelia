## 概述

KR4（geo SEO网站上线）在 Brain 中显示 25% 进度，但网站实际已完全满足验收标准。进度未及时更新导致 SelfDrive 持续派发重复诊断任务。

### 根本原因

1. **KR 进度未在 PR 合并后回写**：PR #2115 修复并部署后未执行 `key_results.progress` 更新，Brain 仍显示旧进度（25）。
2. **Brain 无 PATCH KR API**：没有 `PATCH /api/brain/okr/key_results/:id` 路由，需直接 SQL 更新。
3. **字段名陷阱**：`key_results` 表用 `progress`（int），而非 API 返回的 `progress_pct`。需同时更新 `progress` 和 `current_value`。

- [ ] `key_results.progress` = int（0-100），`current_value` = numeric，两者都要更新
