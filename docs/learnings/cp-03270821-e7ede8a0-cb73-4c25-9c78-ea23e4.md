# Learning: 文档+聊天分栏界面

**分支**: cp-03270821-e7ede8a0-cb73-4c25-9c78-ea23e4
**日期**: 2026-03-27

## 功能概述

新增 Notion AI 风格的文档+聊天分栏界面（DocEditor/DocList），配合 Brain doc-chat 端点，
实现 Claude 可通过 `<doc_update>` 标记实时更新左侧文档内容。

---

### 根本原因

code_review_gate 发现两个问题：

1. **XSS 漏洞**：`renderMarkdown` 直接将用户/AI 生成内容插入 `dangerouslySetInnerHTML`，
   未对 HTML 特殊字符转义，AI 通过 prompt injection 可触发 XSS。

2. **callLLM 未注册 agent ID**：使用 `callLLM('doc-editor', ...)` 但 model-profile 无此配置，
   依赖 fallback 行为，隐式而脆弱。

---

### 下次预防

- [ ] 凡使用 `dangerouslySetInnerHTML` 的地方，必须先对输入执行 `escapeHtml()` 再做 markdown 转换
- [ ] `callLLM(agentId, ...)` 只能用已注册的 agent ID（`mouth`/`thalamus`/`perception`），
  若需自定义模型，用 `options.model` 覆盖，不要创造新 agent ID
- [ ] 新建 AI 内容渲染组件时，默认假设输入不可信，先转义再渲染
