# Learning: fix(brain): status.js 缺失 IDEMPOTENCY_TTL/ALLOWED_ACTIONS 导入

## 根本原因

`packages/brain/src/routes/status.js` 在使用 `IDEMPOTENCY_TTL` 和 `ALLOWED_ACTIONS` 时，import 语句遗漏了这两个常量，导致 `/api/brain/status` 每次调用都抛出 `ReferenceError: IDEMPOTENCY_TTL is not defined`。

## 修复

在 import 语句中补充 `IDEMPOTENCY_TTL` 和 `ALLOWED_ACTIONS`：

```js
import { getActivePolicy, getWorkingMemory, getTopTasks, getRecentDecisions, IDEMPOTENCY_TTL, ALLOWED_ACTIONS } from './shared.js';
```

## 下次预防

- [ ] 新增对常量的使用时，立即检查 import 是否包含该常量
- [ ] Brain 路由文件使用共享常量时，需从 `./shared.js` 显式导入
