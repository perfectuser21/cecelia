# Brain 重启使新代码生效 (PR #874)

## 场景

PR #874（billing pause 熔断）合并后，Brain 进程仍运行旧代码，`/api/brain/billing-pause` 返回 404。

## 操作

Brain 由 LaunchDaemon 管理（`/Library/LaunchDaemons/com.cecelia.brain.plist`），`KeepAlive=true`。

重启方式：
```bash
sudo launchctl stop com.cecelia.brain
# KeepAlive 自动重启，约 5-10 秒后进程恢复
```

## 注意事项

1. **30min tick 保护**：重启后 Brain 检测上次 tick 距今 < 30 分钟，不自动启动 tick loop。这是正常保护机制，30 分钟后自动恢复，或 `POST /api/brain/tick` 手动触发。

2. **health=degraded**：重启后 scheduler 暂未启动，health 显示 degraded，属预期行为。

3. **验证方式**：
   ```bash
   curl -s localhost:5221/api/brain/billing-pause  # {"success":true,"active":false}
   curl -X POST localhost:5221/api/brain/tick       # 手动触发 tick 验证
   ```
