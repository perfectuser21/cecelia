# Learning: Gate 5 B1+B2 凭据健康巡检 + 每日 E2E smoke

## 根本原因

lint-test-pairing 规则要求测试文件命名必须与源文件名完全匹配（`<name>.test.js` 或 `__tests__/<name>.test.js`），否则 CI 拒绝合并。历史上 `credentials-health.test.js` 和 `daily-smoke.test.js` 的命名不符合该规则。

smoke.sh 中通过 `node --input-type=module` 直接 import ESM 模块会失败（依赖链需要 node_modules，worktree 无 node_modules）。改为用 `node -e + readFileSync + 正则提取常量` 方式验证纯函数逻辑，无需 import 链。
