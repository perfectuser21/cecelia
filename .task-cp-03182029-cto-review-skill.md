# Task Card: cto-review skill

## 需求（What & Why）

**功能描述**: 实现 `/cto-review` skill，供 Brain 派发的 `cto_review` 类型 task 在西安 Codex 执行。以 CTO 视角对 PR 进行整体审查：enriched PRD + DoD + 核心代码 diff，判断实现是否真正交付了用户意图。

**背景**: Brain 系统已有 `intent_expand` 和 `cto_review` 两阶段前置审查流程（最新 commit #1083 已合并），但 `cto-review` skill 本体尚未创建。需要补全这个 skill 的 SKILL.md 文件。

**不做什么**:
- 不实现逐行代码质量审查（那是 /review 的职责）
- 不触碰 `packages/workflows/skills/intent-expand/`（另一个 agent 负责）
- 不修改 Brain 后端代码

## 成功标准

1. [ARTIFACT] `packages/workflows/skills/cto-review/SKILL.md` 文件存在
2. [ARTIFACT] `packages/workflows/skills/cto-review/review` 脚本目录存在（或直接内嵌执行说明）
3. [BEHAVIOR] skill 包含完整的 5 步执行流程（获取 enriched PRD → DoD → diff → 审查 → 输出结果）
4. [BEHAVIOR] skill 输出格式包含 `REVIEW_RESULT: PASS` 或 `REVIEW_RESULT: FAIL + FAIL_REASONS`
5. [GATE] CI 全部通过
