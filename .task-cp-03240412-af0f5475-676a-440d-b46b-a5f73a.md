# Task Card: Knowledge & Documentation System

## 需求（What & Why）

**功能描述**: 构建统一文档与知识系统，包含 3 张新表 + 3 组 Brain API + 5 个前端页面 + PR 自动写入 + 每日日报 Tick。

**背景**: Cecelia 目前缺乏统一的开发记录、设计文档和用户批注存储，学习材料分散在 component_evolutions / learnings / knowledge 等多处，无法形成结构化的知识地图。本次将新建 `dev_records` / `design_docs` / `user_annotations` 三张表，并提供 Brain API 和 5 个前端页面。

**不做什么**:
- 不修改现有 component_evolutions / learnings / knowledge 表结构
- 不做全文搜索（搜索留后续）
- 前端页面只做基础列表，不做编辑器富文本
