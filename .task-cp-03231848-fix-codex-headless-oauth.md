# Task Card: fix-codex-headless-oauth

## 背景
callCodexHeadless 强制注入 OPENAI_API_KEY，导致 codex exec 走直接 API 计费而非 OAuth 订阅账号，造成 $6 意外消耗。

## 目标
修复 callCodexHeadless：优先使用 ~/.codex-teamX OAuth 账号，fallback 到 API key。

## 成功标准

- [ ] [ARTIFACT] callCodexHeadless 代码不再包含 `OPENAI_API_KEY: apiKey` 强制注入
- [ ] [ARTIFACT] getNextCodexTeamHome() 函数存在，实现 round-robin 轮换 team1/team2
- [ ] [BEHAVIOR] CODEX_HOME 被设置为 OAuth team 账号目录时，env 不含 OPENAI_API_KEY
- [ ] [BEHAVIOR] team 账号 auth.json 存在且有 tokens 时选用该账号
- [ ] [BEHAVIOR] 无可用 team 账号时 fallback 到 API key
- [ ] [GATE] npm test 通过（callCodexHeadless 相关单元测试）
