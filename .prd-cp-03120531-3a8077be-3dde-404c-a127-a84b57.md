## PRD - learning-quality-scorer.js — 基于内容特征自动评分（空壳检测规则）

**需求来源**: Brain 自动调度 — 知识过滤器 Initiative PR2
**功能描述**: 新建 learning-quality-scorer.js 模块，实装评分器，在 learning 写入时自动计算 quality_score + source_type
**涉及文件**:
- `packages/brain/src/learning-quality-scorer.js`（新建）
- `packages/brain/src/actions.js` 或 learning 写入逻辑（调用评分器）
- `packages/brain/src/__tests__/learning-quality-scorer.test.js`（新建单测）

**成功标准**:
- 评分器对已知空壳样本打分 < 40
- 真实 learning 样本打分 >= 60
- 单测覆盖各规则分支

**非目标**: 不修改 DB schema，不改 learning 的存储字段名（假设 quality_score 字段已存在）

## 成功标准

- [ ] 新建 learning-quality-scorer.js，导出 scoreLearning(content) 函数
- [ ] 低分规则（< 40）实装：内容长度 < 20 字、含 test/ok/pass 等空壳词、无根因/无改进点、纯格式套话
- [ ] learning 写入时自动调用评分器填充 quality_score + source_type
- [ ] 空壳样本打分 < 40
- [ ] 真实 learning 样本打分 >= 60
- [ ] 单测覆盖各规则分支

### 异常处理 & 报警规范

N/A - 无 IO 操作（纯函数评分器）
