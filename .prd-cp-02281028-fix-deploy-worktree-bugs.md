# PRD: 修复 deploy-local.sh worktree 路径 bug

## 问题
deploy-local.sh 在 worktree 中执行时有两个严重 bug：
1. Dashboard build 产物在 worktree，但 Docker 容器挂载主仓库 dist/
2. brain-deploy.sh 依赖 .env.docker（gitignore），worktree 无此文件

## 修复
通过 git rev-parse --git-common-dir 找到主仓库路径，所有部署操作在主仓库执行。
