# PRD: 修复 actions-domain-role.test.js domain 参数索引断裂

## 背景

PR #768 在 `actions.js createTask()` 末尾新增了 `delivery_type` 参数，导致：
- 12-param 路径变为 13 参数（delivery_type 追加为最后一个）
- 13-param 路径变为 14 参数（delivery_type 追加为最后一个）

`actions-domain-role.test.js` 中多处使用 `params[params.length - 1]` 取 domain，
现在取到的是 delivery_type（'code-only'），导致断言失败。

## 目标

修复 `packages/brain/src/__tests__/actions-domain-role.test.js` 中所有用
`params[params.length - 1]` 取 domain 的地方，改为正确的索引。

## 验收标准

- actions-domain-role.test.js 所有测试通过
- 不修改 actions.js 业务逻辑
