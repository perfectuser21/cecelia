# Task Card: feat(dashboard): DocChatPage v2 — 对话持久化 + 文件选择 + Analyze Capture

## PRD

### 背景
DocChatPage 当前问题：
1. 右栏聊天消息以 Markdown 渲染，应该是纯文本
2. 左栏只能看固定文档，无法切换其他 design_doc 文件
3. 对话历史刷新即丢，无跨会话记忆
4. 没有「分析并捕获」功能——无法把对话/文档内容转成 Brain captures

### 目标

**A. 右栏去 Markdown**：聊天消息纯文本显示（pre-wrap）

**B. 左栏文件选择器**：顶部加下拉，列出所有 design_docs，切换后 URL 跳转、内容刷新

**C. 对话历史持久化**：
- DB：`design_docs` 表加 `chat_history jsonb DEFAULT '[]'`
- 页面打开时加载历史（Brain GET /:id 已返回，扩展即可）
- 每次对话后自动保存（PUT /:id `{ chat_history }` ）
- 前端历史上限 100 条（超出截取最近 100 条存储）

**D. Analyze & Capture（增量去重）**：
- Brain 新增 `POST /:id/analyze`
- 逻辑：查已有 `captures` where `source = 'doc-chat:{id}'`，把文档内容 + 对话 + 已有 captures 传给 Claude
- Claude 只返回新的 actionable items（去重）
- 批量创建 captures（POST `/api/captures`），`source = 'doc-chat:{id}'`
- 返回本次新创建了几条
