# Task Card: CI 支持 Task Card 格式（.task-{branch}.md）

## 需求（What & Why）

**功能描述**: 修改 ci-l1-process.yml，让 cleanup-check 和 required-paths-check 两个 job 能识别新的 `.task-{branch}.md` 格式，同时向后兼容旧的 `.prd-*.md` + `.dod-*.md` 格式。
**背景**: dev 工作流从 v4.0 起改用统一 Task Card（`.task-cp-{branch}.md`），但 CI 仍只认旧格式，导致使用新格式的 PR 无法通过 CI。
**不做什么**: 不修改 check-prd.sh、check-dod-mapping.cjs 等其他脚本；不删除旧格式支持。

## 成功标准

1. [ARTIFACT] cleanup-check job 优先查找 `.task-${GITHUB_HEAD_REF}.md`，找不到再找 `.dod-${GITHUB_HEAD_REF}.md`，最后找 `.dod.md`
2. [ARTIFACT] required-paths-check job 优先查找 `.task-${HEAD_REF}.md`，找不到再找 `.prd-${HEAD_REF}.md`，最后找 `.prd-task-*.md`
3. [BEHAVIOR] 旧格式 `.prd-*.md` + `.dod-*.md` 继续有效，不受影响
4. [GATE] CI 全部通过

## 验收条件（DoD）

### 功能验收

- [ ] [ARTIFACT] cleanup-check job 的 DoD 文件查找逻辑新增 task card 支持
  Test: manual:bash -c "grep -c 'task-' .github/workflows/ci-l1-process.yml"

- [ ] [ARTIFACT] required-paths-check job 的 PRD 文件查找逻辑新增 task card 支持
  Test: manual:bash -c "grep -c '.task-' .github/workflows/ci-l1-process.yml"

- [ ] [BEHAVIOR] 两处改动均使用 if/elif 结构保持向后兼容
  Test: manual:bash -c "grep -c 'elif' .github/workflows/ci-l1-process.yml"

### 测试验收

- [ ] [GATE] 所有现有测试通过
  Test: manual:bash -c "npm test 2>&1 | tail -5"

## 实现方案

**要改的文件**: `.github/workflows/ci-l1-process.yml`

**改动1 — cleanup-check job（第 121-126 行）**:
```yaml
# 原来：
if [ -f ".dod-${GITHUB_HEAD_REF}.md" ]; then
  DOD_FILE=".dod-${GITHUB_HEAD_REF}.md"
elif [ -f ".dod.md" ]; then
  DOD_FILE=".dod.md"
fi

# 改为（优先 task card）：
if [ -f ".task-${GITHUB_HEAD_REF}.md" ]; then
  DOD_FILE=".task-${GITHUB_HEAD_REF}.md"
elif [ -f ".dod-${GITHUB_HEAD_REF}.md" ]; then
  DOD_FILE=".dod-${GITHUB_HEAD_REF}.md"
elif [ -f ".dod.md" ]; then
  DOD_FILE=".dod.md"
fi
```

**改动2 — required-paths-check job（第 340-341 行）**:
```yaml
# 原来：
PRD_FILE=".prd-${HEAD_REF}.md"
if [ ! -f "$PRD_FILE" ] && ! ls .prd-task-*.md 2>/dev/null | head -1 | grep -q .; then

# 改为（优先 task card）：
if [ -f ".task-${HEAD_REF}.md" ]; then
  PRD_FILE=".task-${HEAD_REF}.md"
elif [ -f ".prd-${HEAD_REF}.md" ]; then
  PRD_FILE=".prd-${HEAD_REF}.md"
else
  PRD_FILE=$(ls .prd-task-*.md 2>/dev/null | head -1)
fi
```
