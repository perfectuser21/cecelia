## PRD - 支持 task payload skill_override 字段

**需求来源**: 用户直接提供
**功能描述**: 在 executor.js 的 preparePrompt() 中，如果 task.payload.skill_override 存在，优先使用它而非基于 task_type 的默认映射。
**涉及文件**: brain/src/executor.js, brain/src/__tests__/executor-skill-override.test.js
**成功标准**:
- preparePrompt() 检查 task.payload?.skill_override，若存在则以它为 skill 而非调用 getSkillForTaskType()
- 有 unit test 覆盖 skill_override 优先于默认映射
- facts-check.mjs 通过
- CI 通过
**非目标**: 不修改 thalamus.js、task-router.js 或任何路由逻辑
