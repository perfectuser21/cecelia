## 背景

PR #1422 新增了 codex/* 和 gpt-5.4-* 系列模型，但只更新了 reflection 器官的 allowed_models。其他器官（thalamus/cortex/mouth/memory/rumination/narrative）仍然只有旧的白名单，导致用户在 Dashboard 切换模型时报错 "Model codex/gpt-5.4 is not allowed for agent thalamus"。

## 目标

在 `packages/brain/src/model-registry.js` 中为所有 6 个器官扩展 `allowed_models`，加入新注册的 codex 和 openai 模型 ID。

## 成功标准

- thalamus/cortex/mouth/memory/rumination/narrative 的 allowed_models 均包含 codex/gpt-5.4-mini、codex/gpt-5.4、gpt-5.4-mini、gpt-5.4
- Dashboard 切换模型时不再报 "is not allowed" 错误
- Brain 测试通过
