# Task Card: Autonomous Sessions Dashboard 页面

## 需求
实时看 /dev --autonomous 跑到哪，每个 session 什么状态。目前无法看到，只能看 log。

## 技术决策（自主）
- **后端 API** 放 Brain（端口 5221），不放 apps/api — 扫 worktree 更直接，Brain 已有 shell exec 能力
- **前端页面** 放 apps/dashboard
- 路由注册沿用 `system-hub/index.ts` 的 router 模式

## 成功标准

### 后端
- 新增 `GET http://localhost:5221/api/brain/autonomous/sessions`
- 实现：`git -C <PROJECT_ROOT> worktree list --porcelain` → 读每个 worktree 下 `.dev-mode.cp-*` → 解析 YAML-like 字段
- 返回 JSON：`{sessions: [{branch, autonomous_mode, harness_mode, owner_session, started, steps: {step_0, step_1, ...}, task_card_path, worktree_path, elapsed_seconds}]}`
- 按 started 降序

### 前端
- apps/dashboard 新增 `/autonomous` 路由
- 页面组件 `AutonomousSessionsPage.tsx`
- 5 秒轮询 `/api/brain/autonomous/sessions`
- UI：每个 session 一张卡片
  - 头：分支名 + badge（🤖 AUTO / 🏗️ HARNESS / 📋 STANDARD）
  - 5 步进度条：Worktree → Spec → Code → Integrate → Ship
  - 已完成 ✓ 绿 / 当前 ⏳ 蓝 pulse / 未开始 灰
  - 已运行时长
- 空态：无活跃 session

### 路由注册
- `apps/api/features/system-hub/index.ts` 加 `/autonomous` 路由（component 名 `AutonomousSessionsPage`）
- 加到 nav 菜单，放在 `/live-monitor` 附近
