# Task Card: 修复 088 migration ADD CONSTRAINT 不幂等

## 需求（What & Why）

CI L3 smoke.test.js 报错：
```
[FAIL] 088_progress_ledger.sql: constraint "chk_progress_ledger_status" already exists
```

PostgreSQL 的 `ALTER TABLE ... ADD CONSTRAINT` 不支持 `IF NOT EXISTS`。需要用 `DO $$ BEGIN ... EXCEPTION WHEN duplicate_object THEN NULL; END $$` 包裹，实现幂等。

## 成功标准

1. [BEHAVIOR] 088 migration 可重复执行，不因 "already exists" 报错
2. [ARTIFACT] 所有 ADD CONSTRAINT 均用 DO $$ EXCEPTION WHEN duplicate_object 包裹
3. [GATE] CI L3 smoke.test.js 通过

## 验收条件（DoD）

- [x] [BEHAVIOR] migration 中所有 ADD CONSTRAINT 均已加幂等保护

- [x] [ARTIFACT] 4 个 DO $$ BEGIN ... END $$ 块存在

- [x] [GATE] migration 语法正确（不含裸 ADD CONSTRAINT）

## 实现方案

**修改文件**：`packages/brain/migrations/088_progress_ledger.sql`

将 4 个 `ALTER TABLE ... ADD CONSTRAINT ...` 改为：
```sql
DO $$ BEGIN
  ALTER TABLE progress_ledger ADD CONSTRAINT chk_progress_ledger_status CHECK (...);
EXCEPTION WHEN duplicate_object THEN NULL;
END $$;
```
