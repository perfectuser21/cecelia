## 需求（What & Why）

**功能描述**：新建 `packages/brain/src/okr-closer.js`，实现 OKR 新表（okr_initiatives / okr_scopes / okr_projects）的完成检测飞轮机制，并集成到 `tick.js`。同时在 `task-router.js` 添加新 task_type 路由，在 `thalamus.js` 添加白名单条目。

**不做什么**：不修改旧 `initiative-closer.js`；不实现 okr_objectives / okr_key_results 层完成检测（超出本 PR scope）；不实现 KR 进度回算。

## 实现方案

**要改的文件**：
- `packages/brain/src/okr-closer.js`（新建）
- `packages/brain/src/tick.js`（添加 3 个新感知调用，模式同现有 initiative-closer 调用）
- `packages/brain/src/task-router.js`（添加 3 个新 task_type：okr_initiative_plan / okr_scope_plan / okr_project_plan，路由到 /decomp，location xian）
- `packages/brain/src/thalamus.js`（ACTION_WHITELIST 添加 3 个 okr_* 完成事件动作）
- `packages/brain/src/__tests__/okr-closer.test.js`（新建单元测试）

**Scope 锚定**：参照 `initiative-closer.js` 的 `checkInitiativeCompletion` / `checkScopeCompletion` / `checkProjectCompletion` 函数，对应实现 okr_* 表版本。关键逻辑：
- okr_initiatives 下 tasks：`WHERE okr_initiative_id = $1 AND status IN ('queued','in_progress') = 0` → 完成
- okr_scopes 下 initiatives：`WHERE scope_id = $1 AND status NOT IN ('completed','cancelled') = 0` → 完成
- okr_projects 下 scopes：`WHERE project_id = $1 AND status NOT IN ('completed','cancelled') = 0` → 完成
