# PRD: TICK 异常分级快速路由规则

## 功能描述

为 thalamus.js 的 quickRoute 函数完善 TICK 有异常时的分级快速路由规则。

当前有异常的 TICK 全部 fall through 到 Sonnet，但部分场景有明确处理模式，不需要 Sonnet 参与。

## 成功标准

1. TICK + anomaly_type=resource_pressure → [log_event, fallback_to_tick]，confidence 0.85
2. TICK + anomaly_type=stale_tasks → [log_event, reprioritize_task]，confidence 0.8
3. TICK + 其他 anomaly_type → return null（交 Sonnet 深度分析）
4. TICK + 无异常 → fallback_to_tick（原有行为保持不变）
5. ACTION_WHITELIST 包含 log_event, fallback_to_tick, reprioritize_task（均已存在）
6. facts-check.mjs 通过
7. vitest 新增对应测试覆盖
