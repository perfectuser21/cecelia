# Task Card: feat(capture) — Brain Digestion Job（PR2: AI 拆解逻辑）

## 需求（What & Why）

**功能描述**: Brain tick 中增加 capture-digestion job，自动扫描 inbox captures 并调 LLM 拆解为原子事件。

**背景**: capture_atoms 表和 API 已就位（PR #1601）。需要后台 job 自动完成 Capture → Atom 的拆解。

**不做什么**: Review UI、自动路由确认、飞书接入。

## 成功标准

1. capture-digestion.js 文件存在于 packages/brain/src/
2. tick.js 中包含 runCaptureDigestion 调用
3. runCaptureDigestion 函数导出且可调用
4. LLM prompt 包含 6 种 target_type 定义
5. 拆解结果写入 capture_atoms 表
6. capture status 更新为 done
