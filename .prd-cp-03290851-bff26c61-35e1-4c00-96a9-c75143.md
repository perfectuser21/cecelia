# PRD: Playwright Evaluator 层 — 端到端行为验证

## 背景
任务来自 Brain 自动调度（agent_ops 领域）。

## 功能描述
在 /dev Stage 3（CI 通过后）增加 Playwright Evaluator subagent，对照 DoD [BEHAVIOR] 条目逐条验证运行中的 Brain API 和 Dashboard。Evaluator 读取 Sprint Contract（Task Card），动态生成验证脚本，逐条执行，失败则返回详细反馈给主 agent 修复。

## 成功标准

- `packages/engine/lib/devloop-check.sh` 新增条件 3.5：CI 通过后检查 playwright_evaluator seal 文件（`.dev-gate-pw.<branch>`），FAIL → blocked，PASS/SKIP/无seal → pass-through
- `packages/engine/skills/dev/steps/03-integrate.md` 新增步骤 3.3，描述 Playwright Evaluator subagent 触发流程
- `packages/workflows/skills/playwright-evaluator/SKILL.md` 新增 Playwright Evaluator skill 定义

## 不做什么
- 不修改 spec_review / code_review_gate 的 seal 机制
- 不在 CI 流水线内运行 Playwright（仅本地执行）
- Brain API 不可用时 SKIP（pass-through）
