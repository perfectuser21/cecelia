# PRD: [SelfDrive] 优先激活高价值基础平台孤岛

## 背景
Scanner 扫描发现 `learning-absorption` 和 `memory-working` 被误判为孤岛。
实际上两者均在运行：learnings 表有 288 条记录，working_memory 表有数据。
根因是 capabilities 表中 related_skills 和 key_tables 字段为空。

## 目标
为 `learning-absorption` 和 `memory-working` 补充元数据，使 Scanner 能正确识别为 active。

- [ ] migration 161 正确更新 learning-absorption 的 related_skills 和 key_tables
- [ ] migration 161 正确更新 memory-working 的 related_skills 和 key_tables

- [ ] CI 通过
