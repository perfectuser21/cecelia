# PRD: codex_playwright 任务类型

## 背景

新增 `codex_playwright` task_type，让 Brain 能派发 Playwright 自动化任务到西安 M4 的 Codex agent。Codex 用大模型探索目标操作，写出能跑通的 Playwright `.cjs` 脚本，脚本通过 CDP 控制西安 PC（100.97.242.124:19225）。

## 需求

1. Brain `task-router.js` 注册 `codex_playwright`，路由到 `xian`，skill 为 `/playwright`
2. Brain `model-registry.js` 添加 `codex_playwright` agent，使用 OpenAI 模型（与 codex_qa 一致）
3. Brain `token-budget-planner.js` 添加 `codex_playwright` affinity（primary: codex，no_downgrade: true）
4. Brain `slot-allocator.js` 的西安 Codex 查询包含 `codex_playwright`
5. Brain `pre-flight-check.js` 将 `codex_playwright` 加入 SYSTEM_TASK_TYPES（无需 PRD）
6. 新建 `packages/engine/runners/codex/playwright-runner.sh`（Playwright 专用 runner）
7. 新建 `packages/workflows/skills/playwright-auto/SKILL.md`（Codex 工作流指令）

## 成功标准

- `codex_playwright` 在所有 Brain 路由配置文件中正确注册
- `playwright-runner.sh --dry-run` 能正常输出 prompt 不报错
- `SKILL.md` 包含 CDP 连接地址、两阶段工作流、脚本保存路径说明
