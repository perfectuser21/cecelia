## PRD - quickRoute 扩展: RESOURCE_LOW 事件快速路由

**背景**: 当前 quickRoute() 不处理 RESOURCE_LOW 事件，系统资源低告警会上升到 Sonnet，增加延迟。

**目标**: 在 thalamus.js quickRoute() 中为资源低事件添加快速路由。

**具体规则**:
- `resource_low` 且 severity=low（轻微不足）: 返回 log_event + no_action，confidence=0.9
- `resource_low` 且 severity=critical（严重不足）: 返回 log_event + notify_user，confidence=0.85
- `resource_low` 且没有 severity 字段: 返回 null（交给 Sonnet 判断）

**涉及文件**:
- brain/src/thalamus.js（quickRoute 函数新增 RESOURCE_LOW 分支）
- brain/src/__tests__/thalamus.test.js（新增 RESOURCE_LOW quickRoute 单元测试）

**验收标准**:
- [ ] quickRoute 处理 resource_low (severity=low) → log_event + no_action
- [ ] quickRoute 处理 resource_low (severity=critical) → log_event + notify_user
- [ ] quickRoute 对 resource_low 且无 severity 返回 null
- [ ] EVENT_TYPES.RESOURCE_LOW 值正确为 resource_low
- [ ] 单元测试覆盖以上 3 种场景（含 2 actions 链的验证）
- [ ] npm test 通过
- [ ] facts-check.mjs 通过
