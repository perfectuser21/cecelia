# Task Card: Strategy Tree — OKR全链路可视化 + 进度API

## 需求（What & Why）

**功能描述**: 构建 OKR→KR→Project→Scope→Initiative→Task 完整链路可视化页面，让用户在 dashboard 一眼看清项目全局状态、进度和问题。

**背景**:
- DB 里已有完整链路数据（objectives/key_results/okr_projects/okr_scopes/okr_initiatives/tasks），但没有串联 API
- 用户每次对话都要重新解释背景，缺少持久的项目全景视图
- 任务完成后进度不会自动冒泡到上层 KR/OKR，全靠手动维护

**不做什么**:
- 不做编辑/修改功能（本 PR 只读）
- 不做问题冒泡告警（下次迭代）
- 不改 DB schema

## 实现方案

**Brain 新端点** `packages/brain/src/routes/strategy-tree.js`
- 单次 GET 请求返回所有 Area 的完整树（含 non-archived areas only）
- 按层级用 JOIN + GROUP BY 批量查询，避免 N+1
- Initiative 层计算：`SELECT count(*) FILTER (WHERE status='completed') as completed, count(*) as total FROM tasks WHERE okr_initiative_id=...`
- 可选参数：`?area_id=xxx` 过滤单个 Area

**前端** `apps/api/features/knowledge/pages/StrategyTree.tsx`
- 默认展开到 KR 层（Objective 以下折叠）
- 每节点：名称 + 状态色 badge + 进度条（completed/total）
- Initiative 展开后显示 task 列表（title + status + pr_title）
- 使用 `useApi('/api/brain/strategy-tree')` 获取数据
- lucide-react 图标：ChevronRight/Down、Target、Layers、GitBranch 等
