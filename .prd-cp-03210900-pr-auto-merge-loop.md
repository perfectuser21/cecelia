# PRD: 实现 PR 自动合并闭环

## 问题
stop-dev.sh 第 828-830 行检测到 CI 通过且 Stage 4 完成时，只输出 JSON 建议文本，没有真正执行 gh pr merge。
cecelia-run.sh 收到 exit 2 后只是 sleep 2s 重试，形成无限循环。

## 方案
在 stop-dev.sh 和 devloop-check.sh 中，当 CI 全部通过 + Stage 4 Ship 已完成 + PR 未合并时，真正执行 gh pr merge。
合并失败时 exit 1（非 exit 2），避免无限循环。

## 安全要求
- 只在 CI 全部通过 + Stage 4 完成 + Learning 验证通过 的情况下才自动合并
- 合并失败时 exit 1
- 加日志输出方便排查
