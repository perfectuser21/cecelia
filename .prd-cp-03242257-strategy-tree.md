# Strategy Tree — OKR全链路可视化 + 进度API

## 背景

OKR→KR→Project→Scope→Initiative→Task 完整链路数据已在 DB（7张表），但没有串联 API 和前端全景视图。用户每次对话都要重新解释背景，缺乏持久的项目全景视图。

## 功能描述

1. Brain 新端点 `GET /api/brain/strategy-tree`，返回完整 OKR 层级树：Area → Objective → KR → Project → Scope → Initiative → Tasks。每层带 total_tasks / completed_tasks / progress（子层自动 rollup）。
2. 前端页面 `/knowledge/strategy-tree`（StrategyTree.tsx），可折叠树形结构，默认展开到 KR 层，每节点显示进度条+状态色（绿/黄/红），Initiative 节点展开显示 Task 列表（pr_title + learning_summary + status），Task 节点可展开显示 PRD 摘要 + PR 链接。
3. `apps/api/features/knowledge/index.ts` 新增 strategy-tree 路由和导航入口。

## 成功标准

1. `GET /api/brain/strategy-tree` 返回完整树结构，每节点含 completed_tasks/total_tasks/progress
2. `StrategyTree.tsx` 页面存在且能渲染层级树（Area→Initiative 折叠，默认展开到 KR）
3. Initiative 节点展开后显示 Task 列表
4. `knowledge/index.ts` 注册新路由和导航入口（label: 'Strategy Tree'）
5. Brain server.js 注册 strategy-tree 路由
