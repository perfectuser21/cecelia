# PRD: Engine 状态机三层防御 P0+P1

## 功能描述
堵住状态机双层绕行路：
1. bash-guard.sh 补 .dev-mode Bash 写检测 → 调用 verify-step.sh
2. verify-step.sh 通过后写 seal（step_N_seal: verified@timestamp）
3. stop.sh 补 seal 检查：step_N done 但无 seal → exit 2 强制补验

目标：杜绝 AI 通过 Bash 工具绕过 PreToolUse:Write 拦截，并在 Stop Hook 层兜底。

- [ ] devloop-check.sh：step_N done 但无 seal → blocked 强制补验
