# PRD: Cecelia "真正管家" 对话系统

## 背景
用户说人话，Cecelia 理解意图 + 拉取记忆 + 关联 OKR → 翻译成精确机器指令 → 执行 → 反馈。

## 5 个致命问题
1. 对话记忆断裂 — memory-retriever 不搜 orchestrator_chat 事件
2. 意图识别 100% 失败 — intent.js 纯正则，无 LLM 回退
3. 实体链接不存在 — 用户模糊表达无法关联 OKR/Task
4. 执行无反馈 — LLM 回复在 action 执行前生成
5. 用户画像质量差 — prompt bug 把 Cecelia 名字当用户的

## 交付范围
- PR 1: 对话记忆持久化（修复记忆断裂）
- PR 2: LLM 意图理解 + 翻译层
- PR 3: OKR/Task 实体链接
- PR 4: 用户画像修复
- PR 5: 系统 Prompt 增强

## 成功标准
- 用户说"上次聊的那个"，Cecelia 能从最近 7 天对话中找到上下文
- 模糊输入不再返回 UNKNOWN（走 LLM 回退）
- 创建的任务自动关联到相关 OKR/Project
- 用户画像只包含用户事实，不包含 Cecelia 的信息
