### 技术方案（Kuaishou Open Platform API）

```
认证流程：
  user → OAuth authorize URL（一次性手动）
      → 获得 authorization_code
      → POST /oauth2/access_token → access_token + refresh_token
      → 存储到 ~/.credentials/kuaishou.env

发布流程：
  read ~/.credentials/kuaishou.env
      → check token expiry
      → if expired: POST /oauth2/refresh_token → new access_token
      → POST /openapi/photo/image/upload（上传图片）
      → POST /openapi/photo/publish（创建图文）
```

**关键端点（Kuaishou Open Platform）**：
- 授权: `https://open.kuaishou.com/oauth2/authorize`
- Token: `https://open.kuaishou.com/oauth2/access_token`
- 刷新: `https://open.kuaishou.com/oauth2/refresh_token`
- 图片上传: `https://open.kuaishou.com/openapi/photo/image/upload`
- 图文发布: `https://open.kuaishou.com/openapi/photo/publish`

### 凭据要求

```bash
# ~/.credentials/kuaishou.env
KUAISHOU_APP_KEY=xxx
KUAISHOU_APP_SECRET=xxx
KUAISHOU_ACCESS_TOKEN=xxx
KUAISHOU_REFRESH_TOKEN=xxx
KUAISHOU_TOKEN_EXPIRES_AT=1234567890
KUAISHOU_OPEN_ID=xxx
```

### 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| 凭据文件缺失 | 打印引导步骤，exit 1 | P2 | kuaishou_missing_credentials |
| access_token 过期 | 自动用 refresh_token 刷新 | P3 | kuaishou_token_refreshed |
| refresh_token 过期 | 提示重新授权，exit 2 | P1 | kuaishou_reauth_required |
| 图片上传失败 | 打印错误码，exit 1 | P1 | kuaishou_image_upload_failed |
| 发布 API 失败 | 打印错误码，exit 1 | P1 | kuaishou_publish_failed |
| 无图片文件 | 打印说明，exit 1 | P3 | kuaishou_no_images |
