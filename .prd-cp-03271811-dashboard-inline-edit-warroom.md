# PRD: Dashboard 双升级 — 前台可编辑 + 作战室状态页

## 背景
前台目前大部分内容只能查看，无法直接编辑。缺少一个"作战室"页面让用户一眼看到当前系统在往哪个方向跑。

## 成功标准

### D1 GTDOkr title inline 编辑
- Vision/Objective/KR title 可双击进入编辑，blur/Enter 保存
- 调用 PATCH API 保存成功后 UI 更新

### D2 GTDProjects inline 编辑
- Project title 可双击编辑，blur/Enter 保存
- Project status 可点击切换

### D3 WarRoom 页面
- `/gtd/warroom` 路由可访问
- 展示当前 Vision 和活跃 Objectives
- 展示进行中 Tasks
