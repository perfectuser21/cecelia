# PRD: 内容飞轮 I3 — 发布后数据回收

## 背景

content_publish 任务完成 4 小时后，自动触发对应平台的数据 scraper，将阅读/点赞/播放等指标写入 pipeline_publish_stats 表，通过 API 接口对外暴露，闭合内容飞轮效果反馈环节。

## 功能范围

- 创建 pipeline_publish_stats 表（migration）
- 实现 post-publish-data-collector.js（4h 后触发 scraper 任务）
- 在 routes.js 注册 GET /api/brain/pipelines/:id/stats 路由

## 不做

- 不实现 scraper 脚本本身（已有）
- 不做实时数据同步
- 不做各平台账号登录管理
- 不做数据可视化

## 成功标准

- pipeline_publish_stats migration 文件存在
- post-publish-data-collector.js 存在且导出 schedulePostPublishCollection
- GET /api/brain/pipelines/:id/stats 路由可访问
- Brain tick 循环不受影响
- CI 全部通过
