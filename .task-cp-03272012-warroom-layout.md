# Task Card: War Room 横向多列 Dashboard 布局

## 需求（What & Why）
**功能描述**: 将 GTDWarRoom.tsx 从竖向单列滚动改为横向多列 dashboard 布局。顶部固定 Vision 横幅，下方左中右三列：左列 Objectives+KR、中列 Tasks、右列备用。各列独立 overflow-y-auto 滚动。
**背景**: 当前布局 `max-w-4xl space-y-6` 在宽屏浪费空间，且容器高度约束导致整体上下滚动失效（用户反馈"上下刷不动"）。
**不做什么**: 不改数据逻辑、不改 API 调用、不改路由注册。

## 成功标准

1. [ARTIFACT] GTDWarRoom.tsx 使用 `h-full flex-col` 撑满容器高度，Vision 在顶部
2. [BEHAVIOR] 下方内容区为 `grid-cols-3` 三列，各列有 `overflow-y-auto` 独立滚动
3. [GATE] CI 全部通过，无 TypeScript 编译错误
