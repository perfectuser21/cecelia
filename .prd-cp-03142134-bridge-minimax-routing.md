# PRD: cecelia-run bridge 自动启动修复

## 背景

Brain 启动后，cecelia-bridge（port 3457）不会自动启动，导致 Brain 无法派发 Claude Code 任务。
每次需要手动 `node packages/brain/scripts/cecelia-bridge.js` 才能让任务执行。

## 问题

1. `server.js` 没有 bridge 自动启动逻辑
2. `checkCeceliaRunAvailable()` 使用 `GET /` 检测（返回 404 视为运行），语义不清晰
3. bridge 已有 `/health` 端点但检测代码注释说"没有"，导致代码与注释不一致

## 成功标准

- Brain 启动后 cecelia-bridge 自动随之启动
- 若 bridge 已运行，不重复启动（幂等）
- `checkCeceliaRunAvailable()` 使用 `/health` 端点，返回 200 表示可用
- bridge 进程日志写入 `/tmp/cecelia-bridge.log`

## 范围

- `packages/brain/server.js`：添加 bridge auto-start
- `packages/brain/src/executor.js`：更新 `checkCeceliaRunAvailable()` 使用 `/health`
