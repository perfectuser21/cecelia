# PRD: fix GTDWarRoom 绝对 URL 导致暂无数据

## 背景
GTDWarRoom.tsx 第53行硬编码了 `http://localhost:5221/api/brain/tasks`。
浏览器通过 `http://perfect21:5211` 访问时，`localhost:5221` 解析为客户端本机，
请求失败导致 Promise.all 整体 reject，catch 块把 tree 也清空，页面显示"暂无活跃数据"。

## 修复内容
1. 将绝对 URL 改为相对路径 `/api/brain/tasks`（走 vite proxy → Brain）
2. 用 `Promise.allSettled` 替代 `Promise.all`，让两个请求独立容错

## 成功标准

- [ ] `GTDWarRoom.tsx` 不再包含 `http://localhost:5221`
- [ ] 使用 `Promise.allSettled` 独立容错
