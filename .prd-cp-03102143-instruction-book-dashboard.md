# PRD: Instruction Book Dashboard 页面

## 背景

`docs/instruction-book/` 目录下已有用户层说明书体系（skills/ + features/ 两个分类）。
用户希望在 Dashboard 前台能直接浏览这些文档，而不是通过命令行查看 markdown 文件。

## 目标

在 Dashboard `/knowledge/instruction-book` 路径下新增一个页面，展示 instruction-book 内容。

## 功能需求

1. **Brain API**：新增 `GET /api/brain/docs/instruction-book` 端点，扫描 `docs/instruction-book/` 目录，返回所有 entry 的结构化 JSON（标题、分类、各章节内容）
2. **前端页面**：`InstructionBook.tsx` 页面，左侧分类列表（Skills / Features），右侧展示选中 entry 的内容（What / Trigger / How to use / Output 四个章节）
3. **Knowledge 首页**：在 KnowledgeHome 加入 Instruction Book 入口卡片
4. **路由注册**：在 knowledge feature manifest 中注册 `/knowledge/instruction-book` 路由

## 非目标

- 不做 markdown 编辑功能
- 不做搜索功能
- 不做权限控制

## 成功标准

用户打开 Dashboard → Knowledge → Instruction Book，能看到所有 skills 和 features 条目，点击可查看详细内容。
