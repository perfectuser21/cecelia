# PRD: 全量注册 /dev pipeline + Brain 核心能力到 RCI

## 背景

通过全量扫描，发现现有 regression-contract.yaml 的 160 条 RCI 严重偏科：
Engine/Hook 层覆盖过度，但 /dev pipeline 关键行为和 Brain 认知层几乎裸奔。
最核心的 20 个能力点没有回归保护——任何重构都可能悄悄破坏这些机制而 CI 不报警。

## 目标

在 `packages/engine/regression-contract.yaml` 新增 20 条 RCI，覆盖：
- /dev 4-Stage Pipeline 的关键门控行为（P0×4）
- Brain 核心模块行为（P0×3 + P1×7）
- 长期健康必要行为（P2×6）

每条 RCI 必须有对应的自动测试（auto method），部分可用 manual+evidence。

## 范围

**文件**：`packages/engine/regression-contract.yaml`
**测试目录**：`packages/engine/tests/`（新增测试文件）

## P0（7条）— 坏了系统核心立即失效

1. **DEV-S2-OOM-001** verify_step2 OOM-aware：Brain test script 能识别 worker exit 但 all passed
2. **DEV-S2-SEAL-001** .dev-seal 格式：sha/branch/timestamp 三字段必须存在
3. **DEV-S3-REQPATHS-001** Required Dev Paths：高风险路径必须有 PRD 证据
4. **DEV-S4-LEARNING-001** Learning 前置：合并前 Learning 文件必须存在且包含根本原因章节
5. **BRAIN-EXEC-001** Executor 切换：B-class 任务支持 Claude Code / Codex 两种 executor
6. **BRAIN-TICK-REENTRY-001** Tick 重入保护：_tickRunning 防止并发执行
7. **BRAIN-THALAMUS-001** Thalamus 任务创建权：createTask 唯一入口

## P1（7条）— 坏了某功能失效

8. **BRAIN-PATROL-001** Pipeline Patrol 5 分钟周期
9. **BRAIN-SELFDRV-DEDUP-001** Self-Drive 去重+相似度检查
10. **BRAIN-SELFDRV-MAX-001** Self-Drive 最大任务数限制（DEFAULT_MAX_TASKS=3）
11. **BRAIN-EXEC-LB-001** Executor 负载均衡（最低负载 Bridge）
12. **DEV-WORKTREE-001** Worktree 多会话隔离
13. **BRAIN-ROUTER-FALLBACK-001** Task Router 降级链
14. **BRAIN-EVOSCAN-001** Evolution Scanner 写 working_memory

## P2（6条）— 长期健康

15. **BRAIN-DESIRE-001** Desire 系统推荐任务
16. **BRAIN-RUMINATION-001** Rumination 反思闭环
17. **BRAIN-SELFDRV-INTERVAL-001** Self-Drive 4小时周期
18. **BRAIN-ZOMBIE-001** Zombie Sweep 30分钟清理
19. **BRAIN-DISPATCH-TIMEOUT-001** Dispatch 超时60分钟
20. **UI-STATE-001** Dashboard-Brain 状态同步

## 成功标准

- `packages/engine/regression-contract.yaml` 新增 20 条 RCI
- 每条 P0/P1 RCI 有 test 字段指向真实测试文件或 manual: 命令
- `node packages/engine/scripts/devgate/scan-rci-coverage.cjs` 覆盖率不下降
- CI Engine L3 通过
