# PRD: Cecelia 自我感知层

## 背景
Cecelia 丘脑每次 LLM 调用只有 800 token 历史记忆，不知道：
- 自己有哪些 Skills（63个）
- 当前任务队列状态（多少排队/执行中/失败）
- 组件健康状态
- 派给 Claude Code 的 prompt 没有系统背景

结果：Cecelia 的 LLM 和 Claude Code 对话一样"盲"。

## 目标
给 Cecelia 构建自我感知层，让丘脑 LLM 和派出的 Claude Code 都知道"系统现在是什么状态、自己能做什么"。

## 方案

### 1. thalamus.js：注入自我感知上下文
- 新增 `buildSelfAwarenessContext(pool)` 函数，5分钟缓存
- 查询：当前任务队列统计、最近失败任务
- 注入：Skills 能力地图摘要（核心 task_type → skill 路由）
- 在 `analyzeEvent()` 中将自我感知上下文加到 prompt 里

### 2. executor.js：dispatch 时带系统上下文
- `preparePrompt()` 在所有类型任务 prompt 头部追加系统背景块
- 内容：Cecelia 是什么、Brain API 地址、任务完成后如何回调

## 成功标准
- 丘脑 LLM 每次调用都包含当前队列状态和 Skills 摘要
- Claude Code 收到的 prompt 包含系统背景信息
- 缓存 TTL 5分钟，不影响性能
- 所有现有测试继续通过
