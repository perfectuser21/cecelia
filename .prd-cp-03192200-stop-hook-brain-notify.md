# PRD: Stop Hook 超时时创建 Brain 诊断任务

## 背景
`packages/engine/hooks/stop-dev.sh` 第 155 行 MAX_RETRIES=30，超过后 exit 0 放行但无人跟进修复。任务卡住后默默失败，没有后续诊断。

## 目标
超过 30 次重试时，调用 Brain API 创建 P0 诊断任务（task_type: stuck_diagnosis），然后 exit 0 放行当前会话。Brain 收到后派发新会话诊断根因。

## 成功标准

- stop-dev.sh 超时代码块中在 exit 0 前调用 Brain POST API 创建 stuck_diagnosis 任务
- curl 使用 --max-time 5 非阻塞
- 保留现有 track.sh fail 和 Brain PATCH 逻辑
- hooks/stop-dev.sh 根目录副本同步更新
