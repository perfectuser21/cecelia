## PRD - fix(brain): 修复集成测试失败，降低 brain-integration-baseline

**需求来源**: Brain 集成测试有 151 个失败（main 只有 81），需要找出根因并修复
**功能描述**: 系统地分析所有失败的集成测试，修复测试代码 bug（错误 mock 路径、缺失 export 等）和源代码 bug，目标把失败数降回到接近 main 的水平（≤100），并相应更新 brain-integration-baseline.txt

**涉及文件**:
  - `packages/brain/src/__tests__/` 下所有集成测试
  - `packages/brain/src/` 相关源文件
  - `packages/brain/ci/brain-integration-baseline.txt`

**非目标**: 不实现全新功能（如 callback revision_round 等复杂逻辑），只修已知 bug

## 成功标准

- 集成测试失败数 ≤ 125（不超过当前 baseline）
- L4 CI 通过
- 至少修复 tick-dispatch.integration.test.js 的 mock 路径错误
- 更新 brain-integration-baseline.txt 为合理值
