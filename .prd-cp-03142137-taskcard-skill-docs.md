# PRD: /dev Skill 文档重组为6步结构（Task Card 格式）

## 功能描述
把 /dev 工作流文档从12步重组为6步外壳，并引入 Task Card 格式
（把 .prd-cp-xxx.md + .dod-cp-xxx.md 合并成单一 .task-cp-xxx.md）。

内部逻辑一个不少，只是外层步骤编号和文件从12个合并成6个。

## 成功标准
- SKILL.md 流程图更新为6步，步骤映射表正确
- 新建 steps/01-taskcard.md（替代原01-prd + 05-dod前半）
- steps/03-branch.md 的 .dev-mode 格式更新为 task_card 字段

本次重组将外层步骤压缩为6步，并引入 Task Card 格式统一需求和验收文档。
