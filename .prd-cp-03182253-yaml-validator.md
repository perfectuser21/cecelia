# PRD: YAML 验证器（启动时检查格式）（PR4）

## 背景

PR1/PR3 已建立内容类型 YAML 注册表和 Pipeline 集成。需要新增轻量验证器，
在 Brain 启动时（selfcheck 阶段）检查所有 YAML 配置格式是否正确，
有问题时打印 WARN 日志（不阻断启动）。

## 成功标准

1. 新增 content-type-validator.js，export validateContentType + validateAllContentTypes
2. validateContentType 对缺少必填字段的配置返回 errors 数组
3. validateAllContentTypes 汇总所有 YAML 校验结果
4. selfcheck.js 调用并打印 WARN（不阻断启动）
5. CI 全部通过
