# PRD: feat(dashboard): 实现 /projects/compare 项目并排对比页面

## 背景
后端 API POST /api/brain/projects/compare/report 已在 PR #783 合并。需实现对应前端页面。

## 功能描述

1. 新建 `apps/api/features/planning/pages/ProjectCompare.tsx`
   - 多选下拉选 2-4 个项目
   - 并排对比卡片（评分、进度条、任务统计、优劣势）

2. 在 `apps/api/features/planning/index.ts` 注册路由 `/projects/compare` 和组件 `ProjectCompare`

3. `ProjectsDashboard.tsx` 添加「对比项目」入口按钮

## API

- `GET /api/tasks/projects` — 拉项目列表
- `POST /api/brain/projects/compare/report` — 拿对比数据

## 成功标准
- `/projects/compare` 路由可访问
- 选择 2+ 项目后展示并排对比
- 响应式布局，深色主题一致
