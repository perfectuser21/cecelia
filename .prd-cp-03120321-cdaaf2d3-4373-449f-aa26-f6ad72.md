## PRD - KR 进度诚实化修复 — 基于 initiative 完成率重写 goals.progress

**需求来源**: Brain 自动调度任务
**功能描述**: 
KR审计结果(2026-03-12): self-model KR progress=100但initiative完成率5/18=28%; 
免疫系统KR progress=100但initiative完成率8/16=50%; 
组织架构KR progress=100但project完成率0/1=0%.

实现三个功能：
1. GET /api/brain/goals/audit 端点返回 stated_progress vs actual_progress 对比
2. 一次性 migration 将虚标 KR progress 更正为实际值（基于 initiative 完成率）
3. 写入 memory_stream 记录修正事件

**涉及文件**:
- packages/brain/src/routes/goals.js（新增 /audit 端点）
- packages/brain/migrations/（新增 migration SQL）
- packages/brain/src/selfcheck.js（更新 EXPECTED_SCHEMA_VERSION）

**成功标准**:
## 成功标准
- GET /api/brain/goals/audit 端点可用，返回 stated_progress vs actual_progress
- 4个虚标 KR（progress=100 但 initiative 完成率低）不再全是 100
- memory_stream 中有修正事件记录可追溯

**非目标**:
- 不做自动持续同步（只做一次性修正 + audit 端点）
- 不修改 UI
