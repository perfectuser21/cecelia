# PRD: 修复嘴巴层 MiniMax 调用

## 需求来源
orchestrator-chat.js 的 callMiniMax 函数实际调用 Anthropic API，但容器没有 ANTHROPIC_API_KEY，导致 fallback 到 thalamus，把内部分析文本（rationale + 建议动作）作为聊天回复显示给用户，体验极差。

## 功能描述
将 callMiniMax 改为真正调用 MiniMax API（与 thalamus.js 相同方式）：
- 读取 ~/.credentials/minimax.json 获取 api_key
- 调用 https://api.minimaxi.com/v1/chat/completions
- 使用 MiniMax-M2.5-highspeed 模型
- 实现 stripThinking 去除 <think> 块

## 成功标准
- 发送 "你好" 后收到自然对话回复，不含 thalamus 分析文本
- 测试通过
