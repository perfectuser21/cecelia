# PRD: Dashboard 大脑模型切换面板

## 背景

Brain 的各个 organ（丘脑/皮层/口/记忆等）各自调用不同的 LLM 模型。
当前模型配置只能改代码，没有前端界面。
用户需要在 Dashboard 直接切换模型 profile 或单独调整某个 organ 的模型。

## 需求

### 核心功能

1. **Profile 切换**：显示 MiniMax 主力 / Anthropic 主力两个 profile，点击一键切换激活
2. **Organ 状态展示**：列出所有 organ（丘脑/皮层/口/记忆/叙述/反思/沉思），显示当前各自用哪个模型
3. **单 Organ 调整**：点击某个 organ，可以选择该 organ 使用的模型（从 allowed_models 列表选）

### API 对接

- `GET /api/brain/model-profiles` — 列出所有 profile
- `GET /api/brain/model-profiles/active` — 获取当前激活 profile
- `PUT /api/brain/model-profiles/active` — 切换 profile（body: `{profileId}`）
- `PATCH /api/brain/model-profiles/active/agent` — 修改单个 organ（body: `{agentId, model, provider}`）
- `GET /api/brain/model-profiles/models` — 获取可用模型列表

### 路由

新页面路由：`/brain-models`
页面组件：`apps/dashboard/src/pages/brain-models/BrainModelsPage.tsx`

## 成功标准

- 能看到当前激活 profile 名称
- 能看到 thalamus/cortex/mouth/memory 各自的当前模型
- 点击 profile 卡片能切换激活 profile
- 页面样式与 AccountUsagePage 风格一致（深色主题）
