# PRD: B类任务支持 executor 切换

**Task ID**: 9728b7cb-e230-4fd0-b26a-6a1e2ec33d14
**Branch**: cp-03222206-b-class-executor-switch

## 背景

B类任务锁定在美国M4机器，但执行模型可选：
- Claude Code（默认，cecelia-bridge port 3457）
- Codex CLI（triggerLocalCodexExec，本机 2-slot 池）

## 目标

前台 TaskTypeConfigPage 允许为 B类任务切换 executor，
后台 executor.js 根据 task_type_configs.executor 字段路由。

## 成功标准

- B类任务显示执行器选择 UI
- executor=codex 时路由到 triggerLocalCodexExec
- executor=claude_code 时路由到 cecelia-bridge（默认行为不变）
