# PRD: Engine CI 守护规则强化

## 背景
Brain 调度任务。清理 RCI 垃圾条目后，需要守护规则防止未来再次引入无效条目。

## 功能描述
1. check-rci-health.mjs 新增孤儿测试检测（第4项检查）
2. 新建 check-coverage-completeness.mjs（源文件必须有测试）
3. rci-execution-gate.sh 新增 --engine-rci 参数，可扫描 engine RCI
4. ci-l1-process.yml 接入 check-rci-health.mjs

## 成功标准
- check-coverage-completeness.mjs 可运行
- check-rci-health.mjs 支持 --check orphan-tests
- ci-l1-process.yml 包含 check-rci-health.mjs 调用
- rci-execution-gate.sh 支持 --engine-rci 参数
