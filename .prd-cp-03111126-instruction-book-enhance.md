# PRD: Instruction Book 增强 — Changelog + 子章节下钻

## 背景

当前 Instruction Book（`/knowledge/instruction-book`）展示了 Skills 和 Features 的说明，
但存在以下不足：
1. Changelog 只在 MD frontmatter 中，未在 UI 展示
2. 内容扁平，无法下钻到子章节（`###` 级别）
3. 用户无法看到功能的迭代历史

## 目标

让 Instruction Book 成为一个功能完整的说明书，支持：
1. **Changelog 显示**：每个 skill/feature 卡片下有版本变更历史
2. **子章节导航**：右侧详情页支持 `###` 级别小节，可跳转/展开

## 成功标准

- Brain API `/api/brain/docs/instruction-book` 返回 `changelog` 数组字段
- Brain API 返回 `subsections`（`###` 子章节）嵌套结构
- 前端 `InstructionBook.tsx` 展示 changelog timeline
- 前端支持子章节内容块展开显示
- 现有 3 个 skill MD 文件的 changelog 在 UI 可见

## 不做的事

- 不重写整个 Instruction Book 布局
- 不改变 MD 文件的基本格式（只利用现有 frontmatter changelog）
- 不做搜索功能

## 技术范围

- `packages/brain/src/routes.js`（或独立 docs 路由）— 解析 changelog + 子章节
- `apps/api/features/knowledge/pages/InstructionBook.tsx` — 展示 changelog + 子章节
