# PRD: ci-l1-process.yml 支持 Task Card 格式

## 需求来源
dev 工作流 v4.0 引入统一 Task Card（`.task-cp-{branch}.md`），但 CI 仍只认旧格式（`.prd-*.md` + `.dod-*.md`），导致使用新格式的 PR 无法通过 CI。

## 功能描述
修改 `.github/workflows/ci-l1-process.yml`，让 `cleanup-check` 和 `required-paths-check` 两个 job 能识别新的 `.task-{branch}.md` 格式，同时向后兼容旧格式。

## 涉及文件
- `.github/workflows/ci-l1-process.yml`（cleanup-check job 和 required-paths-check job）

## 成功标准
- cleanup-check job 优先查找 `.task-${GITHUB_HEAD_REF}.md`，找不到再找旧格式
- required-paths-check job 优先查找 `.task-${HEAD_REF}.md`，找不到再找旧格式
- 旧格式 `.prd-*.md` + `.dod-*.md` 继续有效
- CI 全部通过

## 非目标
- 不修改 check-prd.sh、check-dod-mapping.cjs 等其他脚本
- 不删除旧格式支持
- 不修改 branch-protect.sh 本地 hook

## 异常处理

N/A - 纯 CI 配置修改，无 IO 操作
