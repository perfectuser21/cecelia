# Task Card: DeepSeek V3 自动 PR Review CI

## 需求（What & Why）
**功能描述**: 在 Cecelia 仓库每个 PR 创建或更新时，自动触发 DeepSeek V3 对代码变更进行 AI review 并以中文评论形式发布到 PR 上。
**背景**: 人工 code review 耗时，引入 AI review 可以在人工审查前自动发现常见问题、提升 PR 质量，降低漏检风险。
**不做什么**: 不修改现有 CI 流程（L1/L2/L3/L4）；不集成到 Brain 决策引擎；不做 review 结果的结构化存储；不阻塞 PR 合并（仅评论，不设置 required check）。

## 成功标准
> [ARTIFACT] = 产出物（文件/API/配置）
> [BEHAVIOR] = 运行时行为（系统如何响应输入）
> [GATE] = 门禁（CI/测试/代码质量）

1. [ARTIFACT] `.github/workflows/pr-review.yml` 存在，使用 `hustcer/deepseek-review@v1`，触发条件为 `pull_request` 的 `opened`/`synchronize` 事件
2. [BEHAVIOR] 向该仓库提交 PR 后，GitHub Actions 自动运行 pr-review workflow，DeepSeek 以中文在 PR 上发表 review 评论
3. [GATE] 新增的 pr-review.yml 格式合法（GitHub Actions YAML 无语法错误），且不破坏现有 CI 通过状态
