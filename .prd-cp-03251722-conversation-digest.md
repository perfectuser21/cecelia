# PRD: conversation-digest — Brain 自动读取 Claude Code 对话日志

## 背景

用户与 Claude Code 的规划对话（before-coding 的讨论：decisions、ideas、open questions）目前完全没有被捕获。
对话日志以 `.jsonl` 格式机械写入磁盘，Brain 在同一台机器可直接读取。

不依赖 LLM 记忆或 prompt 注入，而是把机械写入的 artifact 交给 Brain 异步分析。

## 成功标准

1. conversation_log_cursors 表存在（tracking 处理进度）
2. conversation-digest.js 模块存在（scan + analyze + persist）
3. tick.js 集成 digest（每 5 分钟）
4. selfcheck.js EXPECTED_SCHEMA_VERSION = '196'
