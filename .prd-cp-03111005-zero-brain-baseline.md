# PRD: Brain 测试 Baseline 归零

## 背景

PR #818 修复了 isolate:false 导致的 74 个测试失败。CI L4 现在实际失败数为 4（`cpu-sampler.test.js` 3个 + `minimax-provider.test.js` 1个），但 baseline 仍为 125，形成虚假容忍。

## 目标

1. 修复剩余 4 个测试失败（根因：platform-utils mock 缺 `getAvailableMemoryMB`）
2. 将 `brain-integration-baseline.txt` 从 125 降至 0

## 根因

PR #812 新增了 `getAvailableMemoryMB()` 到 `platform-utils.js`，`executor.js` 开始使用它，但两个测试文件的 mock 没有同步更新，导致 vitest 报错 "No getAvailableMemoryMB export is defined on mock"。

## 成功标准

- Brain integration CI（L4）失败数 = 0
- `brain-integration-baseline.txt` = 0
- `cpu-sampler.test.js` 和 `minimax-provider.test.js` 全部通过
