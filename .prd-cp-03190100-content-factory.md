# PRD: Dashboard 内容工厂触发页

## 背景

Brain 已提供 content-pipeline API（PR #1101），Dashboard 需要一个触发页面让用户能启动和查看 Pipeline。

## 需求

在 Dashboard (apps/dashboard → apps/api/features) 新增「内容工厂」页面。

## 成功标准

1. [ARTIFACT] 新增 ContentFactory 页面组件，路由 /content-factory
2. [BEHAVIOR] 页面能从 Brain API 获取内容类型列表并展示下拉选项
3. [BEHAVIOR] 表单提交调用 POST /api/brain/pipelines 创建 content-pipeline 任务
4. [BEHAVIOR] 页面下方展示 Pipeline 列表（title/status/created_at）
5. [GATE] CI 全部通过

## 不做什么

不做图片预览，不做 Human Approve 交互，不做发布按钮。
