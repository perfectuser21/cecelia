# PRD: Fix Engine macOS Compatibility (fix-engine-macos-compat)

## 背景

Cecelia Engine (`packages/engine`) 是整个自主系统的元层（meta-layer）。
本地 macOS 开发环境（`npm test`）有 37 个测试失败，而 CI（Linux）是绿色的。
这意味着 Engine 在 Mac 本地开发时存在 bug，无法被早期发现。

## 成功标准

- `npm test` 在 macOS 上零失败（0 failed）
- branch-protect.sh 修复 bash 3.2 多字节字符变量名解析 bug
- runner.sh 修复同类 bash 3.2 兼容性问题
- 测试中 `git checkout master` → 动态检测 main/master
- 测试中 `git checkout -b main` → 兼容已存在的 main 分支
- 不破坏现有 CI（Linux green stays green）
