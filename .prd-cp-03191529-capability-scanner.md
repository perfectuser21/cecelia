# Task Card: 能力孤岛扫描器

## 需求（What & Why）

**功能描述**: 持续扫描 Cecelia 所有已注册能力和实际使用数据，找出孤岛（从未调用或连续失败的能力），生成能力健康地图。

**背景**:
- Cecelia 有 37 个 capability、多种 task_type，但不知道哪些能力是孤岛
- 之前做过一次性孤岛分析（2月），但没有持续探测机制
- 与 Initiative 1 的 Capability Probe 联动——Probe 探测链路通不通，Scanner 发现哪些能力是孤岛

**不做什么**:
- 不改变 capability 表结构
- 不改变任务派发逻辑

## 成功标准

1. [ARTIFACT] 新增 capability-scanner.js，扫描 capabilities 表 vs tasks 实际使用
2. [BEHAVIOR] 能识别出从未被使用的 capability（孤岛）和长期失败的 capability
3. [BEHAVIOR] 扫描结果写入 cecelia_events + API 端点暴露
4. [GATE] 所有现有测试通过
