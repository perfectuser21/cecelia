# PRD: PR2 — Brain 代码全量切换新 OKR 表

## 背景

migration 179 已将旧表数据迁移到新 OKR 层级表。Brain 核心代码（调度/决策/分析模块）仍在读写旧的 `goals`/`projects` 表。本 PR 将非路由层的 Brain 代码切换到新表。

## 目标

将 packages/brain/src/ 下非路由文件中对 `goals`/`projects` 表的 SQL 引用切换到新 OKR 层级表。

## 列名映射

| 旧列 | 新列 | 适用表 |
|------|------|--------|
| goals.target_date | .end_date | visions/objectives/key_results |
| projects.name | .title | okr_projects/okr_scopes/okr_initiatives |
| projects.deadline | .end_date | okr_projects/okr_scopes/okr_initiatives |
| projects.goal_id | .kr_id (NULL) | okr_projects |
| goals.parent_id (area_okr) | .vision_id | objectives |
| projects.parent_id (scope) | .project_id | okr_scopes |
| projects.parent_id (initiative) | .scope_id | okr_initiatives |

## 类型→表映射

- goals WHERE type='vision' → visions
- goals WHERE type='area_okr' → objectives
- goals WHERE type IN ('area_kr','global_kr') → key_results
- projects WHERE type='project' → okr_projects
- projects WHERE type='scope' → okr_scopes
- projects WHERE type='initiative' → okr_initiatives
