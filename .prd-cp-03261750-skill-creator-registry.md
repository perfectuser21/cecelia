# PRD: skill-creator Step 0 接入 Brain Registry API

## 背景

skill-creator 的 Step 0 查重依赖静态文件 `skill-index.md`，不可靠：
- 文件手动维护，容易过期
- Brain 新增了 system_registry 表，已成为 SSOT
- 应该调 Brain API 查重，静态文件作为降级方案

## 目标

更新 `skill-creator/SKILL.md` 的 Step 0：
1. 创建新 skill 前先调 `GET /api/brain/registry/exists?type=skill&name=/xxx`
2. 也用关键词搜索 `GET /api/brain/registry?type=skill&search=关键词`
3. Brain 不可用时降级到读静态 `skills-index.md`
4. 创建完成后调 `POST /api/brain/registry` 登记

## 成功标准

- skill-creator/SKILL.md Step 0 包含 Brain API 查重步骤
- Step 0 包含创建后登记步骤
- Step 0 包含 Brain 不可用时的降级说明
