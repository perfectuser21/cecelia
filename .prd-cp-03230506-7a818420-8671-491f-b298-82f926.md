# PRD: OKR体系统一 PR3——API routes 层切新表

## 背景
PR1 已完成数据迁移，新 OKR 表（objectives/key_results/okr_projects）中已有数据。
此 PR 负责 API 路由层切换，将 packages/brain/src/routes/ 下 OKR 专用路由从旧表切换到新表。

- task-projects.js: projects 表 → okr_projects 表
