# PRD: Knowledge Modules 深度知识页接入 Dashboard

## 背景

西安 Codex 已生成 86 个 Cecelia 系统知识页（HTML 格式），存放在 docs/knowledge/ 下，
通过 BACKLOG.yaml 追踪状态。需要将这些知识页整合进 Dashboard，用原生 React 组件渲染，
不使用 iframe。

## 目标

1. /knowledge/modules 页面展示所有 86 个模块（按 brain/engine/workflows/system 分组）
2. 每张卡片显示标题、摘要描述、优先级标签
3. 点击卡片进入详情页，可看到来源文件列表
4. 已完成的模块（有 output_url）可在 Dashboard 内联查看 HTML 知识页
5. KnowledgeHome 新增深度知识页入口，现有 4 个卡片保持不变
6. 不使用 iframe，全部 Dashboard 原生组件渲染

## 成功标准

- /knowledge/modules 展示所有模块，按 brain/engine/system/workflows 分组
- 每张卡片显示：标题、摘要描述（desc）、优先级标签（P0/P1/P2）
- 点击卡片可看到来源文件列表（source_files）
- GET /api/brain/knowledge/modules 从 BACKLOG.yaml 读取并返回结构化数据
- KnowledgeHome 新增深度知识页入口，现有 4 个卡片保持不变
- 不使用 iframe，全部 Dashboard 原生组件渲染
