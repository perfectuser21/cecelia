## PRD - feat(dashboard): 实现 /projects/compare 项目并排对比页面

**需求来源**: Brain 自动调度任务（agent_ops 领域）
**功能描述**: 后端 API POST /api/brain/projects/compare/report 已在 PR #783 合并。需实现对应前端页面，支持多选 2-4 个项目并排对比。
**涉及文件**:
- 新建: `apps/api/features/planning/pages/ProjectCompare.tsx`
- 修改: `apps/api/features/planning/index.ts`（注册路由）
- 修改: `apps/dashboard/src/pages/ProjectsDashboard.tsx`（添加入口按钮）

**成功标准**:
- /projects/compare 路由可访问
- 多选下拉选择 2-4 个项目
- 并排对比卡片展示：评分、进度条、任务统计、优劣势
- 响应式布局
- 深色主题一致

**非目标**: 不修改后端 API，不修改数据库 schema

**API**:
- GET /api/tasks/projects — 获取项目列表
- POST /api/brain/projects/compare/report — 获取对比数据

### 异常处理 & 报警规范

| 异常场景 | 处理方式 | 报警级别 | eventType |
|----------|----------|----------|-----------|
| 项目列表加载失败 | 显示错误提示，允许重试 | P3 | projects_load_error |
| 对比报告请求失败 | 显示错误提示，保留已选项目 | P3 | compare_report_error |
| 选择项目数量不足 2 个 | 禁用对比按钮，提示用户 | P3 | N/A |
