# Task Card: Gate 0c diff-only 扫描

## 需求（What & Why）

**功能描述**: 修复 `verify-step.sh` Gate 0c，将全文件 grep 改为只扫描 diff 新增行。

**背景**: Gate 0c 对整个文件执行 console.log 扫描，导致任何涉及 executor.js（含 58 处结构化日志）的 PR 都被误报。改为只扫 diff 新增行语义更准确：禁止新增调试代码，不影响预存在的结构化日志。

**不做什么**: 不改变 Gate 0c 的阻止能力，只改变扫描范围。

## 成功标准

1. [PRESERVE] Gate 0c 检查仍然存在且能阻止新增 console.log
2. [BEHAVIOR] Gate 0c 使用 diff 新增行检查（diff_added 变量存在）
3. [GATE] CI 全部通过
