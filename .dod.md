## 测试方法

**自动化测试**:
- CI workflow 验证文件存在性
- YAML 格式验证（profiles/web.yml）
- JSON schema 验证（dashboard/schema.json）

**手动验证**:
- 执行 `./run.sh check --profile=web` 验证入口脚本
- 执行 `./run.sh export --profile=web` 验证导出功能
- 检查生成的 quality-status.json 符合 schema 定义
