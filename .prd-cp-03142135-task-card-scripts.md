# PRD: 引擎脚本支持 Task Card 格式（向后兼容）

**需求来源**: 新增 .task-cp-{branch}.md 格式作为 PRD+DoD 合并文件，需要让现有检查脚本能识别此格式。
**功能描述**: 修改 check-prd.sh 和 check-dod-mapping.cjs，支持新的 .task-{branch}.md 格式，向后兼容旧的 .prd-*.md + .dod-*.md 格式。
**涉及文件**:
- `packages/engine/scripts/devgate/check-prd.sh`
- `packages/engine/scripts/devgate/check-dod-mapping.cjs`

**非目标**: 不修改其他脚本、不改变现有格式的行为。

## 成功标准

- check-prd.sh 优先查找 `.task-{BRANCH}.md`，找不到则 fallback 到 `.prd-{BRANCH}.md`
- check-dod-mapping.cjs 优先查找 `.task-{branch}.md`，找不到则 fallback 到 `.dod-{branch}.md`
- 两种格式的现有行为不被破坏（向后兼容）
- CI 全部通过

N/A - 无 IO 操作，纯脚本逻辑修改。
