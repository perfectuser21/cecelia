# PRD: KR 优先级收敛与依赖排序引擎

branch: cp-03231715-57a9b810-d1ef-40b7-8bc6-2ba21b

## 功能描述

为 Brain 增加一个自动收敛层，对当前所有活跃 KR 按价值、依赖关系、进度和阻塞程度进行综合评分，
输出 top 3 主线焦点，并给出暂停/降级建议。避免多个项目同时摊薄资源。

## 成功标准

- GET /api/brain/kr/convergence 接口可调用，返回含 top3 + pause_candidates 的 JSON
- 评分算法综合 4 个维度：进度权重(35%)、任务活跃度(30%)、项目密度(20%)、指标动量(15%)
- top3 结果可解释：每个 KR 附带 score、rank、reason 字段
- pause_candidates 包含低分 KR 及 suggestion 字段（"暂停" 或 "降级"）
- 单元测试覆盖评分逻辑（含边界：0 个 KR、全部 0 分、并列分）
