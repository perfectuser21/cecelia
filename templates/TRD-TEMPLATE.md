## 📦 PRD 拆解

### PRD-01: 数据层
- **目标**: 创建/修改数据库表
- **依赖**: 无
- **产物**: migrations/, models.py
- **验收**: 迁移可执行，表结构正确

### PRD-02: API 层
- **目标**: 实现 CRUD API
- **依赖**: PRD-01
- **产物**: routes.py, schemas.py
- **验收**: API 测试通过

### PRD-03: 业务逻辑
- **目标**: 核心算法实现
- **依赖**: PRD-02
- **产物**: services/, utils/
- **验收**: 单元测试 + 集成测试

### PRD-04: 前端
- **目标**: UI 实现
- **依赖**: PRD-02
- **产物**: components/, pages/
- **验收**: E2E 测试

### PRD-05: 收尾验证 ⚠️ 必须
- **目标**: 全链路验证
- **依赖**: ALL
- **产物**: 测试报告
- **验收**: 见下方收尾清单
