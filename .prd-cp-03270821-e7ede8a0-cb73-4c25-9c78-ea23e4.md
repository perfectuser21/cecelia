# PRD: 文档+聊天分栏界面（Notion AI 风格）

## 背景
在 Cecelia Dashboard 里建一个 Notion AI 风格的分栏界面：
- 左边：文档内容（markdown 渲染，可编辑）
- 右边：Claude 聊天窗口
- 对话时 Claude 能通过 <doc_update> 标记更新左边文档
- 支持模型选择（Haiku/Sonnet/Opus）
- 文档通过 Brain design_docs API 存取

## 成功标准

- 打开 /docs/:id 页面，左右分栏显示
- 在右边输入，Claude 回复并可通过 <doc_update> 标记更新左边文档
- 模型切换生效（Haiku/Sonnet/Opus）
- 文档内容通过 Brain design_docs API 读写持久化
- 新增 /docs/list 文档列表页
