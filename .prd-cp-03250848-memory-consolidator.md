# PRD: 记忆系统 PR8 — 接入 runConversationConsolidator 到 tick

## 背景

conversation-consolidator.js 注释写明"Brain 每 5 分钟调用一次 runConversationConsolidator()"，
但 tick.js 里既没有 import 也没有调用。
效果：长对话从未被自动压缩成 memory_stream 条目，episodic memory 只有单条对话片段，没有主题摘要。

## 改动

tick.js 新增 10.19 节，fire-and-forget 调用 runConversationConsolidator()：
- 每 tick（5min）触发一次，由 runConversationConsolidator 内部判断是否需要压缩
- 失败不阻塞主 tick 循环

## 成功标准

- tick.js import runConversationConsolidator 并在 10.19 节调用
- 调用有 catch 错误处理
- 原有 10.18 runSuggestionCycle 调用不变
