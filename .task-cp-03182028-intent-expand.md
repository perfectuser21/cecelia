# Task Card: intent-expand skill

## 需求（What & Why）

**功能描述**: 实现 `/intent-expand` skill，Brain 在接收到 `intent_expand` 类型任务时调用此 skill。Skill 从 Brain 数据库沿 Task → Project → KR → OKR → Vision 层级链条查询，将层级上下文与原始 PRD 合并，生成 enriched PRD。

**背景**: Brain 现已支持 `intent_expand` task_type（见 `task-router.js`），但缺少对应的执行 skill。Agent 执行 intent_expand 任务时需要这个 skill 来完成意图扩展工作。

**不做什么**:
- 不自动派发给 Codex（那是 Brain 的 executor 工作）
- 不修改原始 PRD 文件（只读取 + 输出新文件）
- 不依赖 Notion API（只依赖 Brain API + psql）

## 成功标准

1. [ARTIFACT] `packages/engine/skills/intent-expand/SKILL.md` 存在，包含可执行的 Step 1-4 流程
2. [BEHAVIOR] Skill 能通过 Brain API（`GET /api/brain/goals/:id`、`GET /api/brain/projects/:id`）查询层级上下文，输出包含 Vision/OKR/KR/Project 的 enriched PRD
3. [BEHAVIOR] 输出文件 `.enriched-prd-<branch>.md` 写入仓库根目录，格式正确
4. [GATE] CI 全部通过
