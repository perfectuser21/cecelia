# PRD: Knowledge & Documentation System

## 背景
构建统一文档与知识系统，提供结构化的开发记录、设计文档和用户批注存储。

## 功能描述
- 3张新表：dev_records / design_docs / user_annotations
- 3组 Brain API
- 5个前端页面（Dev Log / Decision Registry / Design Vault / Daily Diary / Knowledge Map）
- PR 自动写入 dev_records
- 每日日报 Tick 生成 daily_diary

## 成功标准

- migration 185 执行后3张新表在 DB 存在
- Brain API 3组可用
- evolution-scanner.js 合并 PR 时自动写入 dev_records
- tick.js 每日生成一条 daily_diary user_annotation
- Dashboard /knowledge 路由下可访问5个新页面
