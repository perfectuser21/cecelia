# Task Card: PipelineOutputPage 重构为作品主页（四 Tab）

## 需求（What & Why）
**功能描述**: 将 PipelineOutputPage 重构为包含四个 Tab（Summary / 生成记录 / 发布记录 / 数据记录）的内容作品主页，采用深色风格展示内容产出与多平台发布状态。
**背景**: 现有 PipelineOutputPage 展示内容单薄，无法满足内容运营对"一个作品的全生命周期"可视化需求；需要将内容生成、发布、数据三条线聚合在同一页面。
**不做什么**: 不新建后端数据库表；不修改 Brain API；发布记录与数据记录暂时使用 mock 数据占位。

## 成功标准
> [ARTIFACT] = 产出物（文件/API/配置）
> [BEHAVIOR] = 运行时行为（系统如何响应输入）
> [GATE] = 门禁（CI/测试/代码质量）

1. [ARTIFACT] PipelineOutputPage 组件包含四个 Tab（Summary / 生成记录 / 发布记录 / 数据记录）
2. [BEHAVIOR] 访问 Pipeline 详情页时，默认展示 Summary Tab，可切换至其他三个 Tab
3. [BEHAVIOR] 生成记录 Tab 展示封面图、卡片图片、文案预览，并显示 pipeline 执行阶段信息
4. [BEHAVIOR] 发布记录 Tab 展示 8 个平台各自的发布状态（已发布/定时/未发布/失败），无数据时显示 mock 占位
5. [BEHAVIOR] 数据记录 Tab 展示各平台播放/点赞/评论/收藏数据，无数据时显示 mock 占位
6. [ARTIFACT] 页面整体样式采用深色背景（#07050f）、紫色渐变标题，符合 Justin Welsh 预览页风格
7. [GATE] CI 全部通过
