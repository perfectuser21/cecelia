# Task Card: 自驱引擎 — Cecelia 自主决定下一步

## 需求（What & Why）

**功能描述**: Cecelia 看到自己的体检报告（probe + scan）后，自主分析优先级，创建任务给自己做。闭环：体检 → 分析 → 创建任务 → 执行 → 下次体检验证。

**背景**:
- Probe 和 Scanner 已就位，Cecelia 能看到自己了
- 但看到之后不会自己行动——需要用户下指令
- 缺少"自驱"：我看到问题 → 我决定做什么 → 我创建任务 → 我做

**不做什么**:
- 不改 probe/scanner 的逻辑
- 不改任务派发架构
- 自驱引擎只创建任务，不直接修代码

## 成功标准

1. [ARTIFACT] 新增 self-drive.js，读取 probe+scan 结果，调 LLM 分析优先级
2. [BEHAVIOR] 自驱引擎能自动创建待办任务（调 createTask）
3. [BEHAVIOR] 有防重复机制（不重复创建同类任务）
4. [GATE] 所有现有测试通过
