# PRD: branch-protect.sh 支持 Task Card 格式

**需求来源**: 用户请求——新的 .task-cp-{branch}.md 格式需要被 hook 支持
**功能描述**: 修改 branch-protect.sh，支持 .task-cp-{branch}.md 新格式，向后兼容旧的 .prd-*.md + .dod-*.md 格式。优先级：新格式 (.task-*) > 旧格式 (.prd-* / .dod-*)。
**涉及文件**: packages/engine/hooks/branch-protect.sh
**非目标**: 不修改 CI 相关脚本，不修改其他 hook 文件

## 成功标准

- 写入 .task-cp-xxx.md 时不被 hook 拦截（当前会把它当作未授权文件）
- 查找 PRD/DoD 文件时，优先找 .task-{branch}.md，找不到再找 .prd-/.dod-
- .task-*.md 内容验证：必须包含 "## 成功标准" 且包含 "- [ ]" checkbox
- 现有的 .prd + .dod 格式验证逻辑完全保留（不破坏现有行为）
- CI 全部通过

### 异常处理 & 报警规范

N/A - 无 IO 操作（纯 Shell 脚本逻辑改动）
