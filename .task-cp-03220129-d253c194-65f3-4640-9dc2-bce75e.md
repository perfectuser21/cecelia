# Task Card: 重构 auto-learning.extractTaskSummary（圈复杂度 21 → 10）

## PRD

### 背景

代码复杂度扫描发现 `packages/brain/src/auto-learning.js` 中的 `extractTaskSummary` 函数圈复杂度为 21，超过阈值 10，需要重构降低复杂度。

### 目标

将 `extractTaskSummary` 函数圈复杂度从 21 降至 10 以下，同时保持所有现有行为不变。

### 重构方案

提取两个命名子函数：
1. `_extractRunResult(result, maxLength)` — 处理 cecelia-run 合成结果（exit_code/stderr_tail/failure_class 分支）
2. `_extractObjectSummary(result, maxLength)` — 处理通用对象摘要（属性优先链）

主函数 `extractTaskSummary` 简化为：空值检查 → string 检查 → 对象分派 → 兜底，复杂度降至 ≤ 4。
