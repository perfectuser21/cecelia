# PRD: Brain PATCH custom_props + stop-dev.sh 步骤状态回写

## 背景

状态机双 Gate（Script Gate + Subagent Gate）已在 PR #963 完成。但 Brain 目前不知道每个运行中的 `/dev` 任务当前在哪一步，Dashboard 无法显示实时进度。

## 目标

1. Brain PATCH `/api/brain/tasks/:id` 新增 `custom_props` JSONB merge 支持
2. `stop-dev.sh` 在每次运行时将当前 `/dev` 步骤状态写入任务的 `custom_props`，让 Brain 实时知道每个任务在哪一步

## 成功标准

1. PATCH `/api/brain/tasks/:id` 传 `{custom_props: {dev_step: 2}}` 返回 200，`custom_props` 包含新字段
2. PATCH 同时支持 `status` 和 `custom_props`，两者均可选，但至少需要一个
3. stop-dev.sh 在有 `task_id` 时，每次运行都调用 Brain PATCH API 更新 `custom_props.dev_step`

## 不做什么

- 不改 Dashboard 前端（Task B 负责）
- 不改 task_id 写入逻辑（已有）
