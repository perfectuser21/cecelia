# PRD: 修复 Brain status.js 缺失导入

## 背景
重构拆文件时，`packages/brain/src/routes/status.js` 漏导入了 `IDEMPOTENCY_TTL` 和 `ALLOWED_ACTIONS`，导致 `/api/brain/status` 返回 500 错误。

## 问题
- `status.js` 第 68 行使用 `IDEMPOTENCY_TTL`（计算幂等 TTL）
- `status.js` 第 70 行使用 `ALLOWED_ACTIONS`（列出允许的动作）
- 但 import 行只导入了 `getActivePolicy, getWorkingMemory, getTopTasks, getRecentDecisions`

## 修复方案
在 `status.js` 的 import 行中加上 `IDEMPOTENCY_TTL` 和 `ALLOWED_ACTIONS`。

## 成功标准
- `/api/brain/status` 不再返回 500 错误
- `status.js` 可以正常加载（无 ReferenceError）
