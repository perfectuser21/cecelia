# Task Card: fix(ci): Deploy webhook soft-fail + L4 DoD gate PR-only

## 需求（What & Why）

**功能描述**:
1. Deploy workflow 每次 push to main 都因 HTTP 405 失败（dev-autopilot.zenjoymedia.media 无法接收 POST），导致 CI 整体标红。
2. L4 DoD Verification Gate 在直接 push to main 时因 GITHUB_HEAD_REF 为空而硬失败（该检查只应在 PR 时运行）。

**背景**: CI pipeline 被两个无关紧要的问题持续拖累，影响开发效率和 CI 信噪比。

**不做什么**: 不修复 deploy webhook 服务端（那是另一个问题），只让客户端失败不阻断 CI。

## 成功标准

1. [ARTIFACT] deploy.yml Deploy job 加 `continue-on-error: true`
2. [ARTIFACT] ci-l4-runtime.yml 的 devgate-checks job 加 `if: github.event_name == 'pull_request'` 条件
3. [BEHAVIOR] Deploy job 失败时 CI 整体不挂
4. [BEHAVIOR] L4 DoD gate 仅在 PR 时运行，push to main 时跳过
5. [GATE] CI 全部通过
