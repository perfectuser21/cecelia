# PRD: CI L3 集成测试门禁 — Brain↔Engine↔API 跨模块验证

## 背景

当前系统只有单元测试，改 A 不知道 B 坏了。`apps/api/`、`apps/dashboard/` 完全没有测试扫描，Brain 有 10/151 模块无测试，CI L3 只要求 `feat:` PR 有 `*.test.ts`，单元测试就够，没有跨模块要求。

## 目标

1. 补充 Brain API + Engine Hook 联动契约测试（mock 模式，不依赖真实服务）
2. CI L3 增加 Brain 测试覆盖警告 check（不阻塞，warning 模式）

## 成功标准

- `apps/api/src/__tests__/brain-api-integration.test.ts` 新增 session-start 联动契约测试
- `packages/brain/src/__tests__/integration/brain-endpoint-contracts.test.js` 新增 Brain 端点契约测试
- `.github/workflows/ci-l3-code.yml` 新增 `brain-test-coverage-warning` job
- 所有新测试在 CI ubuntu-latest 无需真实服务即可通过
