# PRD: 修复视频号 iframe 视频上传

## 背景

视频号发布页面 `https://channels.weixin.qq.com/platform/post/create` 使用嵌套 iframe 结构，
直接操作 `page` 对象无法找到上传按钮，导致视频无法上传。

## 目标

实现视频号视频发布脚本，正确穿透 iframe 层级，完成视频上传流程。

## 成功标准

## 成功标准

- 脚本文件 `packages/workflows/skills/shipinhao-publisher/scripts/publish-shipinhao-video.cjs` 存在
- 脚本使用 `frameLocator` 或 `page.frames()` 穿透 iframe 层级
- 支持 `--title`、`--video`、`--desc`、`--dry-run` 参数
- Windows SSH 文件传输后通过 CDP 连接 Playwright 自动化上传
