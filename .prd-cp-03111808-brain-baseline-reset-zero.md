## PRD - Brain 测试 Baseline 同步至当前真实失败数

**需求来源**: 旧 baseline（400/125）高于当前实际失败数，需要同步为真实值以缩小容忍范围
**功能描述**: CI 运行测得当前实际失败数为 unit:235 / integration:10，将 baseline 从虚高值更新为真实值，
阻止将来 PR 引入新测试失败时被旧 baseline 容忍掉
**涉及文件**:
- `packages/brain/ci/brain-test-baseline.txt`（400 → 235，当前实际失败数）
- `packages/brain/ci/brain-integration-baseline.txt`（125 → 10，当前实际失败数）

## 成功标准

- `brain-integration-baseline.txt` 内容为 `10`
