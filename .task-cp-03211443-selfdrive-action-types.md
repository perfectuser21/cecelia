# Task Card: SelfDrive 输出类型扩展

## 需求（What & Why）
**功能描述**: 扩展 self-drive.js 的 runSelfDrive()，使其不仅能创建 dev 任务，还能调整 Project 优先级、暂停/激活 KR、更新 Roadmap（Now/Next/Later）。
**背景**: 当前 SelfDrive 只能 create_task，无法对已有 OKR 体系做调整操作。需要让 SelfDrive 成为完整的自驱决策执行器。
**不做什么**: 不删除 KR/Project（只能暂停），不做 UI 展示。

## 成功标准

1. [BEHAVIOR] buildAnalysisPrompt() 输出格式包含 5 种 action 类型：create_task / adjust_priority / pause_kr / activate_kr / update_roadmap
2. [BEHAVIOR] runSelfDrive() 能根据 action.type 执行对应数据库操作（projects.sequence_order / goals.status / brain_config roadmap）
3. [BEHAVIOR] 每次 SelfDrive 最多执行 2 个调整类 action（adjust_priority/pause_kr/activate_kr/update_roadmap），防止大幅变动
4. [BEHAVIOR] 所有调整类 action 都记录到 decision_log 表（有审计）
5. [GATE] 所有现有测试通过
