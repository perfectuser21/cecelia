# Task Card: 小红书视频发布脚本

## 需求（What & Why）
**功能描述**: 新增 `publish-xiaohongshu-video.cjs`，实现小红书视频内容自动化发布，支持视频文件上传、标题、标签、封面。
**背景**: 小红书 publisher 当前只支持图文（publish-xiaohongshu-image.cjs），缺少视频发布能力。本 PR 与图文方案对齐，使用 CDP WebSocket 直连 Windows PC。
**技术方案**:
- 复用 xiaohongshu-publisher 现有 CDP WebSocket 架构（CDPClient）
- CDP 端口：19225（小红书视频专用 Chrome 实例）
- Windows PC Tailscale: 100.97.242.124
- 发布流程：进入发布页 → 切换视频 tab → SCP 视频到 Windows → 文件上传 → 填写标题/标签/封面 → 发布
- 参数：--video --title --tags --cover（可选）
- 脚本位置：packages/workflows/skills/xiaohongshu-publisher/scripts/publish-xiaohongshu-video.cjs
**不做什么**: 不修改 N8N workflow，不处理批量视频，不做 API 方案。
