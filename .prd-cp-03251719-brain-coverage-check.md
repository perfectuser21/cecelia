# PRD: 扩展 check-coverage-completeness 覆盖 Brain 代码

## 背景

`check-coverage-completeness.mjs` 目前只扫描 Engine 自身，不覆盖 Brain。Brain 有 151 个 src/*.js 文件，其中 13 个无对应测试，但 CI 对此一无所知。

## 目标

在 `check-coverage-completeness.mjs` 中新增 Check 4，扫描 Brain src 覆盖率：
- 高风险模块（tick/thalamus/executor/cortex/planner）缺测试 → error（exit 1）
- 其余模块缺测试 → warning（非阻断）

## 成功标准

- [ ] 运行脚本输出含 "Brain src 覆盖" 章节
- [ ] executor.js 通过前缀匹配识别为有测试
- [ ] 所有高风险模块有测试时 exit 0
- [ ] 高风险模块缺测试时 exit 1
