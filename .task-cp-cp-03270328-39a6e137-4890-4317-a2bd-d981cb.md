# Task Card: Knowledge Modules 页面接入 Dashboard

## 需求（What & Why）
**功能描述**: 将西安生成的 86 个知识页接入 Dashboard，用 Dashboard 原生组件渲染 `/knowledge/modules` 页面。
**背景**: Cecelia 系统文档化工程，Codex 已生成 76/86 个 HTML 知识页，存储在 `docs/knowledge/BACKLOG.yaml` 中，需要在 Dashboard 展示这些模块，方便团队浏览。
**不做什么**: 不用 iframe，不修改已有知识页 HTML，不改现有路由结构，不加认证逻辑。

## 成功标准
- `/knowledge/modules` 页面展示所有模块卡片（brain/engine/system/workflows 四个分组）
- 点进任意模块可看到详情：标题、摘要、关键数字、来源文件
- `GET /api/brain/knowledge/modules` 返回结构化数据（从 BACKLOG.yaml 读）
- 不用 iframe，全部用 Dashboard 原生组件渲染
