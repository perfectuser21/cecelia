# PRD: fix(executor): worktree_creation_failed 未知错误根因修复

## 背景
cecelia-run.sh 中存在 use-after-delete bug：第 427 行 `rm -f "$wt_stderr_log"` 删除临时文件后，
第 431 行再尝试 `cat "$wt_stderr_log"` 读取同一文件，文件已不存在，始终输出 `unknown error`。
导致 5 个 dev 任务以 `worktree_creation_failed: unknown error` 失败，无法诊断真实原因。

## 目标
1. 修复 use-after-delete bug，让 err_log 记录真实 stderr 内容
2. 增加日志详细度（head -10 而非 head -3）

## 成功标准
- cecelia-run.sh 中先保存 stderr 内容到变量，再删除临时文件
- worktree 创建失败时，err_log 中写入真实错误原因而非 unknown error
