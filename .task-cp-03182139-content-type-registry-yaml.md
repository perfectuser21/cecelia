# Task Card: 内容类型注册表（YAML 配置化）— PR1: YAML schema 设计

## 需求（What & Why）
**功能描述**: 在 packages/brain/src/content-types/ 目录下建立 YAML 驱动的内容类型注册表，实现内容类型配置与 Pipeline 代码解耦。
**背景**: 内容工厂 Initiative（330bce6f）— 当前内容类型（solo-company-case）硬编码在 skill 里，无法扩展。需要一个 YAML schema + 加载器，让新增内容类型只需创建一个 YAML 文件，无需改 Pipeline 代码。
**不做什么**: 不修改现有 Pipeline 编排逻辑（content-pipeline-orchestrator.js），不做 Dashboard UI，不做 Publisher 集成。
