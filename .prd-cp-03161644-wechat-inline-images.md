## PRD - 公众号图文混排：HTML 内嵌图片自动上传

**需求来源**: 用户需求，自动发布流水线
**功能描述**: 在 publish-wechat-article.cjs 中新增能力：解析 HTML 正文里的本地图片路径（`<img src="本地路径">`），自动上传到微信 CDN，将 src 替换为微信 URL，再提交草稿发布。同时修复草稿创建缺少 thumb_media_id 导致的 40007 错误。

**涉及文件**:
- 修改 `packages/workflows/skills/wechat-publisher/scripts/publish-wechat-article.cjs`

## 成功标准

1. 传入含本地图片路径的 HTML，发布后公众号文章图片正常显示（微信 CDN URL）
2. 图片路径支持：绝对路径（`/nas/...`）、相对路径（`./images/...`，相对于内容目录）
3. 外链图片（`http://`、`https://`）保持不变，不处理
4. 无本地图片时行为完全不变（向后兼容）
5. 单张图片上传失败时打印警告，继续处理剩余图片，不中止发布
6. `--content` 参数和 `--content-dir` 模式均支持
7. 修复 errcode=40007：草稿创建时传入 thumb_media_id（封面上传后获得）

**非目标**:
- 不处理视频/音频等非图片媒体
- 不处理 CSS background-image
