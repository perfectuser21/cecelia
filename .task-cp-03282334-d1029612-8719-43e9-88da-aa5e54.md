# Task Card: Brain 关键路由集成测试补全

## 需求（What & Why）

**功能描述**: 给 Brain 最关键的 4 个路由补集成测试，使用真实 PostgreSQL 连接（非 mock）。

**背景**: CI 目前只有 mock DB 的单元测试，无集成验证。`/api/brain/context` 刚修复了 SQL bug 但无测试保护，下次 SQL 变更可能无感静默回归。

**不做什么**: 不重写现有 mock 测试；不测试 /health 以外的 goals.js 路由；不引入 Brain 进程级测试。

## 目标路由

| 路由 | 路由文件 | 核心 SQL 表 |
|------|---------|------------|
| GET /api/brain/context | routes/context.js | objectives, key_results, dev_records, tasks |
| GET /api/brain/tasks | routes/task-tasks.js | tasks |
| GET /api/brain/okr/current | routes/okr-hierarchy.js | objectives, key_results |
| GET /api/brain/health | routes/goals.js（via routes.js） | working_memory, events |

## 成功标准

- 测试不 mock db.js（真实 PostgreSQL）

- 对 goals.js（/health）只 mock 无 DB 的外部副作用（如 tick loop start 函数），但保留 DB 查询真实执行

**要改的文件**:
