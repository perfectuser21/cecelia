# PRD：嘴巴 Agent 配置 UI 统一化

## 背景

PR #389 将其他 brain agents（L1 丘脑、L2 皮层、反刍、反思）改为折叠展开式 AgentCard。
但嘴巴（mouth）保留了独立的硬编码 `MouthModeSelector` 2×2 网格组件，UI 风格不一致。

## 目标

删除 `MouthModeSelector` 硬编码组件，让嘴巴和其他 brain agents 一样走 `AgentCard`：
- 折叠态：显示当前选中组合（provider · model · 调用方式）
- 展开态：按 provider 分组显示所有可选组合
- 选项来源：`model-registry.js` 中 mouth 的 `allowed_models`（当前为 claude-sonnet-4-6, claude-haiku-4-5-20251001）

## 范围

- 仅改 `apps/api/features/system/components/BrainLayerConfig.tsx`
- 删除 `MouthModeSelector` 组件
- 删除 `fetchMouthConfig`, `updateMouthConfig` 的使用（改用 `updateBrainAgent`）
- 去掉 `a.id !== 'mouth'` 过滤，mouth 加入 `brainAgents` 列表
- 后端无需改动（`updateBrainAgent('mouth', ...)` 已能正确更新 `config.mouth`）

## 成功标准

1. 嘴巴显示为折叠展开式 AgentCard（和其他 brain agents 一致）
2. TypeScript 编译 0 错误
3. 无遗留 `MouthModeSelector` 相关代码
