# PRD：皮层建议自动转 Brain 任务（autoCreateTaskFromCortex）

## 背景

当 cortex（皮层）完成 RCA 分析后，可能在 `recommended_actions` 里包含 `type: 'create_task'` 的建议。
目前这些建议只被写入 `rca_result.mitigations`，但不会真正在 Brain DB 中创建任务。
这导致皮层的洞察无法自动转化为行动（任务派发）。

## 目标

让皮层的 `create_task` 建议自动在 Brain DB 中创建真实任务：
1. 解析 `rcaResult.recommended_actions` 中的 `create_task` 动作
2. 调用 `createTask()` 将其写入 tasks 表
3. 内置去重（title + goal_id 相同的不重复创建）
4. 如果创建失败（非去重）只打 log，不让整个 cortex 任务失败

## 成功标准

1. `cortex.js` 的 `CORTEX_ACTION_WHITELIST` 包含 `create_task`
2. `actions.js` 的 `isSystemTask` 包含 `'cortex'` 为 systemSource
3. `tick.js` 的 `processCortexTask` 在 RCA 完成后自动创建建议任务
4. 单元测试覆盖：有/无 create_task 建议、去重场景
