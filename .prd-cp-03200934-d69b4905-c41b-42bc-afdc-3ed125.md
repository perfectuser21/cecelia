# PRD - [SelfDrive] Scanner 识别规则修正任务

## 背景
任务来自 Brain 自动调度，需要将已排队的 [SelfDrive] 任务从 queued 状态转为 active，更新 Scanner 识别规则库。

## 功能描述
将已排队的 [SelfDrive] 任务从 queued 状态转为 active，更新 Scanner 识别规则库。目标是区分：
1. 内嵌运行的能力（Hooks、DevGate、熔断器等，不通过 tasks 工作）
2. 真正孤立未激活的能力

完成后将获得准确的能力地图，用于后续孤岛激活决策。

## 成功标准
1. Scanner 识别规则库已更新，能够准确区分内嵌能力和孤立能力
2. 已排队的 [SelfDrive] 任务状态从 queued 转为 active
3. 系统生成准确的能力地图，标识所有能力的激活状态
4. 能力地图 API 端点可供查询
5. CI 全部通过

## 不做什么
- 不改变现有能力的运行逻辑
- 不删除或停用任何现有功能
- 不进行架构重构
