# PRD - 修复 cecelia-bridge Math.min 截断

**需求来源**: Cecelia Brain 自动检测到 Cortex Opus 超时被 120s 上限截断
**功能描述**: cecelia-bridge.js line 74 的 Math.min 调用导致所有超时上限被截断至 120000ms（2分钟），Cortex 调用 Opus 模型时实际超时远超 120s，导致 cecelia-run 进程被强制终止。
**涉及文件**: packages/brain/src/cecelia-bridge.js（已在 PR #858 修复并合并）
**成功标准**: PR #858 已合并到 main，Math.min 截断 bug 不再影响 Cortex Opus 超时

## 成功标准

- [x] cecelia-bridge.js Math.min 截断逻辑已修复（PR #858 合并）
- [x] Cortex Opus 超时不再被 120s 上限阻断
