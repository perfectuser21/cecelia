## 实现方案

**KR 表映射**（根据 task-router.js PR2 确认）:
- KR → `key_results` (id, objective_id, title, status, target_value, current_value, area_id)
- 项目 → `okr_projects` (id, kr_id, title, status, area_id)
- 范围 → `okr_scopes` (id, project_id, title, status)
- Initiative → `okr_initiatives` (id, scope_id, title, status)
- Tasks 关联: `tasks.okr_initiative_id` → `okr_initiatives.id`

**关键变化**:
1. `goals WHERE type='area_okr'` → `objectives`（用于 status/okr-context）
2. KR dispatch 相关 → `key_results`（与 task-router.js 一致）
3. `projects + project_kr_links` → `okr_projects.kr_id` + okr_scopes + okr_initiatives 链
4. 进度 rollup: `goals.progress` → `key_results.current_value`
5. `tasks.project_id` → `tasks.okr_initiative_id`（task-router.js 已用此字段）
