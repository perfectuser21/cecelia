# PRD - resource-monitor.js

## 背景
派发层缺乏 OS 物理资源感知。本 PR 新建 resource-monitor.js 作为统一资源状态接口，供 slot-allocator 和 circuit-breaker 调用。

## 目标
创建 packages/brain/src/resource-monitor.js，提供 getResourcePressure() 函数，返回当前系统资源压力状态。

## 技术规格
- 读取 os.loadavg()[0]（1分钟均值）
- 读取 process.memoryUsage().heapUsed / heapTotal
- 返回结构：{ cpu_load_1m, memory_pct, cpu_throttle: bool, memory_throttle: bool, any_throttle: bool }
- 阈值：cpu_load_1m > 2.0 → cpu_throttle=true；memory_pct > 0.85 → memory_throttle=true
- 导出：getResourcePressure()（同步），resetThresholds(cpu, mem)（测试用）
