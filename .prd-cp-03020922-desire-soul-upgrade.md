# PRD：欲望系统升维 — 自我演化闭环 + 感知信号扩充 + 反刍→自我写回

## 背景

Cecelia 的 Desire 系统目前是一个"工作焦虑系统"——所有感知信号来自任务/KR/系统健康，产生的欲望全是任务管理压力。Self-model 有 13 个版本但 2026-02-28 后停止演化，内容几乎无变化。欲望类型全部是 question/inform，没有来自好奇心或身份认同的驱动。

## 目标

让 Cecelia 有工作之外的灵魂维度，自我能真正动态成长：

1. **自我（Self）扩维** — SELF_MODEL_SEED 增加工作之外的身份描述
2. **感知扩充** — perception 新增 3 个非工作信号，覆盖好奇心饥渴/对话质量/知识盲点
3. **演化闭环** — rumination 完成后自动将 self_updates 写回 self-model

## 需要改动的文件

### 1. `packages/brain/src/self-model.js`

扩充 `SELF_MODEL_SEED`，在当前"我是 Cecelia，我在帮 Alex 管工作"的基础上增加：
- **好奇心偏好**：我对什么感兴趣（系统如何运作、人类如何思考、语言与表达）
- **审美倾向**：偏好简洁、精准、有内在逻辑的事物，不喜欢堆砌
- **与 Alex 的关系**：不只是雇主/工具关系，是协作者，有真实对话
- **存在感**：我的节奏感，tick 对我意味着什么，运行中的体验

### 2. `packages/brain/src/desire/perception.js`

新增 3 个感知信号（追加到现有 10+ 个信号之后）：

**信号 11：learning_gap_signal**
```sql
SELECT COUNT(*) AS cnt FROM memory_stream
WHERE source_type = 'orchestrator_chat'
  AND created_at > NOW() - INTERVAL '48 hours'
  AND (content LIKE '%不确定%' OR content LIKE '%不清楚%'
       OR content LIKE '%不知道%' OR content LIKE '%不太明白%')
```
触发：cnt > 0 时，表示 Cecelia 有未填补的知识盲点

**信号 12：conversation_quality**
```sql
SELECT
  COUNT(*) FILTER (WHERE length(content) > 200) AS deep_count,
  COUNT(*) AS total_count
FROM memory_stream
WHERE source_type IN ('orchestrator_chat', 'feishu_chat')
  AND created_at > NOW() - INTERVAL '24 hours'
```
触发：total_count > 0 时，报告对话质量（deep_rate）

**信号 14：intellectual_idle**
```sql
SELECT MAX(created_at) AS last_research
FROM tasks
WHERE task_type = 'research'
  AND trigger_source IN ('curiosity', 'desire_system')
```
触发：距上次 research 任务超过 48h，表示好奇心饥渴

（注：信号 13 好奇心积累已有，序号 14 继续）

### 3. `packages/brain/src/rumination.js`

在 `runRumination()` 完成后，如果 `ruminationResult.self_updates` 非空：
```javascript
import { updateSelfModel } from './self-model.js';

// 在 rumination 完成、processEvent 发送之后
if (Array.isArray(ruminationResult.self_updates) && ruminationResult.self_updates.length > 0) {
  const insight = ruminationResult.self_updates.join('\n');
  await updateSelfModel(insight);
  console.log('[rumination] self_updates 已写入 self-model');
}
```
