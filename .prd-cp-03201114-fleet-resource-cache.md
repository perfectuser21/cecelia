# PRD: Fleet Resource Cache — Brain 全局资源感知

## 背景
Brain 的 slot-allocator 只看美国本机 CPU/内存，不知道西安两台 Mac mini 的资源状态。西安 Codex Bridge 5 个账号完全空闲。

## 需求
1. 新建 fleet-resource-cache.js，每 30 秒采集 3 台编程机的 CPU/内存
2. 为每台机器算 effectiveSlots（可并发任务数）
3. 新增 GET /api/brain/fleet API 查看全局资源
4. tick.js 启动时注册 fleet cache 刷新
5. infra-status.js export 采集函数供复用

## 成功标准
- curl localhost:5221/api/brain/fleet 返回 3 台机器的实时 CPU/内存/effectiveSlots
- fleet cache 每 30 秒自动刷新
- 离线机器标记 online=false
