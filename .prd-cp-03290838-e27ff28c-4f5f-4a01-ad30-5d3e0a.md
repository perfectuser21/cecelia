# PRD: Generator subagent Stage 2 写代码独立化

## 目标

把 /dev Stage 2 的代码编写从主 agent 内部拆成独立 Generator subagent。
Generator subagent 只接收 Sprint Contract + coding 规范 + 代码库上下文，
不接收 Planner 的推理过程和 brain context。主 agent 变成纯编排者。

## 背景

当前 `packages/engine/skills/dev/steps/02-code.md` 是 v5.3.0，主 agent 直接写代码（2.2 写代码节）。
已部署版本是 v6.0.0，包含 Generator subagent 派发节。需要将 repo 版本同步升级至 v6.0.0。

## 不做什么

不修改 verify-step.sh、不修改 hooks、不修改 CI pipeline。

## 成功标准

- 02-code.md 版本升级到 6.0.0
- 02-code.md 包含 Generator subagent 派发节（2.2）
- 02-code.md 包含 Generator 内部执行节（2.2.5，原 2.2 重命名）
- Generator subagent 隔离约束（禁止调用 Brain API）在文件中明确声明
- Engine 版本文件同步为 13.54.0
- 2.3 自验证节和 2.4 code_review_gate 节保留不变
