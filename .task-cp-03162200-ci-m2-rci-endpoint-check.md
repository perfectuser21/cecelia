# Task Card: CI M2 — 新 Brain API Endpoint 必须有 RCI 回归契约

## 需求

写脚本 `scripts/devgate/check-new-api-endpoints.mjs`，检测 PR diff 中新增的 Brain API 路由，要求 `regression-contract.yaml` 中有对应条目。加入 L2 brain-l2 job。

## 技术方案

1. 脚本用 `git diff origin/main` 提取新增路由行（`+router.(get|post|patch|delete|put)`）
2. 解析出 path 字符串，在 regression-contract.yaml 中查找匹配
3. 无匹配则报错，告知需要补充契约条目
4. 无新路由时直接通过（exit 0）
