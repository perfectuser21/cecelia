# Task Card: CI cross-integration 修复 — Brain变更触发workspace集成测试

## 需求（What & Why）

**功能描述**: 修复 `ci-l4-runtime.yml` 的 cross-integration 问题和 engine L4 devgate 触发逻辑

**背景**:
当前 `ci-l4-runtime.yml` 中：
- `workspace-integration` job 只在 `apps/` 变更时触发
- `brain-integration` job 只在 `packages/brain/` 变更时触发

Brain 改 API 后，workspace 集成测试不跑，导致 `brain-api-integration.test.ts` 无法验证跨组件兼容性。存在跨组件回归漏洞。

**不做什么**:
- 不修改任何测试文件
- 不修改 Brain 或 Workspace 源码
- 不改变 devgate-checks 的具体检查内容（check-dod-mapping.cjs / rci-execution-gate.sh）

## 成功标准

1. [BEHAVIOR] Brain PR 合并后，ci-l4-runtime 同时运行 brain-integration（4 shards）+ workspace-integration
2. [BEHAVIOR] workspace PR 合并后，ci-l4-runtime 同时运行 workspace-integration + brain-integration（4 shards）
3. [GATE] CI L1/L2/L3/L4 全部通过
