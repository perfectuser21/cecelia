# Task Card: Harness v2.0 PR1 — Brain 断链 + Task Router Sprint 循环

## 需求（What & Why）
**功能描述**: 在 Brain 中实现 Initiative Harness v2.0 的 Sprint Generate→Evaluate→Fix 循环断链。新增 3 个 task_type（sprint_generate/sprint_evaluate/sprint_fix），在 execution-callback 中实现对抗循环路由。
**背景**: 复刻 Anthropic Harness 的 Generator↔Evaluator 对抗模式，每个 dev task 写完代码后由独立 Evaluator 测试，FAIL 则 Generator 修复再测，直到 PASS。
**不做什么**: 不实现 Sprint Contract 对抗（v2.1）、不实现 Playwright（v2.2）、不改 /dev skill、不删旧断链。

## 成功标准
> 新增 3 个 task_type 路由 + execution-callback Harness 断链 + 新旧模式兼容
