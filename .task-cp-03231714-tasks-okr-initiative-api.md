## 成功标准

- POST 创建 task 时传 okr_initiative_id，字段写入 DB
- PATCH 可更新 okr_initiative_id 字段（含置 null）
- GET /:id 返回 okr_initiative_id（SELECT * 已覆盖）
- 无 okr_initiative_id 时向后兼容，不影响现有调用
