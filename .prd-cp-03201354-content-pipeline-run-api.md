# PRD: Content Pipeline Run API + 路由改西安

## 概述
为内容工厂 pipeline 添加手动触发 API、子任务查询、产出查询端点，并将 task-router 的 content-* 路由从 US 改到西安 Codex。

## 成功标准
1. [ARTIFACT] POST /api/brain/pipelines/:id/run 端点可用
2. [ARTIFACT] GET /api/brain/pipelines/:id/stages 返回子任务进度
3. [ARTIFACT] GET /api/brain/pipelines/:id/output 返回产出信息
4. [BEHAVIOR] task-router content-* 路由改为 xian
5. [GATE] 不破坏已有 API
