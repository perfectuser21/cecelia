# PRD: /dev Verifier Subagent 状态机

## 背景
/dev 当前问题：探索后锁死薄 Task Card、主 agent 自测有确认偏差、格式检查代替质量检查，导致实现深度只有 0.3。

## 目标
给 /dev 每个关键步骤加独立 Verifier Subagent，实现状态机：每步完成由 subagent 独立检查质量，FAIL 则主 agent 修复直到 PASS。

## 成功标准

1. 02-code.md 加入 DoD 深度挑战（探索后、写代码前）
2. 02-code.md 写代码改为逐条 DoD 内循环 + Verifier Subagent
3. 01-taskcard.md 加 LLM 质量 gate
4. 04-learning.md 加 LLM 质量 gate
5. Stop Hook 机制完全不受影响

## 不做什么
不改 Stop Hook、不改 PR/CI 流程、不引入外部依赖
