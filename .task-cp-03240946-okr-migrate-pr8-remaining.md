### 类型映射规则

| 旧查询 | 新查询 |
|--------|--------|
| `FROM goals WHERE type='area_okr'` | `FROM objectives` |
| `FROM goals WHERE type IN ('area_kr','global_kr','key_result')` | `FROM key_results` |
| `FROM goals WHERE type='vision'` | `FROM visions` |
| `FROM goals WHERE id=$1` (通用查找) | `FROM objectives WHERE id=$1 UNION ALL FROM key_results WHERE id=$1` |
| `FROM projects WHERE type='project'` | `FROM okr_projects` |
| `FROM projects WHERE type='scope'` | `FROM okr_scopes` |
| `FROM projects WHERE type='initiative'` | `FROM okr_initiatives` |
| `FROM projects WHERE parent_id=$1 AND type='initiative'` | `FROM okr_initiatives oi JOIN okr_scopes os ON oi.scope_id=os.id WHERE os.project_id=$1` |
| `UPDATE goals` | `UPDATE objectives` 或 `UPDATE key_results` |
| `projects.name` | `okr_*.title` |
| `projects.kr_id` | `okr_projects.kr_id` |
